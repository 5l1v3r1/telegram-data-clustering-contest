Раджа Кодури поделился деталями об архитектуре графических процессоров Intel Xe

18 Nov 2019, 05:02 by id_1

Начислено вознаграждение
Эта новость написана посетителем сайта, и за неё начислено вознаграждение.

Буквально несколько часов назад руководитель графического подразделения Intel, Раджа Кодури (Raja Koduri), выступая на конференции, посвящённой высокопроизводительным вычислительным системам, поведал новую порцию данных об архитектуре графических процессоров под кодовым названием Xe, которыми делится с широкой общественностью ресурс Wccftech.

реклама

Источник изображения: Intel

реклама

Архитектура Intel Xe станет основой для целого ряда продуктов и будет представлена в трёх вариациях: базой для интегрированных чипов и моделей начального уровня станет Intel Xe LP (5-20 Вт с возможностью повышения до 50 Вт), для продуктов среднего и высокого уровней, а также профессиональных решений будет предназначена Intel Xe HP (75-250 Вт), а Intel Xe HPC найдёт своё применение в GPU для высокопроизводительных систем.

реклама

Источник изображения:  Wccftech

Графические процессоры Xe HPC смогут получить в своё распоряжение до 1000 исполнительных блоков, которые будут сообщаться с несколькими каналами памяти, характеризующимися высокой пропускной способностью, посредством нового масштабируемого интерфейса XEMF (XE Memory Fabric). Кроме того, продукты будут наделены унифицированным кэшем большой ёмкости, который обеспечит высокую производительность в режиме с двойной точностью (FP64) и может быть доступен для нескольких GPU.

Источник изображения: Intel

реклама

При производстве новых графических решений Intel задействует свои передовые наработки — технологии объёмной компоновки Foveros для организации интерконнекта с кэшем и EMIB (Embedded Multi-Die Interconnect) для объединения памяти с графическими процессорами. Связь между CPU и GPU будет осуществляться посредством интерфейса Compute Express Link (CXL).

Источник изображения: Intel

Первым продуктом Intel, предназначенным для высокопроизводительных систем, станет графический процессор Ponte Vecchio, названный в честь моста во Флоренции. Данный GPU будет производиться по 7-нм техпроцессу и состоять из 16 вычислительных чиплетов. Шесть графических процессоров Ponte Vecchio в дуэте с двумя CPU семейства Sapphire Rapids станут «сердцем» ноды суперкомпьютера Aurora, который начнёт свою работу в 2021 году. В качестве ПО выступит программный стек oneAPI, предоставляющий разработчикам весь набор необходимых инструментов, что, по замыслу Intel, должно существенно упростить работу.

Источник изображения: Intel

Что касается моделей игровых видеокарт, то в их основу лягут графические процессоры, выпускаемые по 10-нм технологическому процессу. Судя по имеющимся данным, дебют первых продуктов для массового потребителя намечен на 2020 год.

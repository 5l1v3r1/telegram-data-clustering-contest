Votre PC est-il capable de faire Star Wars Jedi: Fallen Order ? 23 cartes graphiques testées

18 Nov 2019, 07:52 by Aurélien LAGNY

Quand un nouveau jeu PC sort, forcément on a envie de savoir si son PC sera capable de le faire tourner dans de bonnes conditions. Pour cela il y a les informations techniques données par l’éditeur avec les configurations minimales et recommandées. Mais on a beau dire, une bonne mise en situation c'est bien aussi. Alors on vous propose d'aller chez Tech Power Up qui vient de faire une bonne tripotée de benchs sous Star Wars Jedi: Fallen Order, ce qui vous permettra de savoir ou se placera votre PC.

Configuration minimale :

  • OS: Windows 7/8.1/10 64 bits
  • Architecture: 64 bits
  • Processeurs :i3-3220 à 3.7Ghz ou FX-6100
  • Mémoire: 8 Go RAM
  • Carte graphique (AMD): Radeon HD 7750
  • Carte graphique (NVIDIA): GeForce GTX 650
  • DirectX: API 11
  • Espace : 55 Go

Configuration recommandée :

  • OS: Windows 7/8.1/10 64 bits
  • Architecture: 64 bits
  • Processeur :Ryzen 7 1700 ou i7-6700K
  • Mémoire: 32 Go de RAM
  • Carte graphique (AMD): RX Vega 56
  • Carte graphique (NVIDIA): GTX 1070
  • DirectX: API 11
  • Espace : 55 Go

GitHub 設立極地檔案庫，儲存開源碼至少千年

18 Nov 2019, 08:52 by Unwire HK

科學家為了保存地球不同物種的 DNA 資料，在極地等較為穩定的地方設立檔案庫，確保不受天災人禍毀滅。最近 GitHub 亦推行類似計畫，希望長久保存開源程式碼，讓後世得以參考。

GitHub 最近公布新 GitHub Arctic Code Vault，準備在北極設立檔案庫封存開源碼紀錄。他們表示，開源程式碼是現代文明的基石，也是全人類共享的遺產，希望長久儲存最少 1 千年，讓後代也能使用。GitHub 將使用的極地檔案庫在挪威 Svalbard 山區地底 250 公尺深，除 GitHub 外不同國家都在該儲存室以數位方式保存重要紀錄。

GitHub 計劃與史丹佛圖書館、Long Now Foundation、Internet Archive、Software Heritage Foundation、Piql、Microsoft Research 以及牛津大學 Bodleian Library 合作，除了位於北極的極地檔案庫，也會將資料以不同格式儲存在各地。GitHub 將在 2020 年 2 月 2 日儲存全平台資料，屆時所有活躍的 Repository 都會記錄下來成為人類文明的象徵。

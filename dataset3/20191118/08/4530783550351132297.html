Ниж­ний Нов­го­род по­пал в Топ-10 го­ро­дов для встре­чи Де­да Мо­ро­за

18 Nov 2019, 08:21

Самым желанным городом для встречи с зимним волшебником стал Санкт-Петербург.

Нижний Новгород попал в десятку самых желанных городов для встречи с Дедом Морозом. Это выяснил сервис бронирования жилья для отдыха Tvil.ru, проведя опрос пользователей соцсетей ко Дню рождения Дед Мороза, 18 ноября.

Напомним, что с 9 ноября Всероссийский Дед Мороз из Великого Устюга начал свое ежегодное праздничное путешествие по городам страны с командой НТВ.

Главный новогодний волшебник планирует посетить 28 городов. 18 ноября он будет в Новосибирске.

Самым желанным городом, где россияне хотели бы встретиться с Дедом Морозом, оказался Санкт-Петербург. На втором месте — Москва. Нижний Новгород в Топ-10 занял пятое место.

Напомним, что в столице Приволжья новогодний волшебник будет находиться с 3 по 5 декабря. (0+)

У середу цілий мікрорайон у Сумах залишиться без води

18 Nov 2019, 08:39

У зв’язку з проведенням аварійно-ремонтних робіт по заміні  засувки  Д-200 мм по вул. СКД, 20  20 листопада, о 9:00 буде припинено подачу води за наступним адресами:

№№ 1, 3/1, 3/3, 5, 9, 10, 12, 14, 16, 18, 19, 20, 22, 7 /школа №6/  по вулиці СКД,

№№ 1, 3, 5, 7, 9, 10, 12, 14, по вул. К. Зеленко,

№№ 14, 15/1, 15/2, 17, 18/1  у пров. Тихорецький,

№ 12, 15/дит.садок/ по вул. Прокоф’єва.

КП «Міськводоканал» просить завчасно зробити запас води.

Після завершення аварійних робіт водопостачання буде відновлено.

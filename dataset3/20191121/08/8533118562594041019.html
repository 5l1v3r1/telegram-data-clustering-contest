Nike Joyride Dual Run è pronta x l’inverno

Arriva l’inedita la Nike Joyride Dual Run nelle nuove colorazioni autunno/inverno 19/20.

21 Nov 2019, 08:43 by Daniele Milano

La famiglia Joyride si allrga e dopo l’incredibile successo della Nike Joyride Run Flyknit, NIKE presenta ufficialmente al mondo dei runner la Nike Joyride Dual Run nelle inedite colorazioni autunno/inverno 19/20.

Correre senza fare fatica?

La Dual Run si conferma come “recovery shoes” perfetta nelle uscite defaticanti e di running rilassato. Creata infatti per ridurre l’affaticamento delle gambe, mantenendole riposate, la nuova Nike Joyride Dual Run è adatta sia ai runner meno performanti, che ai runner più esperti nelle corse di recupero. Il nuovo modello della famiglia Joyride combina soluzioni innovative Joyride per garantire una nuova percezione della corsa.

Cosa cambia dalla Nike Joyride Run Flyknit?

A differenza della Nike Joyride Run Flyknit che presenta la tecnologia Joyride lungo tutto la soletta interna, con tre celle distinte in corrispondenza di tallone, mesopiede e avampiede, la nuova Dual Run abbina i benefici delle microsfere alla resa di una scarpa da running tradizionale. Diciamo che si tratta di una versione ibrida, che riesce a combinare l’efficacia del sistema Joyride nel tallone e nel mesopiede, a quello del foam più tradizionale, per avere una risposta in fase di spinta dell’avampiede più efficace.

Joyride solo per tallone e mesopiede

  • Mesopiede – Joyride
  • Mesopiede – Joyride
  • Avampiede – Foam reattivo

Le capsule che contengono le microsfere Nike Joyride sono posizionate nel tallone e nel mesopiede così da assicurare un assorbimento ottimale dell’impatto con il terreno e un fit ancora più personalizzato. L’avampiede della nuova Nike Joyride Dual Run è invece formato da una schiuma morbida e reattiva che garantisce una transizione più fluida dal tallone alle dita del piede, dando rispetto al modello Run Flyknit una sensazione di maggior spinta in avanti.

Nike Joyride Dual Run è già disponibile!

La nuova Nike Joyride Dual Run è disponibile per i Nike member dal 14 novembre e per tutti dal 2 Dicembre.

Porsche Design анонсировала свой первый ноутбук премиум-класса

21 Nov 2019, 06:21 by Руслан Фадеев

Porsche Design объявила о выпуске ультрабука под названием Ultra One на базе процессора Intel. Выполненный в полностью металлическом корпусе лэптоп отличается компактными габаритами, небольшим весом и полностью бесшумной системой охлаждения. Компания уже опубликовала дату релиза и стоимость устройства в разных конфигурациях.

Корпус лэптопа с 15,6-дюймовым сенсорным дисплеем выполнен из магниевого сплава с полированными стальными вставками. «Сердцем» устройства стал процессор Core i5 8200Y со встроенной графикой Intel UHD 615. Заявленный тепловой пакет модели составляет 5 Вт, что позволило производителю использовать пассивную систему охлаждения, обеспечив бесшумную работу новинки. Старшая модель линейки получит процессор Intel Core i7.

Аудиосистема Ultra One создана при сотрудничестве с компанией Harman и оптимизирована для воспроизведения мультимедийных файлов на высокой громкости. Для работы с голосовым помощником предусмотрен массив направленных микрофонов. Набор интерфейсных портов (кардридер, HDMI, USB Type-C, и два USB 3.0 Type-A) размещён на задней части корпуса ноутбука.

Ноутбук получил 8 ГБ оперативной памяти и SSD-накопитель объёмом 512 ГБ. Работает новинка под управлением актуальной ОС Windows 10, для быстрой разблокировки устройства с помощью службы Windows Hello предусмотрен сканер отпечатков пальцев.

Технические характеристики:

  • Дисплей: сенсорный, 15,6", 1920x1080
  • Процессор: Intel Core i5 8200Y
  • Оперативная память: 8 ГБ
  • Накопитель: SSD 512 ГБ
  • Разъёмы: 2 x USB Type-A, USB Type-C, HDMI, 3,5-мм аудиоразъём
  • Безопасность: сканер отпечатков пальцев
  • Связь: Wi-Fi, Bluetooth 4.2
  • ОС: Windows 10 Home
  • Габариты: 359 х 249,5 х 13,8 мм;
  • Вес: 1,5 кг

Релиз Porsche Design Ultra One ожидается в конце декабря текущего года по цене $1350. Версия с процессором Intel Core i7 обойдётся покупателям в $1500.

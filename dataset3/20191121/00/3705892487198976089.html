Стали известны цены на шофёрскую комиссию для омичей

21 Nov 2019 by Дарья Молчанская

Омский Минздрав будет следить за новыми тарифами при прохождении шофёрской комиссии.

 

Региональное министерство здравоохранения организовало дополнительные кабинеты для приёма анализов, которые теперь обязательны при прохождении шофёрской медкомиссии. Ранее "Омск Здесь" сообщал, что водители в Омске штурмуют больницы, чтобы получить медицинскую справку. Уже завтра сделать это будет сложнее и дороже.

Кабинеты будут открыты по адресу ул. Космический проспект, д. 20. Кроме этого, продлён график работы кабинетов в наркологическом диспансере. Для жителей сельских районов сбор биологических материалов будет осуществляться в центральных районных больницах.

Как рассказали корреспонденту "Омск Здесь" в омском Минздраве, тариф на дополнительные лабораторные исследования находится на контроле ведомства.

- Отбор биологического объекта (мочи) для проведения ХТИ будет стоить 1730 рублей. Отбор биологического объекта (крови) для исследования на CDT составит 3290 рублей, - уточнили в Минздраве. - Ставить цены выше этих государственные учреждения не имеют права. За частными медицинскими учреждениями мы не следим.

Забор анализов для проведения химико-токсикологического исследования будет производиться ежедневно с 08:00 до 19:00 с понедельника по субботу. Забор крови для определения CDT будет осуществляться ежедневно с 08:00 до 12:00. Таким образом, водителям необходимо будет выложить ещё 5020 руб.

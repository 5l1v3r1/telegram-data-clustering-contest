Samsung Confirms New Galaxy’s Faster Features

21 Nov 2019 by Ewan Spence

Samsung continues to test its new user interface, and hiding in some code is a reference to an impressive upgrade. The latest public beta of ‘One UI 2’ reveals an option for a screen to run at 120Hz.

Faster screens are proving to be attractive option for manufacturers looking to improve the user experience. In data to day use, a faster refresh offers smoother scrolling, snappier transitions, and a subtle feeling of speed and performance. And it opens up a far more immersive experience for smartphone gamers.

A number of premium Android handsets sported 90Hz screens during 2019, and key gaming handsets such as the Razer Phone 2 and the Asus ROG Phone 2 clock in at 120Hz.

Samsung’s move to faster screens will not only benefit the upcoming handsets, it will also give Samsung’s Display division a new product to sell to other manufacturers looking to pick up the latest technology. With the iPhone 12 expected to move to a 120Hz screen next year, a larger order from Apple could well be on the horizon for the South Koreans. 

The One UI beta has already proven to be a happy hunting ground for those looking out for upcoming features in Samsung’s 2020 portfolio. The 120 Hz screen detailed above can now join the discovery of increased exposure time and higher ISO values in the camera to facilitate better low light photography.

I do not expect every new handset from Samsung to use every single feature One UI feature but the flagship handsets such as the upcoming Galaxy S11 family, which is expected to be announced in February 2020, are key candidates for faster screens and improved cameras.

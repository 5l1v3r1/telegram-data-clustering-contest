ทรูออนไลน์จับมือ Linksys ส่งตรงความแรง WiFi ในบ้านระดับ Gigabit

21 Nov 2019 by Totsapon Kritsadangphorn

True Online ผนึกกำลังกับ Linksys เติมเต็มประสบการณ์การใช้งานอินเทอร์เน็ตในระดับ Gigabit ด้วยเทคโนโลยี Premium Mesh WiFi Solution ที่จะมาผสานพลังคู่กับ Gigatex Fiber Router ให้ความแรง WiFi ภายในบ้านแรงถึงระดับ Gigabit แบบเสถียรครอบคลุมการใช้งานทั่วทุกจุดภายในบ้าน โดยมอบสิทธิพิเศษสำหรับลูกค้าทรูออนไลน์ ซื้อ Linksys Velop รุ่น Dual Band และ Tri Band ได้ในราคาพิเศษลด 50% ตั้งแต่วันนี้- 31 ธันวาคม 2562 ที่ร้านทรูช้อปสาขาที่ร่วมรายการทั่วประเทศ

ทรูออนไลน์ ในฐานะผู้นำไฟเบอร์อินเทอร์เน็ตและเป็นสุดยอดแบรนด์ระดับโลก 3 ปีซ้อน ที่ครองใจผู้บริโภคมาโดยตลอด พร้อมสร้างปรากฎการณ์ใหม่ล่าสุด อัปชีวิตแรงเหนือระดับ สำหรับครอบครัวใหญ่ ให้ท่องเน็ตไร้สาย WiFi แบบเต็มสปีด 1 Gigabit ได้ทั่วทุกพื้นที่ภายในบ้าน โดยร่วมกับ Linksys ผู้ผลิตอุปกรณ์เน็ตเวิร์คระดับโลก นำเทคโนโลยี Premium Mesh WiFi Solution ให้บริการแก่ลูกค้าทรูออนไลน์ เพื่อทำงานเสริมกับ Gigatex Fiber Router เพิ่มการครอบคลุมความแรงสัญญาณ WiFi ให้ใช้งานได้เสถียรและเต็มสปีดมากขึ้น เหมาะกับบ้านที่มีพื้นที่ใช้สอยมาก และสามารถรองรับการเชื่อมต่ออุปกรณ์ Internet of Things (IoT) เปลี่ยนบ้านให้เป็น Smart Home Solution ควบคุมการทำงานอุปกรณ์ต่างๆ ผ่านทางอินเทอร์เน็ตได้ด้วยปลายนิ้ว
ซึ่งนับเป็นการต่อยอดความสำเร็จจากการสร้างมาตรฐานใหม่ของเน็ตไฟเบอร์แท้ในเมืองไทย True Gigatex Fiber รายแรกและรายเดียวในไทย โดยได้รับผลตอบรับเป็นอย่างดี จากจำนวนลูกค้าบรอดแบนด์ที่เพิ่มขึ้นใหม่ในไตรมาส 3 ปี 2562 อย่างต่อเนื่อง 9.4 หมื่นราย จนทำให้ปัจจุบันมีลูกค้ารวมทั้งสิ้น 3.7 ล้านราย ทรูออนไลน์ยังคงมุ่งเน้นที่จะให้ความสำคัญเรื่องคุณภาพการให้บริการ และสร้างความแตกต่างให้กับตลาดอยู่เสมอ โดยมอบสิทธิพิเศษสำหรับลูกค้าทรูออนไลน์ รับสิทธิ์ซื้อ Linksys Velop รุ่น Dual-Band และ Tri-Band ได้ในราคาพิเศษลด 50%คุณธนภูมิ ภาคย์วิศาล ผู้อำนวยการและหัวหน้าสายงานการพาณิชย์ทรูออนไลน์ บมจ. ทรู คอร์ปอเรชั่น
Linksys Velop เป็น Mesh WiFi ที่ออกแบบมาพร้อมเทคโนโลยี Intelligent Mesh เอกสิทธิ์เดียวของทาง Linksys ให้สัญญาณ WiFi ครอบคลุมพื้นที่ได้กว้างขึ้น ไม่ว่าจะเป็นภายในบ้านหรือสำนักงาน มีการบริหารจัดการเครือข่ายแบบอัตโนมัติ ทั้งรุ่น Dual Band และ Tri-Band อีกทั้งยังรองรับคุณสมบัติเพิ่มเติมได้ เพื่อยกระดับความสามารถของตัวผลิตภัณฑ์ให้มีประสิทธิภาพการทำงานที่ดียิ่งขึ้น ไม่ว่าจะเป็นการบล็อคเนื้อหาหรือคอนเทนต์ที่ไม่พึงประสงค์ (Linksys Shield) และการตรวจจับการเคลื่อนไหวภายในบ้านด้วยสัญญาณ WiFi (Linksys Aware*)
สำหรับ Linksys Aware เป็นคุณสมบัติใหม่ที่เพิ่มเข้ามาใน Linksys Velop Tri-Band เพื่อใช้ในการการตรวจจับการเคลื่อนไหวภายในบ้าน อาคาร หรือห้องที่มีความเป็นส่วนตัวผ่านสัญญาณ WiFi และมีการแจ้งเตือนเมื่อมีความเคลื่อนไหวเกิดขึ้น ซึ่งจะแสดงให้เห็นว่า Linksys Velop Tri-Band เป็น Intelligent Mesh ที่สามารถอัพเกรดเพื่อเพิ่มสมรรถนะได้อย่างไร้ขีดจำกัด

“สำหรับความร่วมมือกับทาง ทรูออนไลน์ในครั้งนี้จะเป็นยกระดับประสบการณ์ของผู้บริโภคให้สามารถใช้งานสมาร์ทดีไวซ์ในบ้านได้อย่างเต็มประสิทธิภาพ เสริมครอบคลุมทั่วพื้นที่ได้มากขึ้น และได้รับสัญญาณ WiFi ที่ดีในทุกๆ จุดของบ้าน”คุณชาคริต ศิรกุลประดิษฐ ผู้จัดการฝ่ายขายภูมิภาคอาเซียน เบลคิน อินเตอร์เนชั่นแนล

В России разработают «нейросмарт-платформу» для создания «нейропрофиля» человека

20 Nov 2019, 12:45 by Ян Альшевский

Российские ученые займутся разработкой нейросмарт-платформы, которая будет собирать данные о человеке, фиксируя его цифровой след. На его основе можно будет создать нейропрофиль человека, сообщает ТАСС. «Это мегапроект, который позволяет собирать данные и предлагать их для развития других проектов», — пояснила Эльвира Сыманюк, директор УГИ УрФУ.

«Платформа поможет сформировать цифровой след каждого человека, на основе чего можно создать его цифровой двойник — нейропрофиль. Нейросмарт станет базой данных с ограниченным доступом, которая будет собирать информацию из открытых источников», — пояснила Сыманюк.

Собранные данные планируется передавать медицинским и «другим технологическим компаниям».

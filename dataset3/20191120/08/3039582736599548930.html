Windows 版的 iTunes 也要被淘汰了？苹果正在找工程师开发新的多媒体应用

20 Nov 2019, 08:29 by 吴羚

虽然 Mac 系列每次更新都能成为焦点，但不可否认的是 Windows 电脑在个人市场的份额依然有绝对优势，所以苹果也为 iTunes、iCloud 等软件开发了 Windows 版本，而且今后可能还会有更多的苹果应用出现在微软的平台。

知名资讯网站 Neowin 发现苹果发布在领英上的招聘信息中出现了一个名为「高级软件工程师- Windows 多媒体应用」的职位，补充说明中还写道「为 Windows 开发下一代多媒体应用」。

该岗位所属部门为「Media Apps」（多媒体应用团队），「如果你热爱音乐并热衷于写代码」的描述暗示了苹果正在开发的可能是一款类似于 Apple Music 的 Windows 软件，或许还包括 Apple TV。

在今年发布的 macOS Catalina 中，iTunes 已经被 Music、Podcasts 和 TV 三个软件取代，但在 Windows 上，iTunes 仍然是使用 Apple Music、iTunes Store 以及备份 iPhone 的主要方式。

众所周知，苹果的软件一向很少对其他平台开放，而且比起微软在 iOS 和 Mac 平台上从邮箱、日程表到浏览器全方位发力，苹果对 Windows 的支持可以说相当凑合，基本上属于「又不是不能用」的态度，比如劝退很多 iPhone 和 iPod 使用者的桌面版 iTunes，操作之卡顿让人难以相信是以用户体验著称的苹果的产品，直到去年苹果为了适配 Windows 10 上线了的 UPW 版，体验才有所改善。

然而，虽说自有平台内的生态是苹果的竞争优势所在，但随着这几年其增值服务的扩张，苹果也开始重视起其他平台的用户，比如 2015 年上线的 Android 版的 Apple Music 体验就相当不错，遵循 Material Design 规范的设计也让人看到了苹果的诚意，Apple TV 现在也能在 Samsung TV、Roku 和亚马逊 Fire TV 等第三方设备上使用，今年 6 月苹果还和微软合作打造了新的 iCloud for Windows。既然功能臃肿的 iTunes 被淘汰已成定局，那么将 Mac 上的娱乐三件套搬到 Windows 上也在情理之中。

此外，苹果在这次招聘要求中提到拥有 UWP 开发经验是加分项，可以推测接下来在 Windows 上发布的大概率是 UWP 应用。9To5Mac 就指出，UWP 应用同时适用于 Windows 电脑和 Xbox One，很多用户把后者当作多媒体中心，因此在 UWP 平台推出多媒体应用对于 Apple TV+ 的扩张具有重要意义。

鉴于苹果还在找工程师，要用上 Windows 版的应用估计还要等上好一段时间。

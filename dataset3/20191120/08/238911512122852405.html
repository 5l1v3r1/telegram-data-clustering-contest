Firefox现已能够注册成为Windows上的WebP格式图片查看器

20 Nov 2019, 08:19 by raymon725

继 Google Chrome 之后，Windows 用户现也能够通过 Mozilla Firefox 来处理 WebP 格式的图片了。在安装了最新的 Firefox 72 每夜构建版（Nightly Build）之后，用户可以选择在该浏览器中默认打开和查看 WebP 格式图像。除了 Chrome 和 Firefox，Microsoft Edge 等现代浏览器均已支持 WebP 图片格式。

（题图 via Techdows）

WebP 支持有损和无损压缩，与传统 JPEG 和 PNG 格式图像相比，其压缩比更高、文件体积可减少 26% 以上。

有趣的是，尽管 Windows 10 自带的图片查看器和 Photos 应用仍不支持 WebP，但画图应用却支持。

通过安装谷歌提供的 WebP 编解码器，用户可在 Windows 上的图片查看器中，以本机方式查看 WebP 图像。

你甚至可以在受支持的现代 Web 浏览器中打开 WebP 图像，单请注意 Internet Explorer 没有 Microsoft Edge 那么先进。

最后，如果你在 Windows 10 1903 上遇到过无法将 Firefox 注册为默认的 WebP 文件处理应用的 bug，请参考如下方法。

● 转到‘设置 -> 应用 -> 默认应用’；

● 点按文件类型、选择默认应用；

● 找到 .webp 并选择‘Firefox Nightly’作为默认处理程序。

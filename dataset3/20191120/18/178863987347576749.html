Windows сделало важное обновление

20 Nov 2019, 18:54

С 18 на 19 ноября владельцы Windows 10 получили новое обновление 19H2 версии 1909.

Теперь ПК с Windows 10 начнут автоматически получать обновление November 2019 Update (версия 1909). Крупное обновление Windows 10 November 2019 Update (версия 1909) для Windows 10 было доступно еще с 12 ноября.

Вчера Microsoft сообщила о выходе обновления для Windows 10 Update Assistant (помощник по обновлению Windows 10), после установки которого можно будет автоматически обновить ПК до Windows 10 November 2019 Update.

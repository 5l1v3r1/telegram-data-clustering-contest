Представлен электрический универсал Volkswagen ID Space Vizzion c запасом хода почти 500 км

20 Nov 2019

Сегодня в рамках автосалона в Лос-Анджелесе компания Volkswagen представила свой новый электромобиль. Новинка получила универсальный тип кузова, и эта особенность подчеркнута в ее названии — ID Space Vizzion.

По сути, ID Space Vizzion — это тот же седан ID Vizzion, представленный в Женеве в прошлом году, но в иной обертке и с иным типом кузова: универсал вместо седана. А по факту, ID Space Vizzion — это уже седьмой электрический концепт Volkswagen после хэтчбека ID 3, спорткара ID R, большого и маленького кроссоверов ID Roomzz и ID Crozz, багги ID Buggy, микроавтобуса ID Buzz и, собственно, седана ID Vizzion.

Базовая версия ID Space Vizzion будет оснащаться электродвигателем мощностью 275 л.с., размещенным на задней оси. Но в планах есть и полноприводный вариант — суммарная мощность его силовых установок составит 355 л.с. Оценочный запас хода при использовании аккумуляторного блока емкостью 82 кВт·ч — 485 км по методике EPA. Для немаленькой машины (колесная база примерно соответствует колесной базе Volkswagen Atlas/Teramont) это неплохой результат.

В салоне электромобиля — большой дисплей диагональю 15,6 дюйма на центральной консоли и дополнительный экран поменьше — в приборном щитке. Впрочем, в компании считают более подходящим способом для восприятия информации — вывод основных параметров на ветровое стекло. Судя по всему, такая система будет уже в базе.

Универсал получил панорамную крышу, тачпады вместо традиционных дверных ручек (они плохи с точки зрения аэродинамики), а также лонгборды, хранящиеся в специальной нише в багажнике. Видимо, среди целевой аудитории модели маркетологи видят много молодежи.

А вот сможет ли молодежь позволить себе такой автомобиль — непонятно: стоимость пока не объявлена. В продажу в США ID Space Vizzion поступит в 2022 году.

Представлен агрессивный тюнинг для нового Toyota RAV4

20 Nov 2019 by Ирина Котова

Тюнеры автомобильного ателье Climate представили новый агрессивный тюнинг для компактного японского кроссовера пятого поколения Toyota RAV4.

Кроме того, тюнеры показали разработки и для премиальных японских кроссоверов Lexus. Напомним, ранее тюнеры уже показывали доработки для других автомобилей японского производства. Причем, все они пользовались популярностью у покупателей.

Главной особенностью представленных доработок в этот раз становится брутальный обвес, предназначенный только для кроссовера нового поколения. По словам разработчиков, они готовы предложить покупателям два варианта: Custom, разработанный для любителей настоящего внедорожья, а также Sporty, предназначенный для городской эксплуатации.

 

На данный момент можно ознакомиться только с эскизами предстоящих доработок, которые могут быть воплощены в жизнь по желанию заказчика. Производство обвесов начнется в конце текущего года. Но, несмотря на это, сейчас автомобилисты могут обратиться к тюнерам с индивидуальным заказом.

Стоимость новинки начинается от 1 756 000 рублей.

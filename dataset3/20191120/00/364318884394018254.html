Huawei เผยค่าเปลี่ยนอุปกรณ์ของ Mate X ที่สูงมาก : จอใหม่ราคาสูงถึง 30,000 บาท

20 Nov 2019 by Preedee Rirkvaleekul

Huawei ได้เริ่มวางจำหน่ายสมาร์ตโฟนพับจอได้ Mate X ด้วยราคาที่สูงถึง 16,999 หยวน (ประมาณ 72,900 บาท) และล่าสุดทาง Huawei ได้เปิดเผยค่าซ่อนแซมชิ้นส่วนอุปกรณ์ต่าง ๆ ของ Mate X ซึ่งมีราคาที่ค่อนข้างสูงมาก ไม่ว่าจะเป็น

  • แผงหน้าจอพับได้ : 7,080 หยวน (ประมาณ 30,400 บาท)
  • ชิปเซ็ต : 3,579 หยวน (ประมาณ 15,400 บาท)
  • กล้อง : 698 หยวน (ประมาณ 3,000 บาท)
  • แบตเตอรี : 278 หยวน (ประมาณ  1,200 บาท)
  • ฝาตัวเครื่อง : 361 หยวน (ประมาณ 1,500 บาท)

ข้อมูลข้างต้นเป็นราคาอย่างเป็นทางการจากประเทศจีน ซึ่งรวมค่าแรงงานและค่าชิ้นส่วนอุปกรณ์แล้ว และเมื่อ Mate X ได้วางจำหน่ายในตลาดต่างประเทศ ราคาดังกล่าวก็อาจได้รับการปรับเพิ่มตามราคาตลาดของประเทศนั้น ๆ ด้วยเช่นกัน

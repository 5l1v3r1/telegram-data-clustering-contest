Укренерго заявила про нестачу майже 500 млн грн для фінансування «зеленого тарифу»

Укренерго недоотримує кошти зі сплати за послуги передачі від низки підприємств, для яких з липня 2019 року дія нових тарифів на передачу призупинена в судовому порядку.

20 Nov 2019, 11:55

Національній енергетичній компанії Укренерго не вистачає коштів, щоб виплатити у першій декаді листопада аванс держпідприємству Гарантований покупець для компенсації його витрат зі сплати «зеленого тарифу». Дефіцит становить понад 481 млн грн, розмір авансу — 821 млн грн, повідомили в Укренерго.

Протягом липня-жовтня 2019 року Укренерго здійснило всі необхідні платежі для компенсації витрат на «зелений» тариф Гарантованого покупця.

«Основною причиною браку коштів для компенсації „зеленого тарифу“ зараз є нерівномірність та заборгованість по сплаті учасниками ринку послуги НЕК Укренерго з передачі електроенергії. Станом на 14 листопада 2019 року ця заборгованість становить більш ніж 1,2 млрд грн, а рівень розрахунків — 90%. Таким чином виплата коштів для Гарантованого покупця можлива лише за рахунок залишків на банківських рахунках Укренерго. Так, 14 листопада вони дорівнювали близько 780 млн грн, з яких компанії необхідно зарезервувати 530 млн грн для сплати податку на прибуток за підсумками дев’яти місяців 2019 року, який потрібно сплатити до 20 листопада 2019 року», — сказано у повідомленні.

У компанії наголосили, що крім заборгованості учасників ринку зі сплати тарифу за передачу е/е, Укренерго протягом липня-листопада недоотримує кошти зі сплати за послуги передачі від низки підприємств, для яких з липня 2019 року дія тарифів на передачу, які встановлені на друге півріччя 2019 року, призупинена в судовому порядку. В результаті загальне недоотримання коштів за липень-листопад через призупинення тарифів через суд становить 406,62 млн грн. Ці кошти можуть надійти в рахунок сплати тарифу на передачу лише після відповідного рішення судів.

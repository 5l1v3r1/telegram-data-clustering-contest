Llévalo en el bolsillo: cómo crear un servidor de Minecraft en Android

19 Nov 2019, 09:00 by Pedro Reyes

Minecraft es uno de esos videojuegos de los que nadie esperaba demasiado al inicio, pero que se volvió todo un éxito gracias a sus mecánicas de juego. Además, logró romper todos los paradigmas de los jugadores que, al momento de su lanzamiento en 2011, tendían a asociar un buen juego con su calidad gráfica.

Minecraft obtuvo tanta popularidad en PC, que sus desarrolladores, Mojang AB, decidieron portarlo a casi cualquier plataforma que se les ocurriese. Está disponible en Android, iOS, consolas de sobremesa y portátiles, MacOS, Linux, y hasta para Raspberry Pi. Además, recientemente anunciaron su versión con realidad aumentada, que se llamará Minecraft Earth.

Y como a nosotros nos gusta enseñar cosas útiles, pensamos que sería bueno escribir una guía sobre cómo crear un servidor de Minecraft desde un móvil Android.

Requisitos para crear un servidor de Minecraft desde el móvil

Como en todo tutorial, es necesario que contemos con algunas cosas previamente, con la finalidad de saber que podremos finalizar el proceso por completo, o si necesitamos algo extra para poder seguir adelante. Para este tuto necesitaremos:

  • Móvil Android compatible con Termux, una de las aplicaciones favoritas para instalar Linux en Android. Los requisitos de esta app son bajos, tener al menos Android 5 Lollipop y un SoC ARM.
  • Que el móvil tenga 4GB o más de RAM. Puede ser menos (2 o 3GB), pero esta es la cifra mínima ideal para asegurar un excelente funcionamiento.
  • Conexión a Internet estable y banda ancha. Porque necesitaremos descargar cerca de 550MB de datos.
  • Una cuenta de Minecraft. Que no es estrictamente necesaria para poner a funcionar el servidor, pero sí para probarlo.

Pasos para configurar un servidor de Minecraft en Android

Paso 1: conseguir que Termux ejecute un mini Ubuntu en nuestro dispositivo Android.

  • Descargar Termux y AnLinux en nuestro Android desde la Play Store.
  • Seleccionar Ubuntu en AnLinux y copiar el código de instalación.
  • Ejecutar el comando de instalación en Termux.
  • Ejecutar ./start-ubuntu.sh para iniciar la máquina virtual de Ubuntu en el móvil.

Paso 2: instalar OpenJDK 8 en la máquina virtual de Linux.

  • Ejecutamos el comando apt-get install software-properties-common para permitir la instalación.
  • Utilizamos add-apt-repository ppa:openjdk-r/ppa para descargar el repositorio de OpenJDK.
  • Luego, aplicamos el comando apt-get update.
  • Solicitamos la instalación con apt-get install openjdk-8-jre.
  • Probamos corriendo el comando java –version.

Paso 3: iniciar el servidor de Minecraft.

  • Visitamos la página de descarga del servidor, y copiamos el link de descarga.
  • Ejecutamos el comando wget -O minecraft_server.jar, seguido del link que acabamos de copiar.
  • Ejecutamos chmod +x minecraft_server.jar.
  • Corremos la línea de código java -Xmx1024M -Xms1024M -jar minecraft_server.jar nogui para iniciar el servidor.
  • La primera vez que iniciemos el servidor nos arrojará un error por no aceptar la licencia de software.
  • Ejecutamos el comando apt-get install nano, luego nano eula.txt para editarlo, y cambiamos el valor false por true.
  • Salimos utilizando ctrl-x then enter.
  • Ejecutamos java -Xmx1024M -Xms1024M -jar minecraft_server.jar nogui para iniciar el servidor.

Paso 4: instalar Ngrok

  • Ir a Ngrok y copiar el link para Linux (ARM64)
  • Corremos el comando wget -O ngrok.zip https://bin.equinox.io/a/nmkK3DkqZEB/ngrok-2.2.8-linux-arm64.zip (el link variará según la versión disponible de ngrok)
  • Instalamos con apt-get install zip unzip.
  • Descomprimimos con unzip ngrok.zip.

Paso 5: conectar el servidor local con los servidores de Ngrok.

  • Creemos una cuenta de ngrok.
  • Copiamos el token de autorización que nos dan al crear la cuenta, y lo ejecutamos con un comando parecido al siguiente: ./ngrok authtoken awefioauw4u0239840293j023jf0j23f23kj, donde varía el token final.
  • Ejecutamos ./ngrok tcp 25565.
  • Se abrirá una ventana que dice “Session Status online” en verde, copiamos la dirección que va después de “tcp://” para utilizarla más adelante.

Paso 6: iniciar Minecraft

  • Ingresamos en nuestra cuenta Minecraft.
  • Seleccionamos el modo multijugador, luego conexión directa, y pegaremos acá el link copiado al final del paso 5.
  • ¡Listo! Es momento de disfrutar el juego con nuestros amigos.

Como vimos en este tutorial, crear un servidor de Minecraft desde nuestro móvil no es tan difícil. Solo es necesario cuidar los pasos a seguir y ser minucioso en su ejecución para que funcione correctamente. Ahora, atrévete a dar el paso y disfrutar de este excelente juego con tus amigos.

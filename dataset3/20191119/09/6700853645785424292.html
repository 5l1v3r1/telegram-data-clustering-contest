Sempat Diblokir, Jaringan VPN Disdukcapil Aceh Singkil Kembali Aktif

19 Nov 2019, 09:12 by Salihin Barus

KBRN, Singkil : Sempat diblokir, Jaringan Virtual Private Network ( VPN ) di Dinas Kependudukan dan Pencatatan Sipil (Disdukcapil) Kabupaten Aceh Singkil  Kembali diaktifkan.

Hal itu di sebutkan Kepala Dinas Kependudukan dan Catatan Sipil (Disdukcapil) Kabupaten Aceh Singkil Yakup. SE, ketika di konfirmasi RRI di ruang kerjanya Selasa  (19/11/2019).

"Sekitar dua hari ini jaringan Virtual Private Network ( VPN ) kita sudah kembali di aktifkan. Sehingga, perekaman dan percetakan e-KTP, serta pengurusan pindah domilisi dan data entri baru  dapat terlayani. Jelas Yakup 

Kemudian di tambahkanya, penonaktifan jaringan Virtual Private Network ( VPN ) ini memang sempat menjadi permasalahan serius Disdukcapil Aceh Singkil. Pasalnya, beberapa pelayanan bagi masyarakat terganggu.

Sementara itu, di beritakan sebelumnya penonaktifan jaringan Virtual Private Network ( VPN ) itu di sebabkan adanya protes dari salah satu staf Disdukcapil Aceh Singkil yang tidak terima karena di mutasikan ke SKPK lainya. Sehingga Dirjen Disdukcapil memblokir jaringan  Virtual Private Network ( VPN )  tersebut.

" Jaringan di blokir karena adanya salah satu staf kita protes di mutasikan ke- SKPK lainya. Akan tetapi untuk saat ini jaringan sudah kembali di aktifkan." Tegasnya

Kepolisian Filipina Mencabut Sementara Izin Penggunaan Senjata Api Selama SEA Games 2019

19 Nov 2019, 19:15 by Zulfirdaus Harahap

Bola.com, Manila - Berbagai cara dilakukan Kepolisian Nasional Filipina (PNP) untuk menjamin keamanan selama SEA Games 2019. PNP telah menangguhkan izin untuk membawa senjata api bagi masyarakat sipil di daerah Metro Manila, Luzon Tengah, Calabarzon, dan La Union.

Peraturan tersebut diungkap dalam sebuah memorandum tertanggal 5 November 2019 yang dirilis ke media pada Sabtu (16/11/2019).

Pelaksana tugas Kepala Kepolisian Nasional Filipina, Letnan Jenderal Polisi, Archie Gamboa, memerintahkan penangguhan izin penggunaan senjata api di daerah-daerah yang terkait SEA Games 2019.

Archie Gamboa menegaskan hanya anggota kepolisian, penegak hukum lainnya, yang melakukan tugas resmi dan berseragam, yang diperbolehkan membawa senjata api. Langkah ini juga untuk menjamin keselamatan penduduk dan atlet.

"Ini bagian persiapan keamanan untuk memastikan kegiatan akan bebas dari insiden yang berhubungan dengan senjata api. Juga untuk memastikan keselamatan penduduk serta atlet lokal dan asing," bunyi memorandum tersebut seperti dikutip dari Inquirer, Selasa (19/11/2019).

Filipina menjadi negara Asia Tenggara yang melegalkan kepemilikan senjata api oleh penduduk sipil. Sekitar satu setengah juta penduduk Filipina memiliki izin membawa dan menggunakan senjata api.

Namun, banyak kepemilikan dan penggunaan senjata api dilakukan tangan yang salah. Data PBB mencatat, terjadi 8.000 lebih kasus pembunuhan setiap tahun di Filipina karena senjata api.

Pada 2014, Pemerintah Filipina bahkan mengeluarkan Undang-undang baru, yakni UU Republik 10591, yang mengizinkan Pastor, Wartawan, dan Dokter untuk memiliki senjata api. Profesi tersebut diizinkan membawa senjata api selama bertugas.

SEA Games 2019 rencananya akan dibuka pada 30 November 2019 oleh Presiden Filipina Rodrigo Duterte. Upacara pembukaan akan berlangsung di Philippine Arena, Bulacan.

Target 220 Medali Emas

Filipina akan menjadi negara dengan jumlah atlet terbanyak di SEA Games 2019. Sebagai tuan rumah, Filipina bakal menerjunkan 1.115 atlet.

Delegasi Filipina tercatat berjumlah 1.868 orang. Perinciannya adalah 1.115 atlet plus 753 pelatih. Jumlah ini juga menjadi jumlah terbanyak dalam satu kontingen di SEA Games 2019.

Dengan jumlah atlet 1.115 orang, wajar Filipina memasang target tinggi pada SEA Games 2019. Media lokal, Inquirer, menyebut induk olahraga di sana yakin kontingen tuan rumah bisa meraih setidaknya 220 medali emas.

Di sisi lain, target 220 medali emas pada SEA Games 2019 itu dianggap terlalu tinggi buat Filipina. Apalagi pada edisi sebelumnya, di Kuala Lumpur pada 2017, Filipina hanya mampu meraih 24 medali emas.

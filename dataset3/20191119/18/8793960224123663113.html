Bedeng di Pinggir Jalan Gatsu Terbakar Siang Tadi, Tak Ada Korban Jiwa

19 Nov 2019, 18:25 by Samsudhuha Wildansyah

Jakarta - Sebuah bedeng pekerja di pinggir Jalan Gatot Subroto, Jakarta Selatan kebakaran siang tadi. Kebakaran diduga disebabkan akibat arus pendek listrik.

"Iya kebakaran terjadi di bedeng pekerja proyek akibat hubungan arus pendek," kata Kasi Ops Damkar Jaksel, Sugeng saat dikonfirmasi detikcom, Selasa (19/11/2019).

Sugeng mengatakan kebakaran itu terjadi pada pukul 13.50 WIB dan berhasil dipadamkan sekitar pukul 14.20 WIB. Kebakaran itu terjadi di sebuah bedeng yang berlokasi di pinggir Jalan Gatot Subroto tepatnya di samping Pizza Hut.

Saat itu, api dengan cepat membakar area sekitar bedeng karena barang-barang di sana mudah terbakar. Api sempat dipadamkan dengan alat pemadam api ringan (APAR), namun tidak berhasil. Petugas pemadam kebakaran bergerak cepat memadamkan api.

"Penyebab kebakaran itu karena arus pendek pada bedeng pekerja proyek mengakibatkan penyalaan api. Karena bahan mudah terbakar banyak, maka api cepat membesar," jelas Sugeng.

Video detik-detik kebakaran viral di media sosial. Dalam video yang beredar, api tampak melalap kabel listrik di trotoar jalan.

Terlihat asap dari kebakaran menyelimuti Jalan Gatot Subroto. Sejumlah pejalan kaki tampak berhenti untuk mengabadikan momen kebakaran itu dan banyak pula pengendara yang melanjutkan perjalanannya dan menjauhi titik api.

Sugeng mengatakan tidak ada korban jiwa dalam kebakaran tersebut. Sementara arus lalu lintas di Jalan Gatot Subroto sempat tersendat.

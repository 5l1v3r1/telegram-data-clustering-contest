2 Unit Damkar Padamkan Api yang Muncul di Lantai 2 SMK Yadika 6 Bekasi

19 Nov 2019, 14:25 by Rolando Fransiscus Sihombing

Bekasi - Kebakaran susulan yang melanda SMK Yadika 6 Bekasi, Jatiwaringi, Pondok Gede berhasil dipadamkan. Titik api terdapat di sebuah ruangan di lantai 2 sekolah tersebut.

Pantuan detikcom, di SMK Yadika 6 Bekasi, Jalan Wadas, Jatiwaringi, Pondok Gede, api padam sekitar pukul 13.45 WIB, Selasa (19/11/2019). Ada dua unit mobil Damkar dari Dinas Pemadam Kebakaran DKI Jakarta dan Kota Bekasi yang melakukan pendinginan.

"Sudah padam, sedang pendinginan," ujar salah seorang petugas Damkar yang keluar dari Gedung SMK Yadika 6 Bekasi.

Saat ini petugas Damkar masih melakukan pendinginan di dalam gedung sekolah. Pemadam menyemprotkan air ke sisa-sisa kebakaran di lantai 2 Gedung SMK Yadika 6.

Sejumlah warga dan guru yang sebelumnya membantu memadamkan api telah meninggalkan lokasi. Saat ini keseluruhan ditangani oleh petugas Damkar.

Sebelumnya, Api tiba-tiba kembali muncul di sisa kebakaran Gedung SMK Yadika 6 Bekasi, Jatiwaringin, Pondok Gede. Kemunculan api ini membuat panik para guru.

Api itu muncul sekitar pukul 12.30 WIB. Kemunculan api pertama kali diketahui oleh warga sekitar yang sedang melihat-lihat sisa kebakaran.

Api yang tiba-tiba kembali muncul di sisa kebakaran Gedung SMK Yadika 6 Bekasi, membuat para guru panik. Para guru yang dibantu oleh warga sekitar berjibaku memadamkan api.

Sebuah mesin pompa air dibawa seorang warga untuk memadamkan api. Warga dan guru langsung menyalakan mesin tersebut dan menyemprotkan air ke titik api yang menyala.

Sony aibo 機器狗加入新 API 化身智慧家居助理

19 Nov 2019, 00:15 by Unwire HK

Sony 的機器狗 aibo 重出江湖之後，新版本強調雲端服務以及可擴充能力，最近 Sony 就推出了新的 API 讓開發者可以為 aibo 加入更多不同的互動功能，甚至化身智慧家居助理。

Sony 的 aibo 開發者計畫正在邀請開發者製作可以與 aibo 互動的服務和應用軟體，其中日本的日立（Hitachi）已經開始研究家電如何與 aibo 互動，例如忘記關上冰箱門，或者洗衣機程序完成的時候，可以通知 aibo 發出叫聲來提醒主人，另外也可以讓 aibo 充當智慧助理幫忙控制家電，充滿不同的可能性。此外，aibo 也可以被訓練進行不同的動作，例如幫忙撿東西等。

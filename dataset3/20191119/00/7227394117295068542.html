iOS 13.2.3 released with additional bug fixes

19 Nov 2019 by Prasad

Apple is now rolling out iOS and iPadOS 13.2.3 for compatible devices. The update is around 90MB in size but actual size will vary with device.

The list of changes in this update include:

  • Fixes an issue where system search and search within Mail, Files, and Notes might not work
  • Addresses an issue where photos, links, and other attachments might not display in the Messages details view
  • Fixes an issue that could prevent apps from downloading content in the background
  • Resolves issues that may prevent Mail from fetching new messages, and fail to include and quote original message content in Exchange accounts

The latest update is yet another in a string of patches that have come out since Apple released iOS last month. The OS certainly didn't release in a stable state but Apple has been relentlessly updating it ever since and it is definitely in better shape now than since it came out. The latest updates fixed yet more minor bugs that users have noticed in the past few weeks and while there maybe some more left, it's likely Apple is aware of them and already working on a fix.

We would definitely recommend keeping auto updates enabled and downloading every new patch going forward for the most stable performance.

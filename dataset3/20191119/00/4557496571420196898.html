性價比之爭！紅米Note 8 Pro與realme XT手機比較

19 Nov 2019 by genkiboy

Redmi Note 8 Pro 與 realme XT 兩款受到矚目的中階手機皆選在 10 月於台灣市場開賣，兩者不但價格相近，不約而同也採用了四鏡頭主相機設計，不過在處理器方面 Redmi Note 8 Pro 採用聯發科 Helio G90T 八核心處理器，而 realme XT 則是使用 Qualcomm Snapdragon 712 八核心處理器。究竟 Redmi Note 8 Pro 與 realme XT 這兩款極具話題性的手機，有哪些不同之處呢？拍照效果又是如何？請看《SOGI 手機王》的重點實測比較。

▲Redmi Note 8 Pro（左）與 realme XT （右）雖然採用了不同的處理器，但在螢幕尺寸、鏡頭數量與機身設計上都相當接近。

【外觀差異】

▲Redmi Note 8 Pro（左）搭載 6.53 吋 2,340 x 1,080pixels 解析度 IPS 螢幕，螢幕佔比為 91.4%；realme XT （右）搭載 6.4 吋 2,340 x 1,080pixels解析度 Super AMOLED 螢幕，螢幕占比達 91.9%，兩者皆為「水滴」螢幕設計，正面覆蓋康寧第五代大猩猩玻璃。

▲Redmi Note 8 Pro（左）前置電源指示燈、光源感應器與 2,000 萬畫素鏡頭；realme XT （右）前置聽筒與 1,600 萬畫素鏡頭，兩者前鏡頭皆採用 F2.0 光圈，可支援臉部解鎖。

▲Redmi Note 8 Pro（左）背蓋採用 3D 四曲面設計，覆蓋康寧大猩猩第五代玻璃，內置 4,500mAh 電池，可支援 NFC 與 18W 快速充電，並將指紋感應配置於背後；realme XT （右）背蓋亦採用康寧第五代大猩猩玻璃，內置 4,000mAh 電池，具備 VOOC 3.0 閃充。前者重量 199.8g，後者則是 183g，拿在手上的前者會感覺明顯略重一些。

▲Redmi Note 8 Pro（左）後置 200 萬畫素景深鏡頭（F1.89 光圈）+ 6,400 萬畫素主鏡頭（F1.89 光圈）+ 800 萬畫素超廣角鏡頭（120 度超廣角、F2.2 光圈）+ 200 萬畫素微距鏡頭（2cm 微距、F2.4 光圈）；realme XT（右）後置 800 萬畫素超廣角鏡頭（119 度超廣角、F2.25 光圈）+ 6,400 萬畫素主鏡頭（F1.8 光圈）+ 200 萬畫素景深鏡頭（F1.8 光圈）+ 200 萬畫素微距鏡頭（4cm 微距、F2.4 光圈）。

【相機比較】

自拍效果

▲Redmi Note 8 Pro（左）與 realme XT （右）依序使用人像散景模式拍攝，在前鏡頭自拍時可將背景虛化，讓主角更立體。兩者在相同的光源與場景下，背景與主體分離準確率都相當不錯，但對於美化的效果與膚色的潤飾效果不同。（ 兩者皆使用預設美化效果）

日間第一組

▲Redmi Note 8 Pro（上）與 realme XT （下）日間實拍效果，兩者整體拍攝效果都具有相當水準，但後者不論在色彩或是明暗細節的處理上更佳。（拍攝狀態：兩者皆開啟 AI 與自動 HDR）

日間第二組

▲Redmi Note 8 Pro（上）與 realme XT （下）日間實拍效果，兩者整體拍攝效果除了色溫表現稍有不同外，整體效果相當接近，但若以原圖檢視後者的細節仍略佳一些。（拍攝狀態：兩者皆開啟 AI 與自動 HDR）

超廣角第一組

▲Redmi Note 8 Pro（上）與 realme XT （下）超廣角實拍效果，兩者整體拍攝效果接近，但前者在地磚外牆等清晰度處理上較佳。（拍攝狀態：兩者皆開啟 AI 與自動 HDR）

超廣角第二組

▲Redmi Note 8 Pro（上）與 realme XT （下）兩者在傍晚實拍效果，在光線較弱的情形下，兩者所拍攝出的細節都較嫌鬆散，但在色彩表現上後者的磚紅色略為討喜。（拍攝狀態：兩者皆開啟 AI 與自動 HDR）

微距攝影

▲Redmi Note 8 Pro（上）與 realme XT （下）兩者皆具有微距攝影功能，但也因為兩者的微距鏡頭都僅有 200 萬像素，因此在更細微的細節表現都無法展現得更突出。（拍攝狀態：兩者皆開啟 AI 與自動 HDR）

夜間第一組

▲Redmi Note 8 Pro（上）與 realme XT （下）在夜間拍攝時，兩者在複雜的光源下整的色溫處理略有不同，前者在細節部分則較為鬆散。（拍攝狀態：兩者預設皆開啟 AI 與自動 HDR）

夜間第二組

▲Redmi Note 8 Pro（上）與 realme XT （下）在夜間拍攝時，前者在細節部分則較為鬆散，後者在室內燈光過曝的部分修正較為明顯，但也因此在色彩上略有不自然感。（拍攝狀態：兩者預設皆開啟 AI 與自動 HDR）

夜間超低光第一組

▲Redmi Note 8 Pro（上）與 realme XT （下）在使用夜景模式拍攝時，前者在細節部分處理得較為銳利清晰，後者的細節則是相對模糊，但在飽和度部分則是後者更為飽和，略有不自然感。（拍攝狀態：兩者預設皆開啟 AI 與自動 HDR）

夜間超低光第二組

▲Redmi Note 8 Pro（上）與 realme XT （下）在使用夜景模式拍攝時，前者在細節部分依然較後者清晰，後者的色彩相對較不自然。（拍攝狀態：兩者預設皆開啟 AI 與自動 HDR）

人像

▲Redmi Note 8 Pro（上）與 realme XT （下）拍攝人像背景虛化，在相同的光源與場景下，背景與主體分離準確率都相當不錯，但對於美化的效果與膚色的潤飾效果不同。（兩者皆使用人像模式）

▲Redmi Note 8 Pro（上）與 realme XT （下）依序在夜間混合光源的路邊使用進行背景虛化拍攝，在相同的光源與場景下，背景與主體分離準確率都相當不錯，除了對於美化的效果略有不同外，色溫的調整也有明顯不同。（兩者皆使用人像模式）

【短評】

Redmi Note 8 Pro 與 realme XT 不但在鏡頭數量與硬體規格上相當接近，在拍照效果上也互有領先，相信兩款手機都會有人喜歡，但 realme XT 是在此價格帶中少見具螢幕指紋辨識的功能，若無特別的品牌喜好，想要以最超值的價格買到一款具有最新技術的機種，realme XT 自然是相當不錯的選擇之一，但價格比起 Redmi Note 8 Pro 高出一些。

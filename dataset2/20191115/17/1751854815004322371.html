I use a Redmi Note 6 Pro and still check daily if Xiaomi has pushed the new MIUI 11 update to the model. Surprisingly, users of the Redmi 4 in India have reportedly started receiving MIUI 11 update and have even shared evidence online. Apart from the fact that it is MIUI 11, there is nothing spectacular to cheer about this update.

The screenshot shared by a Redmi 4 user shows that the new update comes with the version number MIUI 11.0.2.0.NAMMIXM. It also shows the update file is 483MB large. Interestingly, this new update doesn’t bring a change in the Android version as the Redmi 4 still runs the outdated Android 7.1.2 (Nougat). It also brings along the October 2018 Android security patch that Google released more than a year ago.

The changelog shared by some users on the MIUI forum shows the MIUI 11 brings improved touch controls and refined colour usage to deliver a fresh experience. The update also carries sound effects influenced by nature. The update is also expected to enhance the user experience by offering a minimalistic design. Other features include an always-on lock screen and Dynamic Video Wallpapers.

GitHub запустил проект Security Lab в защиту опенсорсного кода

15 Nov 2019, 17:26 by Александра Кукуть

GitHub запустил новый проект Security Lab — он объединит исследователей и компании в сфере безопасности, которые разделяют убеждённость, что безопасность открытого ПО важна для всех, пишет The Next Web.

Партнёрами Security Lab уже стали Google, Microsoft, Mozilla, Oracle, Uber и HackerOne.

Помимо этого, GitHub открыл исходный код инструмента для анализа и поиска уязвимостей в коде CodeQL от Semmle, который купил в сентябре.

Также GitHub запустил Advisory Database — общедоступную базу рекомендаций, созданных на GitHub.

Недавно сервис поделился инициативой по сохранению кода открытых проектов, таких как Flutter и TensorFlow, «на 1000 лет». Для этого GitHub собирается создать хранилище Arctic Code Vault на полярном норвежском архипелаге Шпицберген.

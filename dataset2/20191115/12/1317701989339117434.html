Microsoft Edge Dev 80.0.328.4 behebt viele Fehler, hat aber auch bekannte Probleme

15 Nov 2019, 12:59 by moinmoin

Das Edge-Team hat den Microsoft Edge Dev aktualisiert auf die Version 80.0.328.4 Neue Funktionen sind soweit nicht hinzugekommen aber viele Fehler wurden behoben. Die Liste der Änderungen ist sicherlich auch wie immer für die Canary Nutzer interessant.

  • Ein Absturz beim Start wurde behoben.
  • Es wurde ein Problem behoben, bei dem das Schließen einer Registerkarte manchmal zum Absturz des Browsers führte.
  • Ein Problem wurde behoben, bei dem die Installation von Erweiterungen manchmal zu einem Browserabsturz führte.
  • Es wurde ein Problem behoben, bei dem die Navigation zu bestimmten Websites den Browser zum Absturz brachte.
  • Beim Schließen des Edge stürzt das Edge ab, so dass die Eingabeaufforderung Restore Tabs beim Start unerwartet erscheint.
  • Es wurde ein Problem behoben, bei dem nur das erste Edge-Fenster, das in einer Sitzung geöffnet wurde, ganz weiß und ohne Benutzeroberfläche war.
  • Es wurde ein Problem behoben, bei dem Edge auf dem Mac hängt, wenn ein Benutzer auf der Seite Einstellungen auf die Schaltfläche Neustart klickt, um ein Update zu installieren.
  • Ein Problem wurde behoben, bei dem Favoriten nicht richtig zwischen den Maschinen synchronisiert wurden.
  • Einige Fehler wurden behoben, bei denen Videos, die durch DRM geschützt sind, z.B. auf Netflix, nicht abgespielt wurden.
  • Es wurde ein Problem behoben, bei dem das Dialogfeld Teilen nicht angezeigt wurde.
  • Es wurde ein Problem behoben, bei dem das Festnageln einer Website an die Taskleiste fehlschlug.
  • Reduzierte Anzahl von Synchronisationsfehlern führt zu einer Aufforderung zur Benutzerinteraktion.
  • Die Zuverlässigkeit von Screenshots wurde verbessert, so dass sie beim Senden von Feedback nicht mehr ganz schwarz erscheinen.

Und weiter geht es

  • Es wurde ein Problem behoben, bei dem Erweiterungen, die aus dem Chrome Web Store installiert wurden, nicht automatisch aktualisiert wurden.
  • Beim Start des Browsers wurden die Fenster von Application Guard wieder geöffnet, auch wenn die Startup-Einstellungen nicht auf die Wiederherstellung der vorherigen Sitzung eingestellt waren.
  • Es gab ein Problem, bei dem das Taskleistensymbol für Edge bei einigen Benutzern nicht auf das neue Symbol aktualisiert wurde.
  • Es wurde ein Problem behoben, bei dem nach einem Edge-Update manchmal zwei Edge-Symbole auf der Taskleiste angezeigt wurden.
  • Ein Problem wurde behoben, bei dem einige UI-Bilder auf dem Mac schwarze Hintergründe hatten.
  • Bei einigen Benutzern wurde ein Problem behoben, bei dem die Erfahrung mit dem ersten Ausführen auf jeder neuen Registerkarte angezeigt wurde.
  • Es wurde ein Problem behoben, bei dem Benutzer in bestimmten Ländern, die nicht mit einem Arbeits- oder Schulkonto bei Windows angemeldet sind, immer noch nicht in der Lage waren, „Arbeit“-Profile zu entfernen, die automatisch zu Edge hinzugefügt wurden, selbst nachdem eine frühere Lösung für dieses Problem veröffentlicht wurde.
  • Beim Ausfüllen von Kreditkarteninformationen auf bestimmten Websites wurde ein Problem behoben, bei dem die Aufforderung zum Speichern des Passworts unerwartet angezeigt wurde.
  • Es wurde ein Problem behoben, bei dem das Menü …. abgeschnitten angezeigt wurde.
  • Es gab ein Problem, bei dem Edge nicht richtig mit anderen Anwendungen interagiert, die versuchen, Dialoge im Vordergrund anzuzeigen.
  • Es wurde ein Problem behoben, bei dem die Verwendung der Mac Touch Bar zum Wechseln von Tabs manchmal nicht richtig funktioniert, wenn die Tabs Videos wiedergeben.
  • Das Problem, bei dem der Zoom in der Leseansicht auf dem Mac ausgelöst wurde, obwohl er nicht ausgeführt werden sollte, wurde behoben.
  • Es wurde ein Problem behoben, bei dem neue Datentypen für die Synchronisierung verfügbar waren, die aber standardmäßig nicht aktiviert sind.
  • Ein Fehler wurde behoben, bei dem die Seite zur Anpassung der Synchronisierung während der ersten Ausführung die Synchronisierung für die ausgewählten Elemente nicht ordnungsgemäß aktiviert hat.
  • Es wurde ein Problem behoben, bei dem die Aktivierung der Synchronisierung während des ersten Laufs keine Synchronisierung für bestimmte Datentypen ermöglichte.
  • Das Verhalten beim Import von Favoriten auf Mac wurde verbessert.
  • Die Seite mit den Synchronisierungseinstellungen wurde aktualisiert, um weitere Datentypen anzuzeigen, die in Zukunft verfügbar sein werden, jedoch mit deaktivierten Schaltern, bis dieser Datentyp verfügbar ist.
  • Es wurde ein Problem behoben, bei dem auf der Einstellungsseite zwei Passwörter umgeschaltet wurden und keine Favoriten umgeschaltet wurden.
  • Es wurde ein Problem behoben, bei dem die Schaltfläche Neustart auf der Seite Spracheinstellungen auf Mac immer sichtbar war.
  • Es wurde ein Problem behoben, bei dem das Importieren von Browserdaten aus anderen Chrome-Profilen als dem Standardprofil während des Importvorgangs einen falschen Profilnamen anzeigt.
  • Es wurde ein Problem behoben, bei dem die Einstellung zum Anzeigen der Favoritenleiste nicht korrekt aus Chrome importiert wurde.
  • Es wurde verbessert, welche Bilder beim Speichern bestimmter Websites in einer Sammlung verwendet werden.
  • Es wurde die Möglichkeit zugefügt, alle Sammlungselemente in die Zwischenablage zu kopieren.

Die bekannten Probleme und nicht mal wenige

  • Es gibt einige Probleme, bei denen Benutzer mit mehreren Audioausgabegeräten manchmal keinen Sound von Edge erhalten. In einem Fall wird Edge im Windows Volume Mixer stummgeschaltet und durch das Deaktivieren von Edge wird es behoben. In einem anderen Fall behebt ein Neustart des Browsers diesen Fehler.
  • Bei bestimmten Zoomstufen ist eine deutliche Linie zwischen der Browseroberfläche und den Webinhalten erkennbar.
  • Letzten Monat erhielten einige Benutzer ein Arbeits-Konto, das automatisch dem Browser hinzugefügt wurde und nicht entfernbar war. Obwohl wir vor kurzem die Möglichkeit für einige Benutzer aktiviert haben, dieses Konto zu entfernen, gibt es immer noch ein Problem, bei dem Benutzer, die mit einem Arbeits- oder Schulkonto bei Windows angemeldet sind, dieses Konto möglicherweise nicht aus dem Browser entfernen können.
  • Favoriten, die so eingestellt sind, dass sie nur das Symbol anzeigen, können bei aktivierter Favoriten-Synchronisation wieder sowohl das Symbol als auch den Text anzeigen.
  • Wenn Sie auf einen Link auf einem virtuellen Desktop klicken, wird derzeit eine neue Registerkarte in einem Fenster auf einem anderen virtuellen Desktop geöffnet, wenn auf dem aktuellen Desktop kein Fenster geöffnet ist, aber auf einem anderen. Dies ist eine Regression des bisherigen Verhaltens, die ein neues Fenster auf dem aktuellen Desktop geöffnet hat.
  • Jumplist-Einträge sind bei einigen Benutzern nicht konsistent zwischen dem Startmenü und der Taskleiste. Wir glauben, dass dies darauf zurückzuführen ist, dass die Verknüpfung im Startmenü nach einem Edge-Update nicht richtig migriert wurde und an einer Lösung arbeitet. – Außerdem gibt es nach dem Update für das neue Symbol noch Stellen im Startmenü, z.B. bei der Suche, die das alte Symbol anzeigen. Andere Stellen wie die Taskleiste können repariert werden, indem man die Edge-Verknüpfungen, die dort bereits vorhanden sind, löst und dann neu fixiert, wenn sie nicht durch die Updates letzte Woche repariert wurden.
  • Minimierte Windows werden manchmal nicht richtig wiederhergestellt, wenn der Browser die vorherige Sitzung aufgrund eines Updates, eines Absturzes usw. wiederherstellt. Stellen Sie als Workaround sicher, dass keine Fenster minimiert sind, bevor Sie den Browser neu starten.
  • Manchmal scheint der Browser nicht auf Benutzereingaben zu reagieren (das Klicken oder Scrollen auf Webseiten führt zu nichts, das Bewegen mit der Maus über die Benutzeroberfläche ändert nichts), aber das Klicken auf bestimmte Schaltflächen funktioniert immer noch (wie das …. Menü). Die Ursache dafür ist ein Fehler im GPU-Prozess, und das Öffnen des Browser-Taskmanagers (Rechtsklick in der Nähe des Fensters Minimieren/Maximieren/Schließen oder Drücken von shift+esc auf der Tastatur) öffnet ein Fenster, das es Ihnen ermöglicht, den GPU-Prozess zu beenden, wodurch das Problem behoben wird.

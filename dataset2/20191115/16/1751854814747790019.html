After pushing stable MIUI 11 update for Redmi Note 8 Pro in China last week, Xiaomi has started rolling out the same update to the Indian version of the devices. Multiple users on the Xiaomi forums have reported that they have received the MIUI 11.0.1.0 stable update on the Redmi Note 8 Pro models. This release is in fact ahead of the scheduled late December MIUI 11 rollout for the model in India.

Despite being rolled out as MIUI 11 stable update for the Redmi Note 8 Pro, it’s likely that this is a test stable build being sent out to a limited number of users. There were a number of users on the forums who didn’t get the update, so it’s possible that many users will have to wait for a week or two to get the final build.

Note that this MIUI 11 update is still based on Android 10. But it brings interesting MIUI 11 features to the device, including a redesigned UI and productivity features. You can read the entire list of MIUI 11 features from here.

The Redmi Note 8 Pro features the MediaTek Helio G90T gaming processor with 6GB / 8GB of RAM and 64GB / 128GB storage. The quad-camera setup at the back comes with an f/1.7 aperture 64-megapixel ISOCELL Bright GW1 as the primary lens, an ultrawide lens of 8-megapixel, a 2-megapixel macro lens with 2cm focal length and a depth sensor of 2-megapixel.

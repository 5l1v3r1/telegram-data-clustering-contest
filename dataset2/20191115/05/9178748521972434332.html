Microsoft неожиданно обновила «мёртвую» Windows Mobile

15 Nov 2019, 05:39 by Руслан Фадеев

Несмотря на объявленную ранее дату «похорон» мобильной версии Windows, компания Microsoft выпустила обновление программного обеспечения для смартфонов под управлением завершающей свой жизненный цикл операционной системы. Апдейт затронул практически все ключевые элементы прошивки.

Новый патч под кодовым номером KB4522811 включает «заплатки» для устранения критических уязвимостей в операционной системе Windows 10 Mobile и некоторых системных приложениях. Обновление затрагивает службы Windows App Platform, Microsoft Graphics Component, Windows Input and Composition, Windows Fundamentals, Windows Cryptography и Windows Virtualization, браузер Microsoft Edge, а также другие сервисы.

Вероятнее всего, апдейт станет последним для операционной системы — поддержка Windows 10 Mobile заканчивается 10 декабря текущего года. После этого срока компания прекратит выпуск апдейтов системы безопасности и регулярных пакетов обновлений мобильной ОС. Ранее Microsoft сообщала, что магазин приложений для Windows-смартфонов продолжит функционировать и по истечении указанного срока.

Windows 10 1909 Funktionen, die entfernt wurden oder nicht weiterentwickelt werden

13 Nov 2019, 11:41 by moinmoin

Auch wenn die Windows 10 1909 nur ein kleines Update von der Windows 10 1903 auf die 1909 ist, gegenüber den üblichen Funktionsupdates wurden auch hier weitere Funktionen entfernt, bzw. werden nicht länger entwickelt werden.

Die Entwicklung der Taskleisten-Synchronisierung (Position oder ausgeblendet etc.) wurde schon mit der Windows 10 1903 eingestellt. Nun wurde diese Funktion entfernt.

Auch das Netzwerkprotokoll zur Namensauflösung – Peer Name Resolution Protocol-Clouddienst (PNRP) wurde schon in der Windows 10 1809 entfernt. Jetzt hat man nun auch die APIs für das Protokoll entfernt, bzw. will es nun abschließen.

Funktionen, die nicht mehr weiterentwickelt werden

  • Hyper-V vSwitch im LBFO-Modus: In einem zukünftigen Release wird der Hyper-V vSwitch nicht mehr in der Lage sein, an ein LBFO-Team gebunden zu sein. Stattdessen kann es über Switch Embedded Teaming (SET) eingebunden werden.
  • Tab Language Community im Feedback Hub: Dieser Tab wurde aus dem Feedback-Hub entfernt
  • Kontakte in der Taskleiste: Dies wird nicht mehr weiterentwickelt und wird mit einem späteren Update entfernt
  • TFS1/TFS2 IME: Text Services Framework TSF1 und TSF2 IME werden in einem zukünftigen Release durch TSF3 IME ersetzt. Die TSF IME sind Windows-Komponenten, die Sie hinzufügen können, um die Eingabe von Text für die Sprachen Japanisch, Vereinfachtes Chinesisch, Traditionelles Chinesisch und Koreanisch zu ermöglichen.
  • Package State Roaming (PSR): PSR wird in einem zukünftigen Update entfernt. PSR ermöglicht es Nicht-Microsoft-Entwicklern, auf Roaming-Daten auf Geräten zuzugreifen, so dass Entwickler von UWP-Anwendungen Daten in Windows schreiben und mit anderen Windows-Instanzen für diesen Benutzer synchronisieren können. Der empfohlene Ersatz für PSR ist Azure App Service. Azure App Service wird bereits unterstützt, ist gut dokumentiert, zuverlässig und unterstützt plattformübergreifende Szenarien wie iOS, Android und Web.

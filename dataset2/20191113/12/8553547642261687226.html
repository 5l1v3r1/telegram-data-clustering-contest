Ноябрьское обновление Windows 10 доступно для загрузки

13 Nov 2019, 12:21

Microsoft начала распространение обновления Windows 10 November 2019 Update, также известного под кодовым названием 19H2. Это самое маленькое обновление для Windows 10 с того времени как компания перешла к практике выпуска их каждые полгода. Отмечается, что оно содержит набор исправлений ошибок, улучшения производительности и несколько новых бизнес-функций.

Ноябрьское обновление принесет несколько улучшений производительности. Некоторые системы получат улучшение времени автономной работы и усовершенствованное планирование ресурсов процессора.

File Explorer получил новые функции поиска. Когда вы введете запрос, вы увидите выпадающее меню со списком предлагаемых файлов. Он также будет искать файлы в учетной записи OneDrive в Интернете, а не только файлы на локальном ПК. Также можно будет щелкнуть правой кнопкой мыши на результате поиска, чтобы открыть местоположение файла.

Ранее на экране блокировки Windows позволяла использовать только встроенный голосовой поощник Cortana. В обновлении система позволит добавлять другие голосовые помощники, такие как Amazon Alexa. Эта функция должна начать работать автоматически после активации со стороны провайдеров голосовых помощников.

Приложение календаря Windows 10 получило свои улучшения. Теперь вы можете создавать события прямо из панели задач. Просто щелкните время на панели задач, чтобы открыть календарь и создать новое событие.

Также отмечается улучшенное управление уведомлениями, новые функции меню «Пуск» и обновления для корпоративной среды.

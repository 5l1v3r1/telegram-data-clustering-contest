The discount is now live on the Argos website but shoppers can use a double discount to get 20% off and a further £15 saving on any toys on the brand's website

12 Nov 2019, 13:06 by James Rodger, Stacey Dutton, https://www.facebook.com/jamesrodgerjournalist

Argos shoppers will be delighted to realise they can get money off presents before Christmas - thanks to a nifty code.

The high street giant, which operates stores in Birmingham, has released a code to get 20% off toys.

Shoppers, customers and consumers can take advantage of the deals, discounts and savings.

These include Lego, Nerf and Barbie - but shoppers can save a lot more using one online trick.

The discount is now live on the Argos website but shoppers can use a double discount to get 20% off and a further £15 saving on any toys on the brand's website.

Argos has launched a  20% off toys  for shoppers already with code  TOYS20.  Entering the code at checkout will bring the price down by 20% on toys ranging from a  £60 Barbie Home  set, which would now cost just £48.

Other discounts include a Lego Advent Calendar building set for £17 - now down to £13.60 using the code.

However, a deal offering an extra £15 off means the Barbie House is now an incredible £33, while the Lego Advent Calendar is now completely free using the two discount deals.

The extra deal comes from deals website Topcashback, which is offering shoppers the chance to earn £15 back if they spend £15 or more on the Argos website.

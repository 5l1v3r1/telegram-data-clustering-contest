.NET Framework Reparatur Tool wurde aktualisiert

12 Nov 2019, 17:01 by moinmoin

Kurz notiert: Microsoft hat sein Repair Tool netfxrepairtool.exe für .NET Framework auf die Version 4.8.04072.0 Release 1.4 aktualisiert. Neu in dieser Version ist, dass nun die Unterstützung für .NET Framework 4.6.2 bis .NET Framework 4.8 vorhanden ist.

Das Tool kann bei Problemen direkt gestartet werden, per Command Line oder auch Offline genutzt werden. Die Befehle für die Eingabeaufforderung stehen auf der Seite gut detailliert. Die Offline Installer für .NET Framework sind dort auch.

Info und Download:

  • support.microsoft.com/microsoft-net-framework-repair-tool
  • Download: aka.ms/DotnetRepairTool

.NET Framework Offline Installer

  • .NET Framework 4.7 NDP47-KB3186497-x86-x64-AllOS-ENU.exe

  • .NET Framework 3.5 SP1, 3.0 SP2, 2.0 SP2 dotnetfx35.exe

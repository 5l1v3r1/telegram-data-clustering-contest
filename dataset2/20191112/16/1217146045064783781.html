Adobe Patches Critical Remote Code Execution Bugs in Illustrator

12 Nov 2019, 16:22 by Sergiu Gatlan

Adobe released security updates to address security issues that could allow attackers to execute malicious code remotely, elevate privileges, and gain unauthorized access to information on systems running unpatched Illustrator, Animate CC, Bridge CC, and Media Encoder versions.

Of particular interest are the two critical memory corruption vulnerabilities that lead to remote code execution discovered in the Adobe Illustator software and tracked as CVE-2019-8247 and CVE-2019-8248.

Since the program is widely used and these security issues could enable attackers to execute code on vulnerable machines, Adobe strongly advises users to update as soon as possible to lock potential exploitation attempts.

The two RCE vulnerabilities found to affect Adobe Illustrator were reported by Kushal Arvind Shah of Fortinet's FortiGuard Labs, while an additional privilege escalation issue rated as important severity and tracked as CVE-2019-7962 was discovered and reported by Yongjun Liu of nsfocus security team.

Below are today's Adobe security updates:

APSB19-34 Security update available for Adobe Animate CC

Adobe has released a security update for Adobe Animate CC that resolves one important severity vulnerability affecting the Windows releases.

The APSB19-34 update addresses an insecure library loading (DLL hijacking) vulnerability that could lead to privilege escalation,.

The vulnerability tracked as CVE-2019-7960 was fixed in Adobe Animate CC 20.0 and was reported by Yongjun Liu of nsfocus security team.

APSB19-36 Security update available for Adobe Illustrator

Adobe has released a security update for Adobe Illustrator that resolves several important and critical severity vulnerabilities affecting the Windows releases.

The APSB19-36 update fixes critical and important vulnerabilities which could lead to Privilege Escalation and Remote Code execution in the context of the current user.

The vulnerabilities were fixed in Illustrator CC 2019 24.0.

APSB19-52 Security update available for Adobe Media Encoder

Adobe has released a security update for Adobe Media Encoder that resolves important and critical severity vulnerabilities affecting the Windows and macOS releases.

The APSB19-52 update addresses multiple file parsing vulnerabilities. Successful exploitation of these issues could lead to information disclosure in the context of the current user and arbitrary code execution.

The vulnerabilities were fixed in Adobe Media Encoder 14.0, and were reported by Mat Powell of Trend Micro Zero Day Initiative (CVE-2019-8241, CVE-2019-8243, CVE-2019-8244) , Francis Provencher working with Trend Micro Zero Day Initiative (CVE-2019-8242), and Wen Guang Jiao of Qihoo 360 Core Security (CVE-2019-8246).

APSB19-53 Security update available for Adobe Bridge CC

Adobe has released a security update for Adobe Bridge CC that resolves two important severity vulnerabilities impacting the macOS and Windows releases.

The APSB19-53 update addresses multiple vulnerabilities that occur when parsing malformed SVG images and could lead to information disclosure in the context of current user upon successful exploitation.

The security issues were fixed in Adobe Bridge CC 10.0 and were reported by Wen Guang Jiao of Qihoo 360 Core Security.

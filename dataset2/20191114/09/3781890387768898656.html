ซอฟท์เดบู จัดงาน "Cloudflare Red Rock Carpet Exclusive Party"

14 Nov 2019, 09:40

Softde'but จับมือ Cloudflare จัดงาน "Cloudflare Red Rock Carpet Exclusive Party 2019” สำหรับตัวแทนจำหน่ายในประเทศไทย

คุณพีรวีท์ จรัสสิริกุลชัย กรรมการผู้จัดการ บริษัท ซอฟท์เดบู จำกัด ได้แสดงวิสัยทัศน์ในเรื่องของ Cloud Technology ว่ามีบทบาทมากขึ้นทั้งด้านธุรกิจและสังคมทั่วโลกที่ในยุคปัจจุบัน “เทคโนโลยี” กลายเป็นกำลังสำคัญในการเปลี่ยนแปลงการดำรงชีวิตของผู้คนและมีการวิวัฒนาการอย่างไม่รู้จบ ทำให้เกิด Disruptive Technology เทคโนโลยีเกิดใหม่ส่งผลกระทบต่อธุรกิจที่ใช้เทคโนโลยีรูปแบบเดิมให้ค่อยๆ หายไป และ Cloudflare สินค้าจากประเทศสหรัฐอเมริกา ผู้ให้บริการ CDN ที่มี Server หลายแห่งทั่วโลก ถือเป็นส่วนสำคัญในการเปลี่ยนแปลง Disruptive Technology 

เนื่องจาก Cloudflare นั้นใช้เทคโนโลยีในการ Caching ข้อมูล เพื่อเพิ่มประสิทธิภาพการเรียกข้อมูลให้มีความรวดเร็วมากยิ่งขึ้น ซึ่งไม่ได้นำมาใช้กับ Web Application เพียงอย่างเดียวเท่านั้น ในอนาคต Cloudflare จะออกผลิตภัณฑ์ที่ตอบโจทย์การใช้งานและครอบคลุมในทุกๆ เทคโนโลยี ไม่ว่าจะเป็น IoT, AR, VR, Blockchain และ Bigdata รวมถึงเทคโนโลยีอื่นๆ ที่กำลังจะมาในอนาคตด้วยเช่นกัน
 

โดย Cloudflare มุ่งเน้น Content Delivery การส่งข้อมูลถึงปลายทางได้อย่างรวดเร็ว เช่น รูปภาพ วีดีโอและไฟล์เสียงต่างๆ อีกทั้งยังให้ความสำคัญในเรื่องความปลอดภัย โดยมี Web Application Firewall (WAF) ที่จะช่วยปกป้องข้อมูลในรูปแบบ Security Cloud Service ซึ่งลดภาระค่าใช้จ่ายด้านอุปกรณ์เทคโนโลยีแบบเก่าและช่วยป้องกันภัยคุกคามได้อย่างมีประสิทธิภาพมากขึ้น
 

นอกจากนี้ ยังมีผู้เชี่ยวชาญจาก Cloudflare มาร่วมให้ความรู้และข้อมูลที่เป็นประโยชน์ ใน Solutions ต่างๆ อาทิ DNS, DDos, SSL/TLS, Firewall,  Rate Limiting, CDN, WAF, Argo Smart Routing และ Load Balancing รวมถึงเทคนิคกุญแจสำคัญในการขาย Cloudflare ให้แก่พาร์ทเนอร์ โดยข้อดีของ Cloudflare คือ ความปลอดภัย ความรวดเร็วและความมีเสถียรภาพ ซึ่งสามารถป้องกันการโจมตีจาก DDos Attacks ป้องกันการละเมิดข้อมูลลูกค้าและการโจมตีจากบอท พร้อมทั้งช่วยเพิ่มความรวดเร็วและประสิทธิภาพในการใช้งานเว็บไซต์ ไม่ว่าจะเป็นเว็บไซต์ที่มีเนื้อหา Static content หรือ Dynamic content  ไม่ใช่เพียงบริการ CDN Cloudflare ยังมีบริการ Domain name server (DNS) ที่ใช้งานง่ายไม่ว่าลูกค้าจะใช้แพลตฟอร์มใดก็ตาม เพียงเปลี่ยนหมายเลข IP มาชี้ที่ Cloudflare ก็สามารถผ่าน Traffic มายังเครื่อง Server ของ Cloudflare ได้โดยไม่ต้องมีการเปลี่ยนแปลงใดๆ

ภายในงานยังมีกิจกรรมพิเศษ Lucky Draw หาผู้โชคดีและกิจกรรมมากมายให้ผู้ร่วมงานได้ร่วมสนุก และร่วมรับประทานอาหาร พร้อมชม Mini-Concert สุดพิเศษ โดยบริษัท ซอฟท์เดบู จำกัด ได้เตรียมทีมงานที่มีความรู้ความสามารถในการช่วย Support ลูกค้าที่ต้องการการดูแลเป็นภาษาไทย โดยจะมีการอำนวยความสะดวกทั้งให้คำปรึกษา แนะนำวิธีการติดตั้ง หรือการให้บริการเต็มรูปแบบ ทั้งจัดจำหน่ายและบริการหลังขาย (SAAS) 24 ชั่วโมง นอกจากนี้ยังมีการเตรียมการขายเต็มรูปแบบ ทั้งเปิดรับตัวแทนจำหน่ายเพื่อรุกตลาด กลุ่มลูกค้าที่ต้องการ ซอฟท์แวร์รักษาความปลอดภัย และช่วยเรื่องการทำ CDN

Google Analytics расширил лимит на количество потоков данных для ресурсов App + Web

14 Nov 2019, 09:15 by Анна Бондарь

Google Analytics позволил создавать до 50 потоков данных для ресурсов типа «Приложение и сайт» (App + Web), используя любую комбинацию потоков данных для сайта или приложения.

Ранее  в ресурс можно было добавить только один веб-поток.

A sua rede Wi-Fi ainda usa chave WEP? Vai deixar de funcionar no Windows 10

14 Nov 2019, 17:02 by Pedro Pinto

O Windows 10 é indiscutivelmente o melhor sistema operativo da atualidade. Além das funcionalidades, este sistema tem apostado forte na segurança dos utilizadores. Uma das mais recentes novidades nesta área da segurança será a perda de suporte de redes Wi-Fi que ainda usem chaves WEP.

Se a sua rede Wi-Fi ainda usa como segurança uma chave WEP está na altura de evoluir.

Actualmente ainda existem muitas redes wireless a usar chaves WEP (Wired Equivalent Privacy) para garantir a segurança. Este protocolo de cifra foi criado em 1997 e “crackado” em 2011, sendo hoje totalmente vulnerável.

Neste sentido, a substituição da cifra WEP pela cifra WPA2 ou WPA3 é fundamental, pois temos uma melhor criptografia dos dados. A Microsoft recomenda a utilização da cifra AES.

Windows 10 sem suporte para redes WI-FI com chave WEP

Desde maio deste ano que quem tem Windows 10 e acede a uma rede Wi-Fi com chave WEP que recebe uma mensagem de “alerta”. No entanto, a Microsoft fez agora saber que em próximas versões as redes WI-FI com chave WEP não serão suportadas. Isto que dizer que se tentar ligar a uma rede Wi-Fi com chave WEP o sistema simplesmente não o permitirá.

Since the 1903 release, a warning message has appeared when connecting to Wi-Fi networks secured with WEP or TKIP (which are not as secure as those using WPA2 or WPA3). In a future release, any connection to a Wi-Fi network using these old ciphers will be disallowed. Wi-Fi routers should be updated to use AES ciphers, available with WPA2 or WPA3.

Não está claro quando a Microsoft irá deixar de oferecer suporte para redes Wi-Fi com chaves WEP. No entanto, o aviso está dado e nesse sentido é preciso evoluir.

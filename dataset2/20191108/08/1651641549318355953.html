Microsoft To Do for Android gets updated with new features

08 Nov 2019, 08:09 by Raka

Microsoft’s task management app for Android, famously known as Microsoft To Do has received an update. The update brings a couple of new features and a lot of bug fixes.

With the latest update installed, you will be able to see things like due date, which lists a task belong to and also, whether you’ve added it to My Day. Moreover, if you switch to Dark Mode, you’ll notice a change in the colors of the icons. Taking the app to Version 2.4.136, the update brings plenty of bug fixes and performance improvements. You can see the official changelog below.

Changelog

   •  We’ll now show you more metadata in the widget, so you can see the due date, which list it belongs to and whether you’ve added it to My Day.
   •  We’ve changed the colors for the icons when you’re in dark mode.
   •  We made a lot of accessibility fixes and small improvements.

You may soon be able to run Windows 10 IoT Core on a Calculator

08 Nov 2019, 17:00 by Raka

Windows 10 IoT Core is for the smallest embedded systems like washing machine, automatic door lock, and of course, the Rasberry Pi, which is popularly known as the “The credit-card-sized computer.”

But aside from Rasberry Pi, you might soon be able to run Windows Core IoT Core on your Calculator. A developer(known as @imbushuo on Twitter) seems to have managed to successfully install Windows 10 IoT Core on an HP Prime Graphing Calculator.

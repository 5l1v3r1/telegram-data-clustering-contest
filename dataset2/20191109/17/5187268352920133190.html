Esconda-se na Internet! Use a rede TOR e o Privoxy

09 Nov 2019, 17:02 by Pedro Pinto

Aceder à rede Tor é hoje uma tarefa bastante simples. Para isso basta que usem o Tor Browser, que inclui nativamente todas as configurações necessárias.

No entanto, quem pretender usar outros browsers pode sempre configurar um proxy na própria máquina. Saiba como podem instalar o Privoxy e configurar-lo para aceder à rede anónima.

Navegar de forma anónima não é uma tarefa fácil especialmente porque tudo aquilo que fazemos fica registado. No entanto, é possível “proteger” a nossa privacidade online através de ferramentas adicionais que nos ajudam a esconder a nossa identidade..

Para este tutorial vamos usar o Linux Kali como distribuição Linux. Para instalar o Tor e o Privoxy devem usar o seguinte comando:

sudo apt-get install tor privoxy

sudo apt-get install tor privoxy

Depois de instalado, devem ir à configuração do Privoxy em /etc/privoxy/config e inserir a seguinte linha:

forward-socks5 / 127.0.0.1:9050 .

forward-socks5 / 127.0.0.1:9050 .

O próximo passo é iniciar o serviço tor e o privoxy

sudo service tor start sudo service privoxy start

sudo service tor start sudo service privoxy start

Como testar usando o Firefox?

Usando o browser Firefox, vamos agora dar indicações que as comunicações fluem via Proxy. Para isso vamos às Definições do Firefox e depois Preferences.

Em seguida, em Network Proxy escolham a opção Manual proxy configuration e definam:

  • HTTP Proxy: 127.0.0.1 8181
  • Socks Host (v5): 127.0.0.1 9050

Salvem a configuração e testem se está tudo operacional. Para isso basta que abram o browser novamente e acedam ao endereço https://check.torproject.org/. O resultado deverá ser idêntico ao seguinte.

Configurar o Privoxy para a rede Tor é relativamente simples. Este proxy tem outras funcionalidades bastante interessantes que podem ser analisadas e testadas através do ficheiro de configuração (/etc/privoxy/config). Se têm um Linux aí por perto, experimentem.

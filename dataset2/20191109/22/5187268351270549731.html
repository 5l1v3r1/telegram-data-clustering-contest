Office no Linux? Aprenda a instalar o fantástico Freeoffice

09 Nov 2019, 22:08 by Pedro Pinto

A suite de produção da Microsoft é indiscutivelmente a melhor e a mais completa que existe no mercado. No entanto, existem vários projetos, inspirados no produto da Microsoft, que têm vindo a evoluir significativamente. Um dos projetos mais conhecidos é o LibreOffice, mas há muitos outros como é o caso do Freeoffice.

Saiba como instalar o Freeoffice no Linux.

Chama-se FreeOffice e é uma suite de produtividade desenvolvida pela empresa alemã SoftMaker. Esta suite incluiu uma alternativa gratuita ao Word, Excel e Powerpoint e, além disso, todos os documentos criados nesta plataforma são também compatíveis com o Office da Microsoft.

É verdade que o Freeoffice não é tão completo que o Office da Microsoft. No entanto, para as tarefas “normais”/básicas, o FreeOffice chega na perfeição. Em termos de interface, o FreeOffice oferece uma boa UI, muito semelhante ao Office da Microsoft.

Como instalar o FreeOffice no Linux?

Para instalar o FreeOffice no Linux deve começar por aceder ao site e proceder ao download do pacote respetivo para a sua distribuição. Vamos, por exemplo, considerar que têm como distribuição Linux o Ubuntu. Para outras distribuições, basta que carreguem no link adequado.

  • Debian
  • Ubuntu
  • Linux Mint
  • openSUSE
  • Fedora

Para isso devem começar por obter o pacote e proceder à instalação no Ubuntu usando os seguintes comandos:

wget -c https://www.softmaker.net/down/softmaker-freeoffice-2018_934-01_amd64.deb -O softmaker-freeoffice.deb   sudo dpkg -i softmaker-freeoffice.deb sudo apt -f install

wget -c https://www.softmaker.net/down/softmaker-freeoffice-2018_934-01_amd64.deb -O softmaker-freeoffice.deb sudo dpkg -i softmaker-freeoffice.deb sudo apt -f install

O Freeoffice traz consigo uma ferramenta tipo Word (TextMaker), uma tipo Excel (PlanMaker) e também uma ferramenta para apresentações (Presentations).

Para quem se queixa que para o Linux não existem boas plataformas de produtividade experimentem este Freeoffice… vão ficar surpreendidos.

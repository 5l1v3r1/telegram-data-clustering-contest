Rockstar выпустила патч для предотвращения вылетов из Rockstar Games Launcher

09 Nov 2019, 09:33 by Vlad Tepes

Судя по комментариям, обновление решает проблему, актуальную для игроков в Red Dead Redemption 2.

Согласно описанию, патч 1.0.12.172 устраняет неисправность, из-за которой владельцы процессоров AMD сталкиваются с неожиданными вылетами из Rockstar Games Launcher.

О проблеме массово сообщали игроки в Red Dead Redemption 2, которая 5 ноября вышла на ПК. До выхода патча пользователям приходилось решать её через обновление BIOS, изменение файла подкачки и удаление всех папок в директории RDR 2.

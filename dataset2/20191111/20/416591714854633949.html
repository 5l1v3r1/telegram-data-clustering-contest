Facebook testa nova interface do aplicativo para Android

Principais alterações são na barra de navegação

11 Nov 2019, 20:07 by Matheus Luque

O Facebook costuma fazer mudanças no visual de seu aplicativo, como aconteceu em março desse ano. Agora, a rede social de Mark Zuckerberg está testando uma nova interface do app para Android.

Uma das principais alterações envolve o ícone do Facebook, um grande "F", na barra de navegação. Ele aparece ao lado dos atalhos para a tela de notificações e o feed de notícias, levando o usuário para a tela de menu que, atualmente, é acessada pelo ícone de "hambúrguer" - as três linhas horizontais.

Na imagem, é possível detectar as alterações com mais clareza. Há uma pequena mudança de posição dos tópicos e os ícones continuam os mesmos. Cada uma das seções ficou destacada em rótulos com fundo branco, facilitando o toque para seleção em tablets e smartphones. A nova interface do app também facilita o acesso ao seu perfil e à busca, na parte superior.

Segundo o Android Police, os testes estão sendo feitos com usuários na versão 246.0.0.49.121. Nem todos que possuem esta versão receberam o novo visual.

Mesmo que a mudança se torne oficial, a empresa de Mark Zuckerberg não deve descartar a FB5, como foi chamada a nova interface para iPhone, Android e web. Ela oferece mais velocidade e opção de modo noturno para o aplicativo.

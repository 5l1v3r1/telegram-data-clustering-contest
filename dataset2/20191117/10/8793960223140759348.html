Api di Pabrik Pengolahan Karet di Jambi Akhirnya Berhasil Dipadamkan

17 Nov 2019, 10:56 by Ferdi Almunanda

Jambi - Petugas pemadam kebakaran akhirnya dapat memadamkan kebakaran hebat yang melanda pabrik pengolahan karet di kawasan Desa Kubu Kandang, Kecamatan Pemayung, Kabupaten Batanghari, Jambi. Api berhasil dipadamkan setelah lebih dari 12 jam.

Pabrik pengolahan karet ini terbakar sejak Sabtu (16/11) malam sekitar pukul 21.00 WIB. Pabrik milik PT ABP (Aneka Bumi Pratama) diduga terbakar akibat arus pendek listrik yang terjadi.

Dari kebakaran itu pemadam kebakaran sempat mengerahkan 4 unit mobil Pemadaman dengan dibantu pihak BPBD. Petugas juga bahkan sempat kewalahan dalam upaya pemadaman tersebut karena pabrik keseluruhan berisakan pengolahan getah karet.

Kendati demikian, kobaran api itu tidak menyambar permukiman warga. Kebakaran dipabrik tersebut juga tidak sampai memakan korban jiwa.

"Korban jiwa tidak ada, karena saat itu karyawan pada libur. Sekarang api sudah dapat kita padamkan walau tadi agak sedikit kesulitan memadamkannya karena itu tempat pengolahan karet. Sekarang sudah padam apinya. Dan sedang proses pendinginan oleh petugas," kata Kepala Bidang Penanggulangan Kebakaran Dinas Damkar Kabupaten Batanghari Jambi, Amir Hamzah kepada detikcom, Minggu (17/11/2019).

Petugas juga sempat berupaya memadamkan api dengan menggunakan racun api. Hanya saja, api yang begitu besar dari dalam pabrik membuat petugas terpaksa menggunakan air deterjen untuk memadamkan api di pabrik itu hingga akhirnya dapat dipadamkan.

Pabrik yang terbakar itu kini juga terlihat rata setelah dilalap si jago merah. 12 jam lamanya petugas berupaya memadamkan api di pabrik itu. Petugas hingga kini juga belum dapat memastikan secara rinci berapa jumlah kerugian atas kejadian terbakarnya pabrik pengolahan karet tersebut.

Pabrik Pengolahan Karet di Jambi Kebakaran

17 Nov 2019, 02:28 by Ferdi Almunanda

Jambi - Kebakaran terjadi di salah satu pabrik pengolahan karet di kawasan Desa Kubu Kandang, Kecamatan Pemayung, Kabupaten Batanghari, Jambi. Diketahui, pabrik pengolahan ini milik PT ABP (Aneka Bumi Pratama).

"Info yang kita terima yang terbakar adalah tempat pengolahan karet di pabrik itu. Mess karyawan juga jauh dari lokasi kebakaran. Dan karyawan juga sedang libur," ujar Kepala Bidang Penanggulangan Kebakaran Dinas Damkar Kabupaten Batanghari Jambi, Amir Hamzah kepada detikcom, Minggu (17/11/2019) dini hari.

Api yang membesar di area pabrik itu membuat petugas pemadam kebakaran sempat kewalahan. Hingga saat ini kobaran api belum dapat dipadamkan. Petugas hingga kini masih berjibaku memadamkan api yang semakin membesar.

"Informasi yang kita terima kebakaran itu pada pukul 23.40WIB tadi. Api hingga saat ini masih berkobar. Armada kita kerahkan ada 4 unit plus dibantu armada dari Kabupaten Muaro Jambi dan BPBD," ujar Amir.

Amir juga belum dapat menyimpulkan secara pasti penyebab dari kebakaran pengolahan pabrik karet itu. Kobaran api saat ini juga sedang dipadamkan agar tidak menyebar ke objek lain ataupun kepermukiman warga.

"Saat ini personel kita masih berupaya melakukan pemadaman dan isolasi agar api tidak menyebar ke objek lain. Lokasi pabrik juga jauh dari permukiman warga," ujarnya.

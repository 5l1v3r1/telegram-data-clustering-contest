Sony Aibo 機械狗加入新 API 化身智能家居助理

17 Nov 2019, 14:10 by 藍骨

Sony 的機械狗 AIBO 重出江湖之後，新版本強調雲端服務以及可擴充能力，最近 Sony 就推出了新的 API 讓開發者可以為 AIBO 加入更多不同的互動功能，甚至化身智能家居助理。

Sony 的 aibo 開發者計劃正在邀請開發者製作可以與 aibo 互動的服務和應用，其中日本的日立（Hitachi）就已經開始研究家電如何與 aibo 互動，例如忘記關上雪櫃門，或者洗衣機程序完成的時候，可以通知 aibo 發出吠聲來提醒主人，另外也可以讓 aibo 充當智能助理幫忙控制家電，充滿不同的可能性。此外，aibo 也可以被訓練進行不同的動作，例如幫忙執拾等等。

MIUI 11 notifications’ sound effect changes with time
4 (80%) 11 votes

The stable version of MIUI 11 started rolling out in mid-October. While Xiaomi is preparing this update for more devices, it is also adding some new features. Earlier, there were reports that Xiaomi is preparing Focus mode, Vientiane screen custom image & Class schedule features for MIUI 11. However, these new features are still in the internal testing stage. In addition, the company also added a new drawer mode to the system desktop. Now, its latest announcement on Twitter shows that MIUI 11 notifications’ sound effect changes with time. Of course, the changes in notification sound depends on user settings. The company said on Twitter

“You’ve got a message from #MIUI11 Surprised! We are happy to announce that on MIUI11, notifications’ sound effect is changed by time. Tune-up to experience it now”.

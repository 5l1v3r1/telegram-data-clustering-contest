桌面版 Opera 65 推出   全新佈局方便用戶瀏覽

17 Nov 2019, 03:00 by 唐美鳳

Opera Software 日前推出全新版本的桌面瀏覽器 Opera 65，新版本主要針對 Tracker Blocker 作出更新，用戶能夠查看被 Opera 阻截的 Tracker。此外，網址欄的設計亦獲得更新，書籤欄和歷史記錄則設在側邊欄，都屬於全新的設計佈局。

Opera 65 桌面瀏覽器提供 Windows、macOS 和 GNU / Linux 版本。Opera 65 的 Tracker 阻截功能，讓用戶可以特色在瀏覽網站時，知道哪些 Tracker 正在追蹤用戶的數碼足跡。Opera Software 表示阻截 Tracker 可以令網站頁面載入速度提升約 20%，假如配合 Opera 的內置廣告阻截功能，速度更可以提升至 76%。

Opera 65 重新設計了網誌欄，提供更快的瀏覽體驗，網址欄將會為用戶提供瀏覽歷史建議，亦加入書籤和快捷鍵元素。Opera Software 表示更新外觀和佈局，包括將書籤欄和歷史記錄放在側邊欄，是回應用戶們的要求，讓他們在瀏覽時更加方便。

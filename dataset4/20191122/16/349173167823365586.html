Microsoft уже разрешили продавать Huawei массовое ПО, на очереди Google?

22 Nov 2019, 16:48 by Владимир Скрипин

Буквально на днях министерство торговли США выдало Huawei очередную (уже третью по счету) временную лицензию на торговлю с американскими компаниями сроком на 90 дней. Но сама по себе она, как и две предыдущие, только часть механизма — в период действия лицензии  американские компании должны получать отдельные разрешения на торговлю с Huawei. Подтверждая предварительные сведения о том, что механизм наконец заработал и министерство торговли США начало выдавать лицензии, Microsoft официально сообщила о получении соответствующего документа, разрешающего ей возобновить экспорт программного обеспечения для Huawei Technologies.

«20 ноября министерство торговли США удовлетворило заявку Microsoft на выдачу лицензии на экспорт для Huawei программного обеспечения для массового рынка. Мы благодарны департаменту за это», — сообщил представитель Microsoft в ответ на запрос Bloomberg.

И хотя формулировка звучит расплывчато, Microsoft продавала Huawei лицензии Windows и Office компании. Вполне вероятно, что Microsoft сможет хотя бы возобновить продажу лицензий Windows для Huawei. Ранее Huawei была вынуждена отложить выпуск нового ноутбука на платформе Windows, а Microsoft ненадолго прекратила продажи ноутбуков Huawei MateBook X Pro в своем интернет-магазине, поскольку могла продавать только экземпляры из старых партий, не подпадающие под санкции.

Это разрешение не менее важно для Microsoft, поскольку это возможность возобновить сотрудничество по Azure. Обе компании используют Microsoft Azure Stack — гибридное облачное решение, которое позволяет запускать сервисы Azure в собственном центре обработки данных организации. И работу Azure Stack обеспечивают серверы Huawei, сертифицированные Microsoft.

Ранее в министерстве торговли США сообщили, что по состоянию на 20 ноября было получено около 300 заявок и успели рассмотреть примерно половину, из которых утвердили только 50%. Какие еще компании, помимо Microsoft, получили разрешение на торговлю с Huawei — пока неизвестно. Но вполне вероятно, что Google в ближайшие дни тоже получит разрешение, после чего Huawei сможет использовать сервисы и приложения Google в своем новом флагмане Mate 30 Pro и будущих смартфонах.

Источник: Bloomberg

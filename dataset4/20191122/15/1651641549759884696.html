Facebook Beta app, with new features is now live on Microsoft Store

22 Nov 2019, 15:15 by Rahul

We’ve earlier reported that Facebook is working on some useful new features for its Facebook Beta app for Windows 10. These new features are now rolling out to those who are using the Facebook Beta app on their PC. The new Facebook Beta app update brings the following new features.

  • Ability to permanently delete sent messages
  • Introduction of new themes (dark theme, grey theme)
  • Ability to send files
  • Full-screen mode
  • Ability to hide chats
  • New icon
  • Updated emoticons

It worth noting that Facebook is rolling these new features in a phased manner, meaning not everyone will get the update today. If you don’t get it today, you’ll have to wait a few more days for the update to arrive.

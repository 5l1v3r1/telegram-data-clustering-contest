Google выплатит 1,5 млн долларов за взлом собственных смартфонов

22 Nov 2019, 21:21

За взлом смартфонов серии Pixel компания Google выплатит денежное вознаграждение.

Попытку взломать смартфон может каждый желающий, но это нужно сделать в рамках программы Android Security Rewards.

В программе Android Security Rewards сейчас участвуют Pixel 3, Pixel 3 XL, Pixel 3a, Pixel 3a XL и Pixel 4.

В зависимости от ошибки будет меняться размер выплаты, причем даже может быть выплачена премия в размере 50%.

Но для получения премии нужно будет продемонстрировать возможность взлома в определенных предварительных версиях «Андроид» для разработчиков.

Самое большое вознаграждение будет выплачено за обнаружение возможности взлома процессора безопасности Pixel Titan M.

Все подробности про программу Android Security Rewards можно узнать на официальном сайте Google.

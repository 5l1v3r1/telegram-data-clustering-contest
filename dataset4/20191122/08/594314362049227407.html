相信不少用家都有在家中安裝一個 IP Cam 的打算，尤其是對於家中有長者、小朋友或者寵物的用家，更加是有此需要。不過亦有用家擔心放置一個 IP Cam 在家中，萬一被駭客入侵可能會令私隱外洩，最近 Imou 就推出了 Ranger 2 A22EP 家居智能網絡攝影機，可以在不需要使用時一鍵隱藏鏡頭，應該可以幫到大家。

全高清解像度全方位監察

Imou Ranger 2 A22EP 智能安全網絡攝影機內建了可以拍攝 1080p 解像度的鏡頭，而且支援以先進 H.265 壓縮格式儲存，不但可減少網絡佔用頻寬，亦可有效降低儲存使用率。而且鏡頭可提供 93°(H), 48°(V), 115°(D) 視野，更內建可支援 0 度至 355 度水平、80 度垂直兼下轉 -5 度的旋轉雲台，不但視野夠廣闊，而且監視角度靈活，方便用家隨時隨地查看家中不同位置。另外，鏡頭造型的骰輕巧，機身大小只有 106.1 x 77.4 x 77.4mm，方便固定在不同位置，而且亦內置喇叭及咪高峰，方便用家可透過 Imou APP 隨時隨地與家人互動溝通或收聽現場聲音。此外，為家居增添新意，Imou 亦為 Ranger 2 A22EP 度身訂造專屬的矽膠頭套（需另購），造型可愛有趣，令鏡頭也可成為有趣可愛的家居裝飾。

Privacy Mask 一鍵隱藏鏡頭保障私隱

另外，Imou Ranger 2 A22EP 亦有一項貼心設計，就是設有一鍵隱藏鏡頭的功能，方便用家在回家後，以 Imou APP 一鍵即時隱藏鏡頭，這樣即使鏡頭被駭客入侵，駭客也不會容易即時開啟鏡頭，侵犯用家私隱。

各項智能功能主動防護家居

Imou Ranger 2 A22EP 既然稱為智能家居網絡攝影機，自然擁有不少智能功能。首先它內置了人形辨識演算法及異常聲響偵測警報，前者可識別場景中人形目標，而後者就可供用家設定不同聲音分貝進行現場偵測，當檢測到有人物闖入監看範圍，或者有異常聲響時，例如：嬰兒哭喊、寵物叫喚等，就會自動推送警報通知及觸發錄影功能，如有需要，例如有可疑人等入侵，更可設定攝影機主動發出警笛聲響以作出即時阻嚇。此外，它亦設有智能追蹤模式，當有移動物體進入監測範圍，鏡頭就會自動跟蹤拍攝，讓用家可捕捉到更多細節。值得一提的是，它亦配置了高性能紅外燈，夜視距離可達 10 米，即使在完全黑暗的環境中，也可以透過影片提供豐富細節。

安裝超容易 + 多樣化及高隱私度儲存功能

至於安裝方面，Imou Ranger 2 A22EP 亦十分簡單，只要透過 Imou APP 經 Wi-Fi 或 LAN 線，即可簡單 3 步完成安裝設定。此外，它亦內建了 microSD 記憶卡槽，最高可支援 256GB，用家可設定將影片或擷圖儲存在記憶卡中，亦可以儲存到 Imou Cloud 雲端錄影儲存服務。值得一提的是，這個雲端儲存服務具備高私隱度的特點，影片更可使用自定義密碼進行加密，為用家提供額外的安全保障。Imou Cloud 更提供 3 天、7 天或 30 天的巡環事件錄影雲端儲存套裝供用家選擇，用家可以月費或年費方式支付，十分靈活可靠。

Disponibili nuove notifiche server per gli abbonamenti su App Store

22 Nov 2019, 17:56

Apple ha annunciato che nuove notifiche da server a server per gli abbonamenti all’App Store sono ora disponibili per gli sviluppatori.

Le notifiche da server a server dell’App Store consentono di ricevere aggiornamenti in tempo reale sullo stato di un abbonamento, in modo da poter offrire esperienze personalizzate. Le nuove notifiche sono ora disponibili, e ci segnaleranno ad esempio quando un rinnovo dell’abbonamento non riesce a causa di un problema di fatturazione, o quando l’App Store risolve un problema di fatturazione. Inoltre, nella notifica è ora inclusa una ricevuta unificata contenente informazioni sulle ultime 100 transazioni di acquisti in-app.

Se avete già abilitato le notifiche da server a server in App Store Connect, riceverete automaticamente le nuove notifiche e la ricevuta unificata per gli abbonamenti.

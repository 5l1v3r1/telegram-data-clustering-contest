Яндекс рассказал, какие функции появятся в Турбо-страницах

22 Nov 2019, 14:48 by Марина Ибушева

Сегодня, 22 ноября, технологии Турбо исполнилось 2 года. На конференции Вебмастерская представители Яндекса рассказали о планах развития Турбо-страниц на ближайшее время.

Какие функции появятся в Турбо-страницах 

Новые возможности для брендинга:

  • передача RSS и собственная HTML-верстка;
  • фиксированная шапка меню с постоянным напоминанием о бренде. 

Планы по развитию рекламных форматов:

  • возможность размещения на страницах Interscroller с параллакс-эффектом. Такой блок повышает в среднем на 10% CPMV. Появится до конца года;
  • функциональность залипающих skicky-блоков; 
  • видеореклама формата Inpage и Instream для Турбостраниц. Будет доступна в ADFOX;
  • возможность размещать Inpage видеорекламу в ручной расстановке.

Планы по улучшению вовлечения аудитории:

  • функция подписки прямо на Турбо-страницах;
  • возможность использовать существующие виджеты и создавать свои собственные;
  • возможность публикации комментариев пользователей Яндекса. Данный функционал уже запущен. Подключить комментарии можно в Вебмастере. После этого зарегистрированные пользователи Яндекса смогут оставлять комментарии на страницах, получать уведомления об ответах. В рамках функциональности предусмотрена бесплатная модерация комментариев;
  • развитие механизмов социальности на Турбо-страницах – лайки и другое. 

Планы по развитию Турбо-страниц для интернет-магазинов включают в себя развитие по трем основным направлениям.

  • Рост конверсий внутри Турбо подразумевает внедрение новой функциональности: товарные рекомендации разных типов (в том числе персонализированные рекомендации с помощью методов машинного обучения), встроенный поиск с поисковыми подсказками, главные страницы для магазинов. 
  • Также появятся дополнительные настройки: расширенные возможности кастомизации, API для синхронизации с CMS сайта. 
  • В рамках работы над увеличением продаж Яндекс проведет эксперименты по допродаже брошенных корзин.

Напомним, также на конференции Вебмастерская Яндекс рассказал об изменениях в поиске и запуске новых сервисов.

Nokia 7.2 receiving November Security update 2019 now

22 Nov 2019, 13:42 by Nayan

Nokia 7.2 has started receiving the November Android Security update 2019 in many markets. These are the first Nokia smartphones to receive the November security patch. Check below for the update size, list of markets and the update changelog.

For all software update news related to other Nokia smartphones click here.

On the basis of tips received from our readers, we will collate a list of markets for the Nokia 7.2 for which November Security update is now available. So, do let us know if you have received the update in the comments section. You can also try the VPN trick for getting the update and see if it works.

List of markets:

  • Nokia 7.2 in India

Nokia 7.2 November update size:

The update size for Nokia 7.2 is 68.5 MB in size. You will either be prompted to download this update or you can check by going to Settings and searching system updates and then by checking for the update.

Nokia 7.2 November Security update changelog:

The changelog only mentions the November Android Security update for Nokia 7.2. The most severe of these issues is a critical security vulnerability in the System component that could enable a remote attacker using a specially crafted file to execute arbitrary code within the context of a privileged process.

Read the complete details of the November Security update here.

HMD has been quite consistent in terms of releasing Monthly Security updates and OS updates for Nokia smartphones already available in the market. HMD has also made it clear that all Nokia Android Phones will receive a minimum of two years of OS update support.

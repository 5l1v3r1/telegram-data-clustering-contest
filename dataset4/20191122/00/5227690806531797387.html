Google Cloud Print sera abandonné en 2021

Technologie : Le géant californien vient d’annoncer la fin de son service d’impression de documents multi OS via Internet qui interviendra en janvier 2021.

22 Nov 2019 by L'agence EP

Cloud Print va rejoindre le cimetière des services et applications Google qui commence à être bien peuplé. La plateforme d’impression via des serveurs web sur des imprimantes partagées cessera de fonctionner à compter de janvier 2021. Officiellement en version bêta depuis son introduction en 2011, Cloud Print permet d’imprimer tout type de documents depuis un smartphone, tablette, PC, Chromebook connecté. La file d’attention s'exécute en ligne sur les serveurs Google avant d’être envoyée à l’imprimante associée au terminal émetteur. L’atout maître de Cloud de Print est qu’il ne nécessite aucune installation de pilote d’impression. 

"Nous vous recommandons d'identifier une solution alternative et d'exécuter une stratégie de migration au cours de la prochaine année”, conseille Google. Pour Chrome OS, la firme de Mountain View a d’ores et déjà adopté le Common Unix Printing System qui suppléera Cloud Print le moment venu. Pour les autres environnement ou les scénarios multi OS, les utilisateurs sont invités à se tourner vers les solutions natives ou un service tiers. (Eureka Presse)

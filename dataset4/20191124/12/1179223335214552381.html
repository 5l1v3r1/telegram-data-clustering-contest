AVG- und Avast-Antivirus: Microsoft blockiert Windows 10 1909-Updates

24 Nov 2019, 12:58 by Nadine Juliana Dressler

Microsoft hat eine weitere Update-Blockade eingerichtet. Dieses Mal betrifft es Nutzer, die auf das neue Windows 10 November Update alias Version 1909 aktualisieren wollen, aber veraltete Versionen von AVG- und Avast-Antivirus installiert haben. Das berichtet das Online-Magazin TechDows und zitiert dabei das aktualisierte Support-Dokument von Microsoft. Darin heißt es, dass Microsoft mit den Entwicklern der Avast und AVG Anti-Virus-Produkte arbeitet, um die Kompatibilitätsprobleme zu beheben. Probleme gibt es aber nur mit bestimmten Versionen der Programme. Wie sich die Fehler genau äußern wurde bisher nicht beschrieben.

Die Liste der Safeguards wächst

Für alle Windows 10-Anwender gilt, dass vor einem möglichen OS-Update geprüft werden sollte, ob es auch Aktualisierungen für andere eingesetzte Software gibt - so lassen sich die meisten Fehler im Vorfeld vermeiden. Die Liste der Safeguards, also der Update-Blockaden durch Microsoft, wächst derzeit.

Das Problem ist laut dem Microsoft-Support im Grunde schon behoben. Man hat den Fehler zwar in die Liste der Safeguards aufgenommen, aber auch als "extern behoben" markiert.

Im Support-Dokument heißt es:

  • "Probleme mit einigen älteren Versionen von Avast und AVG Anti-Virus-Produkten"
  • Microsoft und Avast haben Kompatibilitätsprobleme mit einigen Versionen von Avast Antivirus und AVG Antivirus festgestellt. Jede Anwendung von Avast oder AVG, die eine Antivirenversion 19.5.4444.567 oder früher enthält, ist betroffen.
  • Um Ihr Upgrade-Erlebnis zu gewährleisten, haben wir Geräte mit betroffenem Avast und AVG Antivirus von dem Update-Angebot für Windows 10, Version 1903 oder Windows 10, Version 1909, bis zum Update der Anwendung ausgeschlossen.
  • Betroffene Plattformen:
  • Endkunden: Windows 10, Version 1909; Windows 10, Version 1903
  • Server: Windows Server, Version 1909; Windows Server, Version 1903
  • Abhilfemaßnahme: Vor dem Update auf Windows 10, Version 1903 oder Windows 10, Version 1909, müssen Sie eine aktualisierte Version Ihrer Avast- oder AVG-Anwendung herunterladen und installieren. Anleitungen für Avast- und AVG-Kunden finden Sie in den folgenden Support-Artikeln:
  • bei Avast
  • bei AVG
  • Hinweis: Wir empfehlen Ihnen, nicht zu versuchen, ein manuelles Update über die Schaltfläche "Jetzt aktualisieren" oder das Medienerstellungs-Tool durchzuführen, bis eine neue Version Ihrer Avast oder AVG-Anwendung installiert wurde und Ihnen automatisch das Feature-Update für Windows 10, Version 1903 oder Windows 10, Version 1909, angeboten wurde.

Windows 10X 将支持 Win32 应用与 Windows Sandbox

24 Nov 2019, 08:59 by ugmbbc

微软官方确认了 Windows 10X 系统将支持 Win32 应用、Windows Sandbox 与 Windows Defender Application Guard。上个月初，微软在一个秋季活动中介绍了双屏设备 Surface Neo 预览版，该设备运行的操作系统称为传闻中的 Windows Core Santorini —— Windows 10X。

现在微软在招聘信息中透露，Azure Core OS Kernel 团队正在与 Windows 团队合作开发相关容器，通过容器技术将增强 Windows 10X 的安全性、隔离性与兼容性。

招聘信息如上图所示：

The Azure Core OS Kernel team is seeking an experienced development lead to manage the Containers team.  The Containers team collaborates with teammates throughout Windows to design, develop, and enable new scenarios that leverage containers for improved security, isolation and compatibility.  These technologies form the basis for Store-delivered Win32 applications, Windows Server Containers, Windows Defender Application Guard, Windows Sandbox, and Win32 application support for Windows 10X on dual-screen devices like Surface Neo.

大意是说 Azure Core OS Kernel 团队正在寻找经验丰富的开发主管来管理容器团队，容器团队与整个 Windows 的队友合作、设计、开发和启用利用容器提高安全性、隔离性和兼容性的新方案。这些技术将会在 Windows 10X 上支持商店交付的 Win32 应用、Windows Server 容器、Windows Defender Application Guard、Windows Sandbox 与传统 Win32 应用。

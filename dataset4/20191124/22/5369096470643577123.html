Μπορεί να φταίει το ζώδιο τους, μπορεί να φταίει ο ωροσκόπος τους, μπορεί και να φταίει ο πλανήτης που έχουν στον Σκορπιό.

Μπορεί να φταίει το ζώδιο τους, μπορεί να φταίει ο ωροσκόπος τους, μπορεί και να φταίει ο πλανήτης που έχουν στον Σκορπιό.

Κριός

Άγγελος Ευαγγέλου, ο... ευαγγελιστής της «εξυγίανσης»

Δεν μπορείς να είσαι με έναν άνθρωπο που εξαρτάται από εσένα. Θέλεις να έχεις τον χώρο σου και την ελευθερία σου, οπότε ένας άνθρωπος υπερβολικά τρυφερός ίσως και να μην κάνει για εσένα. Χάνεις τον ενθουσιασμό σου όταν κάποιος δείχνει μεγάλο ενδιαφέρον.

Ταύρος

Οι υπερβολικά ισχυρογγώμονες άνθρωποι σου τη σπάνε. Επειδή είσαι και εσύ ισχυρογνώμον και δεν σου αρέσει να συμβιβάζεσαι. Χρειάζεσαι κάποιον που να είναι πιο χαλαρός και επιεικής ώστε να επικοινωνείτε και να μην σε επηρεάζει στις καθημερινές σου αποφάσεις.

Δίδυμοι

Ένας άνθρωπος συνεσταλμένος μάλλον δεν κάνει για εσένα. Είσαι γεμάτος ενέργεια και ζωή και χρειάζεσαι κάποιον που ξέρει πώς να αγαπάει και να ζει την κάθε στιγμή της ζωής του. Ένας άνθρωπος που του αρέσει η ησυχία και η απομόνωση είναι μια χαρά, απλώς όχι για εσένα.

Καρκίνος

Όταν ένας άνθρωπος «βάζει» τοίχους, είναι απαγορευτικό σημάδι για εσένα. Αναζητάς την τρυφερότητα, συζητήσεις εις βάθος και δυνατή συναισθηματική επικοινωνία. Οι άνθρωποι που δεν μπορούν να σου τα προσφέρουν αυτά θα σε αφήνουν συνεχώς να αναρωτιέσαι.

Λέων

Οι ντροπαλοί άνθρωποι, κακά τα ψέματα, δεν σου ταιριάζουν. Σου αρέσει ο σύντροφός σου να είναι τολμηρός και να νιώθει σίγουρος για τον εαυτό του. Θα βαρεθείς πολύ εύκολα με κάποιον αρκετά εσωστρεφή, καθώς σε ελκύουν άνθρωποι με ισχυρές προσωπικότητες.

Παρθένος

Δεν σου κάνουν οι άνθρωποι που είναι πολύ «χύμα», χίπηδες αν προτιμάς. Εσύ θέλεις να οργανώνεις και την παραμικρή λεπτομέρεια και η καριέρα σου σημαίνει πολλά για εσένα. Η σχέση με έναν άνθρωπο που βρίσκεται σε μία διαρκή αναζήτηση του εαυτού του και θέλει να ταξιδεύει συνέχεια ίσως και να μην κρατήσει.

Ζυγός

Ένας νευρικός άνθρωπος θα ήταν αρκετά κόντρα στη δική σου ιδιοσυγκρασία. Σου αρέσει η ηρεμία, η ειρήνη και η αρμονία και χρειάζεσαι τον χρόνο σου με ό,τι κάνεις. Ο σύντροφος που θα σε πιέσει είτε ψυχολογικά, είτε απλώς με τον τρόπο που αντιμετωπίζει την ζωή, δεν σου ταιριάζει ιδιαίτερα.

Σκορπιός

Οι άνθρωποι που κρατούν παθητική στάση δεν είναι για εσένα –και ίσως ούτε εσύ για αυτούς. Σε αντίθεση με τον Ζυγό εσένα σου αρέσουν οι νευρικοί άνθρωποι, αυτοί που έχουν «τσαγανό». Σου αρέσει ο σύντροφος σου να είναι τόσο σίγουρος ώστε να σου δείχνει ότι είσαι μοναδικός. Τι να κάνουμε; Κρύβεις πολύ πάθος μέσα σου.

Τοξότης

Δεν θα άντεχες με τίποτα ένα άνθρωπο επικριτικό. Έχεις πολλούς φίλους και έχεις κρατήσει σχέσεις με πολλούς ανθρώπους που έχουν περάσει από τη ζωή σου. Γενικά κάνεις εύκολα φίλους. Κάποιος που κρίνει διαρκώς τον κύκλο σου δεν θα τα πάει καλά μαζί σου.

Αιγόκερως

Δεν μπορείς να είσαι με έναν άνθρωπο ασταθή. Σου αρέσει η συνέπεια και η τάξη. Κάποιος που φαίνεται αδιάφορος και κυκλοθυμικός δεν θα ταιριάξει μαζί σου. Θέλεις κάποιος που να μπορεί να σου εγγυηθεί την ασφάλεια που χρειάζεσαι για να ανοιχτείς.

Υδροχόος

Ο παραδοσιακός τύπος το ξέρεις ότι δεν κάνει για εσένα, έτσι; Δεν μπορείς να είσαι με κάποιον που του αρέσει να ακολουθεί τους κανόνες ή πιστεύει ότι τα δημιουργικά επαγγέλματα είναι απλώς χόμπι. Χρειάζεσαι κάποιον που θα πιστέψει στα όνειρα σου και θα σε παροτρύνει να τα ακολουθήσεις.

Ιχθείς

Δεν σου κάνουν οι απόμακροι άνθρωποι. Αυτοί που χρειάζονται πολύ χρόνο για να παραδεχτούν τα συναισθήματά τους. Εσένα σου βγαίνει τόσο φυσικά. Είσαι ευαίσθητος και σε οδηγεί το συναίσθημα· δεν θα είσαι ευτυχισμένος με κάποιον που σου βάζει σε αυτό όρια. Θέλεις να είσαι σε θέση να «διαβάσεις» τον σύντροφό σου, όχι να αναρωτιέσαι.

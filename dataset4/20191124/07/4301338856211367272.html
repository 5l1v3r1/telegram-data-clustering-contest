Τα ζώδια σήμερα Κυριακή 24 Νοεμβρίου

24 Nov 2019, 07:07

ΚΡΙΟΣ: Οσοι είστε ακόμα ελεύθεροι αναζητήστε εκείνο το πρόσωπο που θα ταιριάζει με τον χαρακτήρα σας για να το αφήσετε να μπει στην καρδιά σας. Εάν θεωρήσετε ότι κάποιες στιγμές δεν μπορείτε να βασιστείτε στον διάλογο για να επιλύσετε σημαντικά προβλήματα, αφήστε το για αργότερα. Θα υπάρχει έντονος ανταγωνισμός στον εργασιακό σας χώρο και γι’ αυτό θα πρέπει να φροντίσετε να είστε πλήρως ενημερωμένοι για τα τρέχοντα θέματα, ώστε να μην μπορέσει κανείς να σας κουνήσει από τη θέση σας. Βρείτε τρόπο να ελίσσεστε στις προκλήσεις και να λύσετε οριστικά αδιέξοδα.

ΤΑΥΡΟΣ: Οι νέες γνωριμίες που θα κάνετε θα αποδειχθούν αρκετά σημαντικές. Θα έχετε αρκετές προτάσεις για νέες γνωριμίες, ενώ δεν αποκλείεται σε κάποιο ταξίδι που θα κάνετε, ακόμα και αν είναι για επαγγελματικούς λόγους, να συναντήσετε κάποιο πρόσωπο που η γνωριμία σας θα αρχίσει από φιλία και στη συνέχεια θα εξελιχθεί σε σχέση. Αν έχετε ήδη αρχίσει κάποιο φλερτ, θα μπορέσετε αυτόν τον μήνα να σταθεροποιήσετε την κατάσταση. Οι δεσμευμένοι ανανεώστε τον δεσμό σας, συζητήστε με τον σύντροφό σας σημαντικά προβλήματα, ακόμα και τις αδυναμίες που μπορεί να έχει η σχέση σας.

ΔΙΔΥΜΟΙ: Αρκετές φορές θα εξιδανικεύετε τον σύντροφό σας προκειμένου να πείσετε τον περίγυρό σας για τις αρετές του. Ομως, αυτό δεν θα σας βοηθήσει να έχετε ορθή κρίση, αφού θα τα προσαρμόζετε στα δικά σας μέτρα και σταθμά. Βρείτε ισορροπίες, καθώς η σχέση σας δεν φαίνεται να απειλείται από κανέναν, αλλά αντίθετα θα μπορέσετε να ξεκαθαρίσετε πολλά μέσα σας, θα βάλετε τάξη στα συναισθήματα που νιώθετε και θα ενισχύσετε τη σχέση σας με αγάπη, κατανόηση και αλληλοσεβασμό. Θα ζήσετε πιο έντονες στιγμές και θα οδηγηθείτε μαζί με τον σύντροφό σας σε πιο ικανοποιητικά αποτελέσματα.

ΚΑΡΚΙΝΟΣ: Μην επιτρέψετε σε οικογενειακά ζητήματα να ταλαιπωρήσουν τη σχέση σας. Δώστε στους άλλους ξεκάθαρα να καταλάβουν πως η προσωπική σας ζωή αφορά αποκλειστικά εσάς και δεν πρόκειται να αφήσετε κανέναν να πάρει αποφάσεις για εσάς.
Επιβάλλεται ένα διάλειμμα από τις υποχρεώσεις που έχετε αναλάβει και βάλτε σε πρώτη μοίρα τη φυσική σας κατάσταση και την υγεία σας. Οι επαγγελματικές σας συνεργασίες θα αρχίσουν να εξελίσσονται και να στηρίζονται σε πιο γερές βάσεις. Οι διαπραγματεύσεις που ζητάτε θα γίνουν σε πιο ρομαντική ατμόσφαιρα, κάτι που έχετε πραγματικά ανάγκη.

ΛΕΩΝ: Ο,τι σας απασχολούσε από το παρελθόν και σας ταλαιπωρούσε μέχρι τώρα αρχίζει σταδιακά να υποχωρεί. Προσέξτε τι θα πείτε και κυρίως τη συμπεριφορά σας, γιατί θα βρίσκεστε στο επίκεντρο. Μην προσπαθήσετε να παρουσιάσετε διαφορετικά τον εαυτό σας από αυτό που είναι, γιατί οι άλλοι δεν είναι τυφλοί και θα θεωρήσουν ότι παίζετε εσκεμμένα κάποιο παιχνίδι. Εξάλλου, στηρίζεστε πάντα στη λογική και στα πόδια σας, και αυτό το αναγνωρίζουν και το παραδέχονται σχεδόν όλοι από το επαγγελματικό σας περιβάλλον. Είναι μια καλή ευκαιρία να αναθερμάνετε τις σχέσεις σας, δείχνοντας κατανόηση στα αγαπημένα σας πρόσωπα και αποδοχή σε ορισμένες απόψεις τους.

ΠΑΡΘΕΝΟΣ: Θα χρειαστεί ευελιξία από την πλευρά σας, καθώς θα υπάρχει ανταγωνισμός στο παρασκήνιο. Οι περιορισμοί που δέχεστε από οικογενειακά πρόσωπα σας φέρνουν σε δύσκολη θέση και δημιουργούν εντάσεις. Εάν ορισμένα από τα αγαπημένα σας πρόσωπα δεν ανταποκρίνονται στις προσδοκίες σας, μην τους επιτεθείτε λεκτικά, γιατί θα προκύψουν σκληρές αντιπαραθέσεις που δεν είστε έτοιμοι να αντιμετωπίσετε. Ομως, θα έχετε τη δυνατότητα να σταθεροποιήσετε σχέδια που θα αφορούν τη σχέση με τον σύντροφό σας ή τους συνεργάτες σας. Εμπιστευθείτε τη διαίσθησή σας, αλλά να είστε προσεκτικοί για να μην εξαπατηθείτε από φίλους.

ΖΥΓΟΣ: Θα μπείτε σε μια νέα φάση ζωής, κάνοντας μια νέα αρχή, όπου θα μπορείτε με μεγαλύτερη άνεση να υλοποιείτε σχέδια, είτε αυτά θα έχουν σχέση με την εργασία σας είτε με τον προσωπικό και ερωτικό τομέα. Βέβαια, θα εξακολουθούν να σας πιέζουν κάποιες καταστάσεις από το οικογενειακό σας περιβάλλον, αλλά, παρ’ όλα αυτά, θα καταφέρετε να βάλετε μια πιο σταθερή πορεία στη ζωή σας. Θα ανεβείτε μερικά σκαλοπάτια στην ιεραρχία στον εργασιακό χώρο, αρκεί να αυξήσετε την εργασιακή σας απόδοση, να γίνετε πιο δραστήριοι και να προσεγγίσετε με λογική τους στόχους σας.

ΣΚΟΡΠΙΟΣ: Χαλαρώστε και δώστε προτεραιότητα στον εαυτό σας, που είναι ό,τι πιο πολύτιμο έχετε. Λίγο περίεργα θα είναι τα πράγματα στον τομέα των σχέσεών σας και αρκετές φορές θα νιώσετε διχασμένοι. Θα υπάρξουν αρκετές νέες γνωριμίες, ενώ δεν αποκλείεται να σας γίνει και πρόταση γάμου, που όμως καλό θα ήταν να αποφύγετε προς το παρόν, καθώς δεν είστε σε θέση να αντιμετωπίσετε τα γεγονότα που υπάρχουν στη ζωή σας. Τα πολύ κοντινά σας πρόσωπα θα απολαμβάνουν μια πολύ τρυφερή εκδοχή σας, μια που έχετε τη διάθεση να περάσετε καλά μαζί τους.

ΤΟΞΟΤΗΣ: Θα αρχίσετε να συνειδητοποιείτε τις αλλαγές που έγιναν σε κάποιες επαγγελματικές, οικονομικές και ερωτικές σχέσεις σας. Είναι κάτι που χρειαζόσασταν εδώ και πολύ καιρό για να οδηγηθείτε σε πιο αισιόδοξους δρόμους. Οι εργασιακές σχέσεις θα αρχίσουν να βελτιώνονται και θα δημιουργήσετε ένα καλύτερο περιβάλλον, όπου θα εργάζεστε με λιγότερο άγχος. Αναπτύξετε τον κοινωνικό σας κύκλο και κάντε περισσότερες καινούργιες γνωριμίες. Αποφύγετε την άσκοπη ταλαιπωρία και μην απασχολείστε με πράγματα που σας ενοχλούν. Η εμφάνισή σας δεν θα σας απασχολήσει ιδιαίτερα, αλλά καλό θα ήταν να μην ξεχάσετε και την παραμελήσετε εντελώς.

ΑΙΓΟΚΕΡΩΣ: Προσέξτε τις διατροφικές συνήθειες που έχετε αποκτήσει τον τελευταίο καιρό, γιατί, αν η μεζούρα σας δείξει διαφορετικά νούμερα, δεν χάλασε… Θα φταίει που μπερδευτήκατε και, αντί να επεκτείνετε τους στόχους σας, «επεκτείνατε» το σώμα σας. Η τύχη θα είναι συνέχεια στο πλευρό σας, αλλά αυτό δεν σημαίνει πως πρέπει να κάνετε κατάχρηση. Οι περισσότερες ευκαιρίες θα δοθούν στον επαγγελματικό τομέα και, ακόμα και σε δύσκολες συνθήκες, θα χαράξετε καινούργιο δρόμο στην εργασία σας. Προβάλετε τη δουλειά και τις ικανότητές σας, ενώ παράλληλα αποδεχθείτε εξόδους, προσκλήσεις από φίλους και συνεργάτες σε εκδηλώσεις.

ΥΔΡΟΧΟΟΣ: Δεν μπορείτε να λέτε ότι δεν υπάρχει κάποιος για εσάς, απ’ τη στιγμή που δεν το έχετε προσπαθήσει καν. Με το να είστε κλεισμένοι σε ένα γραφείο, αφήνετε τον χρόνο να φεύγει και δεν μπορείτε να τον γυρίσετε πίσω. Μερικές φορές δεν χρειάζεται να πάτε πολύ μακριά ή να κουραστείτε για να βρείτε το άλλο σας μισό. Ισως είναι δίπλα σας, αλλά οι παρωπίδες που φοράτε δεν σας αφήνουν να δείτε. Ανοίξτε τα μάτια σας και θα εντυπωσιαστείτε, όταν ανακαλύψετε πως ένας πολύ καλός σας φίλος ενδιαφέρεται για εσάς, και μάλιστα με σοβαρές προοπτικές σχέσης.

ΙΧΘΥΣ: Δώστε μια ευκαιρία στον σύντροφό σας για να σας αποδείξει ότι σας αγαπά και σας χρειάζεται δίπλα του. Πάψτε να είστε σκληροί μαζί του και αφήστε να «ακουμπήσει» στην καρδιά σας. Είναι παράλογο να τον κατηγορείτε ή να τον κρίνετε για τα ξεσπάσματά του, όταν εσείς ρίχνετε πάνω του το φταίξιμο για ό,τι σας συμβαίνει και του συμπεριφέρεστε λες και είναι το τελευταίο πρόσωπο που σας ενδιαφέρει. Εξάλλου, ακόμα και αν δεν θέλετε να το παραδεχθείτε από εγωισμό, έχετε συνειδητοποιήσει πως τρέφετε βαθιά αισθήματα για το άτομό του και θέλετε να παραμείνει στη ζωή σας.

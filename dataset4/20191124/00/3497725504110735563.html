أسماء الفائزين بانتخابات المجلس الاستشاري في مدينة الشارقة

24 Nov 2019, 00:12

- محمد علي جابر الحمادي ( 403 صوتا )
- عبيد عوض عبيد الطنيجي ( 376 صوتا )
- شريفة موسى حسن المازمي ( 326 صوتا )
- جاسم جمعة عمر المازمي ( 297 صوتا )
- إسماعيل كامل إسماعيل البريمي ( 278 صوتا )

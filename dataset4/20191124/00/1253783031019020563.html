Série XZ2, XZ3 e Xperia 5 são atualizados com pacote de segurança de outubro para 2019

24 Nov 2019

A Sony continua trabalhando para entregar a proteção mais recente possível para diversos de seus smartphones, e isso inclui, é claro, o Xperia 5, o modelo mais recente lançado pela fabricante japonesa – que chegou à Amazon Itália em setembro.

Ele acaba de ser atualizado para o firmware 55.0.A.11.25, que traz o pacote de segurança do Android para outubro de 2019; o mesmo também foi disponibilizado para as linhas Xperia XZ2 e XZ3 com o firmware 52.0.A.11.3.

Ao mesmo tempo, a Sony liberou também updates para a linha Xperia XZ1 (firmware 47.2.A.11.228) e para o Xperia L2 (firmware 49.0.A.6.96) com a proteção do Android para o mês de setembro de 2019.

Lembramos que é sempre aconselhado realizar o backup dos dados importantes antes de iniciar a instalação de qualquer atualização de sistema.

Também é interessante verificar se o aparelho possui pelo menos 50% de bateria, caso contrário, recomendamos plugá-lo na tomada para evitar desligamentos desnecessários.

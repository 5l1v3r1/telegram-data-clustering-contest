Une MAJ 4.1.3 est disponible depuis le milieu de journée pour le server Delta

25 Nov 2019, 13:54 by Isabelle Deromas

Cette mise à jour est effectivement disponible depuis aujourd’hui midi..

Pour en profiter, il suffit de démarrer le Freebox Server.

La mise apporte les changements suivants:

Améliorations:

Plusieurs options de réparation de RAID sont maintenant disponibles (uniquement) quand un Array Raid est dégradé et notamment :

  • si l’array a des disques dégradés, il est possible de les enlever de l’array, puis d’en ajouter d’autres;
  • si l’array a été démarré avec des disques manquants, il est possible de réintégrer ces disques désynchronisés;
  • si l’array ne démarre pas, il est possible de tenter de forcer le démarrage pour accéder aux options de réparation précédentes.

Source : https://dev.freebox.fr/blog/?p=10125.

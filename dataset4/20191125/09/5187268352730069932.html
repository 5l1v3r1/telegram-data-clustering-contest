Dica: Saiba como poupar dados ao navegar na Internet no seu smartphone Android

25 Nov 2019, 09:00 by Pedro Simões

Mesmo não sendo um browser que é imposto aos utilizadores do Android, o Chrome é sem qualquer dúvida o padrão deste sistema. A Google sabe disso e tem tornado esta sua proposta como a mais capaz e a mais integrada.

Traz por isso muitas novidades a este browser e aos utilizadores, melhorando cada vez mais o que este oferece em várias áreas. Uma delas é na poupança de dados consumidos durante a navegação na Internet. Há um pequeno segredo no Chrome que podem sobretudo usar neste campo.

São muitas as opções que a Google tem colocado no Chrome. Estas melhoram principalmente a sua utilização e dão a esta ferramenta a capacidade de explorar ao máximo a Internet. Uma delas está igualmente na poupança de dados ao navegar na Internet, com o Modo Lite.

Saiba como poupar dados no Chrome

Este modo torna as páginas mais leves e, por inerência, mais rápidas a carregar no smartphone. Este modo não está sempre ligado e, por isso, é importante que seja ativado por quem quer fazer esta poupança.

Abram então o menu do Chrome e escolham a opção Definições. Depois disso, e dentro das várias opções deste browser, escolham a opção Modo Lite. Este irá de imediato indicar se está ou não Desativado no browser da Google.

Ativar esta opção no browser da Google

Caso não esteja ativo, devem entrar nessa opção e mudar a posição do interruptor no topo. A Google alerta para que quando este modo está ativo, as páginas são enviadas para um ponto centrar para serem otimizadas. Este modo não se aplica na navegação anónima.

Naturalmente que, assim que este modo é ativado, não vai mostrar qualquer poupança de dados. Com o tempo e com a utilização do Chrome, estes dados vão ser acumulados e apresentados nesta área do browser.

Saber de que forma o Chrome conseguiu poupar dados

Com o passar do tempo, a acumulação de dados torna evidente a poupança de dados que está a ser feita. Com o fim de mostrar essa poupança, o Chrome irá mostrar num gráfico temporal a poupança que foi conseguida.

Além disso, há ainda a quantificação dos dados que não foram consumidos, com uma lista dos sites visitados. Para cada um é mostrada informação sobre os dados utilizados e os que foram poupados. Podem também limpar as estatísticas a qualquer momento.

Esta poupança é sobretudo importante para os utilizadores, que assim podem gastar noutros serviços. Também é uma excelente capacidade para o Chrome, que apenas a tem no Android, deixando o desktop e o iOS sem esta funcionalidade.

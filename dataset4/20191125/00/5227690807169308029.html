Microsoft bloque les mises à jour Windows 10 pour certains utilisateurs Avast et AVG

Sécurité : Microsoft bloque la mise à jour de Windows 10 sur les appareils exécutant certaines versions des pilotes Avast, AVG et Qualcomm.

25 Nov 2019 by Liam Tung

Les utilisateurs de certains logiciels antivirus AVG et Avast peuvent se voir empêchés de passer à la dernière version de Windows 10 en raison d'un problème de compatibilité.

Microsoft a bloqué les mises à jour vers Windows 10 versions 1909 et 1903 pour les utilisateurs d’AVG et d‘Avast exécutant la version 19.5.4444.567 ou une version antérieure.

publicité

"Pour protéger votre expérience de mise à jour, nous avons appliqué une suspension sur les appareils concernés par le problème Avast et AVG Antivirus jusqu'à la mise à jour de l'application", déclare Microsoft.

Pour contourner le blocage, les utilisateurs devront télécharger et installer une version mise à jour d'Avast ou d'AVG. Les mises à jour et les instructions d'installation relatives à la mise à jour du logiciel antivirus sont disponibles sur les pages de support d'Avast et d'AVG.

C’était la première fois que Microsoft évoque le problème de compatibilité avec AVG et Avast mais certains utilisateurs ont déclaré ne pas être en mesure d’installer la version 1903 depuis le mois d’août. Avast a acquis AVG en 2016. La page d'AVG indique que le blocage de mise à jour Windows affecte AVG Ultimate 19.5 et versions antérieures, AVG Internet Security 19.5 et versions antérieures et AVG AntiVirus FREE 19.5 et versions antérieures.

Les produits Avast concernés incluent Avast Premium Security 19.5 et versions antérieures, Avast Premier 19.5 et versions antérieures, Avast Internet Security 19.5 et versions antérieures et Avast Free Antivirus 19.5 et versions antérieures.

Microsoft a également imposé un dispositif similaire sur les mises à jour Windows 10 pour les versions 1903, 1909 et 1809, en raison d'un problème de compatibilité avec certains pilotes Qualcomm Wi-Fi. Les utilisateurs concernés sont avertis par un message dans l’installation de Windows 10, qui attire l’attention sur la nécessité de mettre à jour le pilote.

"Pour protéger votre expérience de mise à jour, nous avons appliqué un blocage sur les appareils dotés du pilote Qualcomm concerné. Les mises à jour vers Windows 10 versions 1809, Windows 10, version 1903 ou Windows 10, version 1909, sont bloquées jusqu'à l'installation du pilote mis à jour", explique Microsoft.

Une mise à jour Qualcomm doit être proposée aux utilisateurs de Windows 10 après avoir sélectionné l’option Vérifier les mises à jour. Mais si la mise à jour n’est pas proposée, Microsoft a des solutions de repli à proposer. "Une fois que vous avez mis à jour vos pilotes de carte Qualcomm Wi-Fi, le blocage devrait avoir été retiré et vous devez pouvoir installer la version plus récente de Windows 10", note Microsoft.

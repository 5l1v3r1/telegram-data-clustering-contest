Fortnite: Angelgerangel gestartet – Wer fängt den mythischen Goldfisch?

23 Nov 2019, 09:31 by Maik Seidl

Am laufenden Wochenende könnt ihr in "Fortnite" an einem Angel-Event teilnehmen. Epic Games schüttet an die Gewinner Preise aus, darunter die Spitzhacke „Spitzhanker“ und eine Lamatrophäe.

Am laufenden Wochenende könnt ihr in „Fortnite“ an einem besonderen Event teilnehmen, in dessen Rahmen ihr eure Fähigkeiten beim Angeln unter Beweis stellen dürft. Dabei handelt es sich um das sogenannte Angelgerangel, das schon am gestrigen Freitag an den Start ging. Erfolgreiche Spieler erhalten die Spitzhacke „Spitzhanker“ oder eine echte Lamatrophäe.

Zudem könnt ihr euch ab dem heutigen Samstag im Item-Shop das Outfit „Schlachtenstäbchen“ abholen und eure Gegner in den Tiefen versenken.

Details zum Angel-Event

Das Angelgerangel läuft bis Montag, den 25. November 2019 um 2:00 Uhr. In jeder Serverregion wird für jede der folgenden Kategorien ein Sieger bestimmt. Für Europa gilt diese Bestenliste.

Bester Angler 

  • Fangt während des Events im Verlauf eines Matches die meisten Fische.
  • Pro Regionsserver gibt es einen Champion.
  • Ihr gewinnt eine gravierte Lamatrophäe.

Kleiner-Fisch-Champion

  • Fangt während des Events die meisten kleinen Fische.
  • Pro Regionsserver gibt es einen Champion.
  • Ihr gewinnt eine gravierte Lamatrophäe.

Zappler-Champion

  • Fangt während des Events die meisten Zappler.
  • Pro Regionsserver gibt es einen Champion.
  • Ihr gewinnt eine gravierte Lamatrophäe.

Schlürffisch-Champion

  • Fangt während des Events die meisten Schlürffische.
  • Pro Regionsserver gibt es einen Champion.
  • Ihr gewinnt eine gravierte Lamatrophäe.

Mythischer Goldfisch = sofortiger Preis

  • Jeder Spieler, der einen mythischen Goldfisch fängt, erhält die Spitzhacke „Spitzhanker“!

Ein guter Grund anzubeißen

  • Jeder Spieler, der während des Events teilnimmt, erhält das Spraymotiv Spiele um den Sieg.

  • Benutzt dafür eine Angel innerhalb der Event-Zeiten.

    • Von: Freitag, 22. November 14:00 Uhr MEZ
    • Bis: Montag, 25. November 2:00 Uhr MEZ

So nehmt ihr am Angelgerangel teil

Zunächst müsst ihr in „Fortnite“ eine Angel aufsammeln, um dann eure Kontrahenten in einer der Kategorien zu übertreffen. Die Sieger werden über das Geschenksystem des Spiels benachrichtigt. Auch erfolgt eine Nachricht an die E-Mail-Adresse, die zur jeweiligen Epic-Konto-ID gehört. Zu beachten ist: Lediglich das Angeln ist erlaubt. Fische, die mit dem Harpunierer erlegt wurden, zählen nicht.

Weitere Details zum „Fortnite“-Event Angelgerangel, darunter auch die weiteren Regeln, findet ihr auf der offiziellen Seite von Epic Games.

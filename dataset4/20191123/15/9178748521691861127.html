Мошенники выдают опасный шифровальщик за обновление Windows

23 Nov 2019, 15:30 by Дмитрий Пугачев

Вскоре после выхода сезонного обновления Windows 10 о необходимости установки апдейта пользователям начали напоминать не только разработчики ОС, но и хакеры. Увы, речь вовсе не о повышении компьютерной грамотности владельцев ПК — под видом свежего патча злоумышленники «уговаривают» их скачать опасный вирус, вымогающий деньги.

Призыв к установке обновления для популярной операционки злоумышленники рассылают по электронной почте. После того как пользователь нажмёт на предложенную ссылку, на его компьютер устанавливается программа-вымогатель Cyborg. Она шифрует все файлы на ПК и требует перевести хакерам $500 на биткойн-кошелёк для получения ключа дешифровки. Злоумышленники также опубликовали инструментарий для создания аналогичных вирусов, из-за чего их количество вскоре может ощутимо возрасти.

Эксперты по кибербезопасности предостерегают владельцев ПК под управлением Windows от запуска вложений из электронных писем с заголовками «Установить последнее обновление Microsoft Windows» или «Критическое обновление Microsoft Windows». Важно помнить, что Microsoft никогда не рассылает апдейты по почте. Получить их можно только через фирменную утилиту Windows Update.

People who do international bank transfers have always been asked for their local bank or recepient’s swifcode. A Swift Code is a standard format of Bank Identifier Codes (BIC) and it is unique identification code for a particular bank. These codes are used when transferring money between banks, particularly for international wire transfers. Banks also used the codes for exchanging other messages between them.

It should be noted that when you send or receive an international wire with your bank, you might lose money on a bad exchange rate and pay hidden fees as a result.

Format of a swiftcode

The Swiftcode consists of 8 or 11 characters. When 8-digits code is given, it refers to the primary office. The code formatted as below;

- Advertisement -

AAAA BB CC DDD

  • First 4 characters – bank code (only letters)
  • Next 2 characters – ISO 3166-1 alpha-2 country code (only letters)
  • Next 2 characters – location code (letters and digits) (passive participant will have “1” in the second character)
  • Last 3 characters – branch code, optional (‘XXX’ for primary office) (letters and digits)

Pabrik Plastik PT Wiharta Karya Agung Nyaris Terbakar

23 Nov 2019 by Deni Ali Setiono

Gresik (beritajatim.com) – Kebakaran nyaris terjadi di sekitar pabrik plastik PT Wiharta Karya Agung (WKA) di Jalan Noto Prayitno Gresik. Penyebabnya, pihak manajemen WKA saat itu sedang membersihkan filter pabrik dengan cara dibakar. Namun, saat dibersihkan, percikan api malah merembet ke alang-alang yang bersebelahan dengan pabrik WKA.

Menurut saksi mata Supardi (55) warga Jalan Noto Prayitno menuturkan, api yang membakar alang-alang di sekitar pabrik cukup besar. Beruntung segera langsung ditangani.

“Semula api kecil lalu tertiup angin membesar dan membakar alang-alang,” tuturnya, Sabtu (23/11/2019).

Untuk memadamkan kebakaran tersebut. Satu unit mobil pemadam kebakaran (Damkar) meluncur ke lokasi. Kurang dari 40 menit api berhasil dipadamkan.

Adanya kebakaran itu dibenarkan oleh Kapolsek Kebomas Kompol M.Mahmud setelah ada laporan dari anggotanya.

“Memang ada kebakaran alang-alang namun cepat segera ditangani oleh petugas pemadam kebakaran,” ujarnya. [dny/but]

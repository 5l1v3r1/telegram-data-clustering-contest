Αυτά θέλουν οι άνδρες: Τι αποκάλυψε μία συνοδός κυριών

Τι αποκάλυψε μια συνοδός κυριών για τις επιθυμίες των ανδρών.

23 Nov 2019, 20:35 by to10.gr

Οι περισσότεροι άνθρωποι θέλουν να είναι σε μια σχέση όπου θα μπορούν ανοιχθούν συναισθηματικά, ωστόσο αυτό είναι κάτι που δεν συμβαίνει συχνά, καθώς όσο και αν αγαπάνε τον άνθρωπο που έχουν δίπλα τους, τους είναι δύσκολο να είναι 100% ειλικρινείς μαζί του και να του εκφράσουν εύκολα όλα αυτά που έχουν στο κεφάλι τους, ή στην καρδιά τους, είτε από φόβο έκθεσης, είτε από ανασφάλεια.

Σε κάποιο βαθμό αυτό είναι φυσιολογικό, αλλά αν μένετε σιωπηλοί για πάρα πολύ καιρό, θα αρχίσετε να έχετε προβλήματα επικοινωνίας, τα οποία μπορεί να οδηγήσουν σε σύγχυση, δυσαρέσκεια και φυσικά σε προβλήματα στην κρεβατοκάμαρα.

Αυτό είναι ένα πρόβλημα που αντιμετωπίζουν πελάτες μιας συνοδού κυριών, όπου σύμφωνα με την ίδια: «Ο λόγος που έρχονται αρκετοί άνδρες σε μένα, όσο περίεργο και αν ακούγεται, δεν είναι το σεξ, αλλά η συναισθηματική σύνδεση».

«Το σεξ είναι σεξ, αλλά δεν χρειάζεται να ξοδέψεις τόσα πολλά χρήματα για να το βρεις. Μπορείς να καλέσεις μια πρώην σου, μια φίλη με προνόμια ή να χρησιμοποιήσεις την πρώτη σου αγάπη, την παλάμη σου!»

«Τα περισσότερα ραντεβού μου διαρκούν αρκετές ώρες”, μας είπε και συνέχισε λέγοντας, “Ποιος άντρας μπορεί να κάνει σεξ για περισσότερο από 15 λεπτά; Όλοι σχεδόν οι πελάτες μου, συνήθως θέλουν να μείνουν μαζί μου για να συζητήσουν για αυτά που τους απασχολούν, καθώς νιώθουν ότι ακούγονται και παίρνουν την επιβεβαίωση που αναζητούν.»

«Οι άνδρες είναι πολύ πιο ευαίσθητοι απ’ ότι νομίζουν οι γυναίκες και έχουν ακριβώς τις ίδιες ανάγκες με αυτές, απλώς δεν το συζητούν.»

Γενικότερα, είναι λανθασμένο το στερεότυπο ότι οι άνδρες είναι σκληροί και δεν μιλούν ποτέ για τα συναισθήματα τους, ενώ οι γυναίκες είναι ομιλητικές και πιο συναισθηματικές. «Ως γυναίκες, υποτίθεται ότι έχουμε μητρικό ένστικο, θέλουμε να νιώθουμε αγάπη και συναισθηματική σύνδεση περισσότερο από τους άνδρες. Ωστόσο, οι περισσότερες γυναίκες που γνωρίζω, εκπληρώνουν τις ανάγκες τους μέσω των παιδιών, των κολλητών και της πετυχημένης σταδιοδρομίας τους. Δεν γνωρίζω πολλές γυναίκες που αναζητούν την αγάπη, αλλά γνωρίζω ότι πολλοί άνδρες ψάχνουν την ιδανική γυναίκα».

Το συμπέρασμα; Οι άνδρες προσπαθούν να μην δείχνουν συναισθηματικά ευάλωτοι, καθώς αυτό είναι κάτι που τους προκαλεί ντροπή. Σύμφωνα με την ίδια: «Δεν είναι αδύναμοι, απλά θέλουν κάποιον για να μιλήσουν. Βλέπω πολλούς άνδρες να αγωνίζονται να ανοιχτούν, αλλά δεν θέλουν να φανούν βαρετοί ή αξιολύπητοι. Προτιμούν να μιλήσουν με μια γυναίκα που φοράει μόνο τα εσώρουχα της, παρά με κάποιον ειδικό!»

Η συμβουλής της είναι: «Δεν έχει σημασία πόσο ανόητοι νομίζετε ότι μπορεί να ακουστείτε, θα πρέπει να εκφράζετε τα συναισθήματα σας, όποια και αν είναι αυτά.»

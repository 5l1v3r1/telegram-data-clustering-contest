Росстандарт запустил сервис для проверки отзываемых автомобилей по VIN-номеру

02 Nov 2019, 06:56
Сервисом может воспользоваться любой посетитель сайта.

Росстандарт запустил интерактивный поиск для проверки автомобилей по VIN-номеру. Сервис помогает определить, подпадает ли конкретное авто под отзыв, сообщает пресс-служба ведомства.

Это открытый сервис, им может воспользоваться любой посетитель сайта.

В 2020 году Росстандарт запустит функцию личного кабинета для производителей транспортных средств, а автовладельцы будут получать не только информацию о наличии транспортного средства в отзывной программе, также сервис начнет отражать информацию о ремонте конкретного автомобиля.

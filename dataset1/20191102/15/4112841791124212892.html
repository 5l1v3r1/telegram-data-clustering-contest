Predstaviteľ Arrowa získava superschopnosti a bojuje proti robotom v technologicko-futuristickom sci-fi Code 8

02 Nov 2019, 15:15 by Dominik Vetrák

Z kraťasu Code 8 dokázali bratranci Amellovci vyžmýkať celovečerný sci-fi film.

Kraťas Code 8 s bratrancami Amellovými vyšiel pred 3 rokmi. Tvorcovia však dokázali vyzbierať dostatočné množstvo peňazí, aby z neho vznikol celovečerný sci-fi film. Toho sa dočkáme 13. decembra prostredníctvom online obchodov. Trailer vyzerá naozaj sľubne a ako amatérsky projekt môže Code 8 prekvapiť.

Perbaikan 2.500 Unit RTLH, Pemkab Klaten Anggarkan 25 M

05 Nov 2019, 05:36 by Adam Sutanto

KBRN,Klaten : Pemerintah Kabupaten Klaten menganggarkan Rp 25 miliar melalui APBD tahun 2020 untuk perbaikan rumah tidak layak huni (RTLH) diwilayah tersebut. Jumlah itu nantinya  akan digunakan untuk memperbaiki 2.500 unit RTLH untuk warga tidak mampu di wilayah Klaten.

“Dari APBD Rp 25 miliar rata di 401 Desa dan Kelurahan. Itu belum menyelesaikan untuk RTLH yang ada di kabupaten Klaten,” kata Bupati Klaten saat penyerahan bantuan dan sosialisasi RTLH di Gedung Sunan Pandanaran komplek RSPD Klaten Senin (4/11/2019).

Pemkab Klaten  terus berusaha  untuk mengurangi angka kemiskinan diwilayah itu melalui perbaikan rumah tidak layak huni. Setiap unitnya akan mendapatkan Rp 10 juta. Selain melalui APBD dirinya juga meminta kepada Kepala Desa untuk menganggarkan dana memlaui Dana Desa untuk memperbaiki rumah tidak layak huni sebanyak sepuluh unit.

“Saya minta , tolong untuk para kepala desa untuk menganggarkan dianggaran dana desanya. Kalau sekarang kan sudah lima unit, kalau besok sepuluh unit. Wajib, biar cepat selesai,” imbuh Sri Mulyani.

Sekarang ini di kabupaten Klaten masih terdapat  hampir 19 ribu rumah tidak layak huni. Untuk menyelesaikan itu akan dilakukan secara bertahap. Salah Seorang Kepala Desa Glodokan  Klaten Selatan, Zainal  Arifin mengaku siap untuk melaksanakan perbaikan rumah tidak layak huni. Pihaknya sudah menganggarkan untuk program itu sebanyak sepuluh unit.

 “Sebagai Kepala Desa saya siap. Agar program tahun 2020 itu cepat selesai. Kami sudah menganggarkan sepuluh unit. Di Desa kami ada 28 RTLH,” kata Zainal arifin.

Sedangkan Kepala Dinas Perumahan dan Kawasan Permukiman Kabupaten Klaten Pramana Agus Wijanarka kepada wartawan mengatakan,untuk mendukung program pengentasan kemiskinan itu selain melalui APBD Klaten juga  bersumber dari Bantuan keuangan  Propinsi Jawa Tengah. Selain itu Dana Alokasi Khusus (DAK), Program Bantuan Stimulan Perumahan Swadaya (BSPS) dan juga CSR. 

Sedangkan untuk tahun 2020, program perbaikan rumah tidak layak huni  sebanyak seribu 500 unit. “Untuk mendukung program RTLH Klaten itu dilaksanakan  melalui berbagai program. Untuk tahun 2020 untuk RTLH sebanyak 1500 unit,” kata Pramana.

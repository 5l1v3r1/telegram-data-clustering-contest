Dica: Conheça a nova forma escondida de matar apps no Windows 10

05 Nov 2019, 08:45 by Pedro Simões

Com o Windows 10 vimos que a Microsoft consegue inovar e criar novas soluções. Pegou num sistema que muitos consideravam quase prefeito e tornou-o ainda melhor, com novas funcionalidades e com algumas surpresas escondidas.

Estas têm vindo a surgir de forma gradual, à medida que este sistema operativo vai certamente sendo explorado ao máximo. Hoje trazemos mais um “segredo” do Windows 10, que mostra a forma como podemos matar apps de uma forma não tradicional.

Todos conhecem a forma tradicional de terminar processo no Windows. Recorrendo ao Gestor de Tarefas este é um processo direto e que se foca num único processo. Apesar de cumprir o pretendido, pode ter um impacto no sistema.

Tudo começa nas Definições do Windows 10

A Microsoft desenvolveu agora uma forma mais direta de matar estes processos de forma a garantir que a app termina também de forma correta. Comecem por abrir as Definições do Windows 10 para iniciarem esse processo.

Aí dentro, na lista de áreas disponíveis, devem escolher a que vai dar acesso às apps. Com o mesmo nome, devem escolher a área Aplicações, para assim continuar este processo que iniciaram.

Área de Aplicações vai trazer as opções pretendidas

De seguida, nesta área, devem escolher o separador Aplicações e funcionalidades. Este dará acesso à lista de apps instaladas no Windows 10. Percorram esta lista até encontrarem a que pretendem terminar.

Escolham então essa app e carreguem nela. Devem ver surgir novas opções, entre elas um link para as Opções avançadas. Devem escolher esta opção. Notem contudo que nem todas as apps permitem este processo.

É aqui que vão matar as apps que não querem abertas

Aí dentro têm igualmente acesso a informações variadas sobre a aplicação que querem matar. Percorram a lista e vão ver surgir a área Terminar. É aqui que querem agir para terminar a aplicação selecionada.

Ao carregarem no botão Terminar a aplicação escolhida irá ser fechada e deixará de estar a ser executada. É este o processo que devem realizar no futuro, sempre que tiverem problemas. Podem entretanto repetir o processo para qualquer app com problemas.

Esta é mais uma forma que têm para controlar as apps do Windows 10 e a forma como correm. Sempre que tiverem problemas podem recorrer a este método para as matar, de forma totalmente gerida e controlada.

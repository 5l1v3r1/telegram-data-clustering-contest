Alibaba Cloud предоставила на GitHub библиотеки машинного обучения

29 Nov 2019, 10:25

Облачное подразделение Alibaba Group опубликовало на GitHub кодовую основу своей платформы Alink — ряд библиотек алгоритмов, поддерживающих пакетную и потоковую обработку. Они используются для реализации задач машинного обучения, таких как онлайновые рекомендации продуктов и прочие интеллектуальные клиентские сервисы.

Alibaba Cloud рассчитывает, что разработчики и аналитики данных будут использовать предоставленный код для создания собственных программ статистического анализа, машинного обучения, прогностики в реальном времени, персонализации рекомендаций и обнаружения аномалий.

Alink построена на унифицированном движке распределённых вычислений Flink и поддерживает многие открытые платформы хранения даных, включая Kafka, HDFS и HBase. Президент Alibaba Cloud, Цзя Янцин (Jia Yangqing), утверждает, что Alink является востребованной опцией для разработчиков, заинтересованных в инструментах Больших Данных и машинного обучения.

Напоминая, что Ali входит в десятку ведущих коллективных вкладчиков в GitHub, Цзя заверил в приверженности китайского вендора к сообществу открытого кода, очередным примером чего стала публикация исходников Alink на GitHub.

Сама Alibaba с успехом использует Alink на торговой площадке Tmall, где благодаря этой платформе количество нажатий на рекомендуемые ссылки выросло на 4% во время шоппингового фестиваля 11.11.

На сегодняшний день, Alibaba представлена на GitHub 20 тысячами вкладчиков и открытыми инициативами, такими как СУБД AliSQL, контейнерный инструмент Pouch и основанная на Java версия Apache Storm — JStorm.

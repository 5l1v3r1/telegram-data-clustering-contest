Режиссер Богомолов за два года снимет для сервиса Start пять проектов

Видеосервис Start подписал с театральным режиссером, худруком театра на Малой Бронной Константином Богомоловым эксклюзивный контракт сроком на два года, в течение которых он должен снять пять новых проектов.

29 Nov 2019, 12:09 by https://www.facebook.com/TelesputnikRU

Режиссер Богомолов за два года снимет для сервиса Start пять проектов

Подписав контракт со Start, Богомолов не сможет снимать для других производителей теле- и киноконтента, сообщили «Ведомости» со ссылкой на гендиректора видеосервиса Юлию Миндубаеву. Богомолов уточнил, что в качестве режиссера или продюсера будет снимать для Start сериалы и полнометражные фильмы.

В настоящее время режиссер снимает для сервиса сериал «Хороший человек» - историю поимки «ангарского маньяка» из Иркутской области, признавшегося в убийстве 81 человека в 1992-2010 годах. Богомолов уже снимал для Start - в марте на видеосервисе состоялась премьера его дебютного сериала Содержанки», который, по словам Миндубаевой, является лидером по просмотрам на платформе. Она уверена, что новые проекты Богомолова также станут хитами.

Миндубаева не исключила возможность дальнейшей международной дистрибуции снятого Богомоловым контента. У Start уже есть опыт международных продаж: в январе Netflix приобрел права на фантастический сериал «Лучше, чем люди», сопродюсерами которого были Start и «Первый канал». По информации «Коммерсанта», сумма сделки могла составить $1 млн. Сериал получил статус Netflix Original, как эксклюзивный контент, доступный только на платформе этого онлайн-кинотеатра.

Start начал работу 7 октября 2017 года. Согласно данным «TМТ консалтинга», по итогам 2018 года сервис Start вошел в топ-10 российских видеосервисов с долей около 1,5%. Выручка Start, включающая доходы видеосервиса и производящей студии в 2018 года составила, по данным самой компании, более 800 млн руб. Количество подписчиков Start, по собственным данным компании на середину ноября, превысило 400 тыс. человек.

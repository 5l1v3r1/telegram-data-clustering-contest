Минфин против финансирования скоростной магистрали Петербург—Москва из бюджета

29 Nov 2019, 13:32

Минфин не намерен выделять бюджетные деньги для строительства высокоскоростной железнодорожной магистрали по маршруту Санкт-Петербург—Москва, сообщил заместитель министра Андрей Иванов РБК. По его словам, для этого проекта достаточно средств по программе развития РЖД.

«В большой федеральной команде есть решение по трассе Москва—Санкт Петербург, и эта трасса, по нашим оценкам, не требует привлечения дополнительных бюджетных финансов. На шестилетку именно по этой трассе есть решения, других пока нет. Достаточно средств, учтенных в долгосрочной программе развития РЖД»,— сказал господин Иванов. Как пишет издание, правительство не исключает возможность финансирования скоростной магистрали до Казани, но решение по этому вопросу еще не принято.

Общая стоимость высокоскоростной магистрали, которая соединит Санкт-Петербург с Нижним Новгородом через Москву оценивается в 2,3 трлн руб. По данным “Ъ”, в октябре РЖД предложила, чтобы из бюджета выделили 600 млрд руб. на этот проект. Кроме того, с Минфином обсуждался вопрос о возможности субсидирования купона по зеленым облигациям на 275 млрд руб.

Подробнее о предложениях РЖД читайте в материале “Ъ” «Путебюджетствие из Петербурга в Москву».

WhatsApp запустил сервис для напоминаний

09 Dec 2019, 09:46

WhatsApp сообщил о старте работы бота Any.do, который будет присылать пользователям напоминания в виде сообщений. Первую неделю после подписки он будет работать бесплатно, в дальнейшем стоимость за месяц составит $2,99.

Для создания напоминания нужно отправить боту сообщение. В качестве примера приводится текст: «Заменить масло в моей машине на следующей неделе». Также для создания задачи можно переслать сообщение из другого чата.

Ранее сообщалось о планах мессенджера запустить функцию автоматического удаления сообщений через определенный промежуток времени.

Мессенджеры занимают первое место по уровню востребованности функций смартфона среди россиян. Быстрее всего аудиторию наращивает Telegram.

Подробности — в материале “Ъ” «Россияне перезвонили по мессенджеру».

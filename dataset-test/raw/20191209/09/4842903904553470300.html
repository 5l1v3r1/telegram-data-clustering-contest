凱基銀落實開放銀行 首創貸款平台可繳7家銀行信貸、28家信用卡費

09 Dec 2019, 09:57 by 鉅亨網記者陳蕙綾 台北

開放銀行 (Open Banking) 正夯，開放 API 多是非金融業者在應用，若延伸到金融業者，以開放 API 共享金融數據，就是「開放銀行」的概念，開發金 (2883-TW) 旗下凱基銀行今 (9) 日宣布，與全國最大貸款比價平台 AlphaLoan 合作，提供市場首創跨銀行貸款帳單管理功能，此貸款平台可繳 7 家銀行信貸、28 家信用卡費，真正落實金融共享的開放銀行概念。

凱基銀行繼攜手推動台灣唯一落實開放銀行精神的數位信貸 Open API 後，今天再度與全國最大貸款比價平台 AlphaLoan 合作，提供市場首創跨銀行貸款帳單管理功能，讓「滑世代」手機族能隨時在一個服務介面上，就享有跨銀行的信用貸款比價、申貸、還款繳費等一站購足式金融服務。

AlphaLoan 共同創辦人林建宏表示，統計目前已超過 25 萬該平台會員的年齡分布，近 80% 在 30 歲 (含) 以下，其中約 30% 是從未與銀行往來的的「金融小白」，這群龐大的數位原生世代除了習慣使用手機進行貸款比價外，更樂於嘗試創新服務。

因此 AlphaLoan 今年 1 月率先與凱基銀行介接國內唯一的「行動身分認證」與「數位信貸」Open API，讓平台會員可以在比價後直接線上申請貸款，整體申辦流程大幅縮短 56% 的身分認證時間、及 30% 的個人資料填寫時間，使用體驗滿意度高達 9.62 分 (滿分 10 分)。

AlphaLoan 每天蒐集平台會員的意見後發現，年輕使用者申貸成功後，希望能進一步使用手機隨時管理個人每月收入與支出。因此凱基銀行即日起再度與 AlphaLoan 合作，推出「繳費鬧鐘」功能，使用者可以自行設定 LINE 推播，定時提醒在凱基、花旗、匯豐、遠東、玉山、安泰、三信等 7 家銀行的信貸還款日，以及 28 家銀行信用卡的帳單繳款日。

一旦皆受到繳費通知，按下「立即繳費」的按鍵，即可輕鬆完成這 7 家銀行或 28 家信用卡的繳款，如此一來，使用者無須再逐一登入各家銀行的行動銀行，於同一平台上即可統籌管理帳戶支出，透過開放 API 達到金融共享，真正落實「開放銀行」理念。

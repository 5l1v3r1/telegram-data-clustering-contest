Сервис напоминаний появится в популярном мессенджере

09 Dec 2019, 16:32

Мессенджер WhatsApp получил новую функцию, которая позволяет создавать задачи и получать соответствующие напоминания. Функция, созданная совместно с сервисом напоминаний Any.do, называется WhatsApp Reminders ("WhatsApp Напоминания") и доступна по платной подписке на всех устройствах с установленным мессенджером.

Для создания задачи необходимо отправить соответствующее сообщение боту Any.do с указанием времени напоминания. Кроме того для автоматического формирования задачи можно пересылать сообщения, полученные от других собеседников.

Ранее сообщалось о тестировании разработчиками WhatsApp новой функции, позволяющей автоматически удалять сообщения пользователя через определенный период времени.

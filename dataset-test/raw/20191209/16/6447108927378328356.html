Minecraft Bedrock edition will launch tomorrow on PS4, adds cross-play

09 Dec 2019, 16:15 by Aran Suddi

Sony has confirmed that Minecraft Bedrock edition will be released tomorrow on PS4, with this version of the game unifying the code across all versions of the game. This means that PS4 players will be able to play with those on Xbox One, Switch, mobile, and PC. Of course, Minecraft is already on PS4 and millions of people will own it already. Good news, current owners. Update 1.14 will add all the content that the Bedrock edition will have so no need to double-dip.

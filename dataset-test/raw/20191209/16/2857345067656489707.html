Nu patchar Microsoft Windows Mobile för sista gången

09 Dec 2019, 16:49

Inga fler säkerhetsuppdateringar för operativsystemet.

Under tisdagen den 10 december kommer Microsoft att uppdatera Windows 10 Mobile för sista gången, skriver ZDNet.

Det betyder att inga fler säkerhetsuppdateringar kommer släppas till företagets nedlagda mobilsatsning Windows Phone.

Windows 10 Mobile släpptes 2015 som en uppföljare till Windows Phone 8.1, men operativsystemet fick aldrig se någon större användarbas.

Microsoft har inte haft någon aktiv utveckling av Windows 10 Mobile sedan 2017 men fortsatt släppa patcher och säkerhetsuppdateringar till systemet fram tills nu.

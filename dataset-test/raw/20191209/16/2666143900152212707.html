Free Mobile : Une nouvelle MAJ pour la messagerie sous Android

09 Dec 2019, 16:41 by Isabelle Deromas

Et c’est la deuxième en moins d’une semaine.

Cette mise à jour apporte plusieurs correctifs à la gestion de la messagerie vocale sous Android, qui était affectée de plusieurs bugs. 

Comment mettre à jour ses applications sous Android ?

Vous pouvez mettre à jour les applications téléchargées et installées sur votre appareil Android une par une ou automatiquement. La mise à jour de vos applications vous permet d’accéder aux nouvelles fonctionnalités, et d’améliorer leur sécurité et leur stabilité.

Remarque : Si une mise à jour corrige une faille de sécurité importante, Google peut forcer la mise à jour de l’application concernée, et ce quels que soient ses paramètres ou ceux de votre appareil.

Pour mettre à jour automatiquement les applications de votre appareil Android, il suffit de procéder comme suit :

  1. Ouvrez l’application Google Play Store.
  2. Appuyez sur Menu   Paramètres.
  3. Appuyez sur Mettre à jour automatiquement les applis.

  4. Sélectionnez une option : 

    • Via n’importe quel réseau pour mettre à jour les applications via le Wi-Fi ou les données mobiles
    • Via le Wi-Fi uniquement pour ne mettre à jour les applications que lorsque vous êtes connecté au Wi-Fi

Remarque : Si un compte sur votre appareil présente une erreur de connexion, il est possible que les applications ne soient pas mises à jour automatiquement. 

Mettre à jour des applications Android individuelles automatiquement

Pour configurer la mise à jour de certaines applications sur votre appareil, procédez comme suit :

  1. Ouvrez l’application Google Play Store .
  2. Appuyez sur Menu   Mes jeux et applications.
  3. Sélectionnez l’application que vous souhaitez mettre à jour.
  4. Appuyez sur Plus .
  5. Appuyez sur Mise à jour auto.

L’application se met à jour automatiquement dès que des mises à jour sont disponibles. Pour désactiver les mises à jour automatiques, il suffit de décocher la case correspondante.

Mettre à jour les applications Android manuellement

Pour mettre à jour les applications individuellement ou de manière groupée avec l’application Google Play Store sur votre appareil mobile, procédez comme suit :

  1. Ouvrez l’application Google Play Store .
  2. Appuyez sur Menu   Mes jeux et applications.
  3. Les jeux pour lesquels une mise à jour est disponible sont signalés par la mention « Mettre à jour ». Vous pouvez également rechercher un jeu spécifique. 
  4. Appuyez sur Mettre à jour.

Remarque : Certaines applications nécessitent de nouvelles autorisations lorsqu’elles sont mises à jour.

Une notification vous invitant à accepter les nouvelles autorisations peut s’afficher.

Conseil : Dans certains cas, vous devrez peut-être redémarrer l’appareil pour mettre une application à jour.

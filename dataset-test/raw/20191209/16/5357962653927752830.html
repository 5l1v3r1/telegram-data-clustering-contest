Grupo pede criação de subprefeitura para atender demandas de Paraisópolis (SP)

Onze líderes de associações de moradores se reuniram, nesta segunda, em São Paulo, para discutir ações para a comunidade de Paraisópolis, onde nove jovens morreram após uma ação policial. Um dos representantes disse que a localidade cresceu muito nos últimos anos, mas não se desenvolveu em muitos setores. Uma das ideias é criar uma subprefeitura de Paraisópolis. Outra proposta é abrir o Parque Paraisópolis.

Linux Kernel 5.5开发启动 首个RC已发布

09 Dec 2019, 13:43 by raymon725

经过两个星期的合并、以及 Linux Kernel 5.4 内核系列的到位，现在我们终于迎来了 Linux Kernel 5.5 的首个首选发布版本。Linus Torvalds 本人刚刚宣布，Linux Kernel 5.5 rc1 已经开启公众测试。感兴趣的朋友，可通过 Kernel.org 官网下载体验。

（截图 via Softpedia）

作为该系列内核开发的首个里程碑，社区现已能够了解 Linux Kernel 5.5-rc1 的新功能和变更。Linus Torvalds 在周日下午表示：

合并窗口很是顺利，提交次数比早几次要略多一些，但没啥真正值得引起人们注意的地方。一半补丁与驱动程序有关，其次是 Arch 更新。

有兴趣参与驱动测试和错误汇报工作的朋友，不妨去 Kernel.org 官网下载体验一番。需要注意的是，作为一个预发行版本，请不要在任何生产环境中用来替换稳定版内核。

作为 2020 年发布的首个内核系列，Linux Kernel 5.5 将于 2020 年 1 月底或 2 月初正式与大家见面，具体取决于到时一共有多少个候选发布（RC）版本。

Lo nuevo de Microsoft es una app para Android que resuelve operaciones matemáticas por ti

09 Dec 2019, 15:01 by Christian Collado

Si las matemáticas no son lo tuyo, puede que la nueva app de Microsoft para móviles sea como un regalo de Navidad anticipado para ti. La compañía de Redmond ha lanzado la beta de Microsoft Math Solver, una nueva app para iOS y Android que permite resolver operaciones matemáticas de manera rápida y sencilla utilizando sistemas basados en inteligencia artificial.

Al igual que otras aplicaciones de este tipo como PhotoMath –que por cierto hace no mucho nombramos como una aplicación demasiado buena para ser gratuita–, la herramienta de Microsoft usa la cámara de nuestro teléfono para capturar la operación a resolver, y además de mostrar el resultado, ofrece los pasos que se deben seguir en cada caso para obtener la solución adecuada a cada problema.

Мэрия Черкесска закупила кресла для руководителей премиум-класса

29 Nov 2019, 11:47 by Анна Светлова

Согласно техзаданию, кресла должны быть с обивкой из натуральной кожи с эффектом патины

Черкесск, 29 ноября. Мэрия муниципального образования города Черкесска закупила для руководства кресла и стулья премиум-класса. Информация о проведенном аукционе размещена на сайте госзакупок.

Согласно представленной документации, мэрии Черкесска понадобилось оснастить предметами интерьера кабинеты в здании городской администрации. Чиновникам понадобились два кресла руководителя, семь кресел для проведения переговоров и два стула для брифинга. Также в закупку были включены вставки в существующий стол для переговоров. На эти цели мэрия Черкесска готова была потратить 1,275 млн рублей, это начальная цена закупки.

Как следует из технического задания, обивка первого кресла для руководителя должна быть из натуральной кожи премиум класса с эффектом патины, а на краях спинки должны быть использованы вставки из натурального дерева.

- Над основной частью подлокотника расположена накладка –полоса из гнутого дерева со вставками из натуральной кожи. Подлокотники должны быть массивные деревянные, с накладками из натуральной кожи, которые должны сочетаться с декоративными деревянными накладками на металлической крестовине с колесами, - отмечается в документации об электронном аукционе «Оснащение предметами интерьера кабинетов в здании мэрии муниципального образования города Черкесска».

Предполагается, что деревянные элементы кресла для руководителя должны быть обязательно изготовлены из древесины твердых пород цветовой гаммы «палисандр». Для особого комфорта чиновника кресло должно иметь регулировку по высоте и мультимеханизм качания с регулировкой жесткости качания. Также кресло должно выдерживать нагрузку не менее 150 кг. Средняя рыночная стоимость кресла для руководителя составляет порядка 143,6 тыс. рублей.

Второе кресло для руководителя еще дороже – 152, 3 тыс. рублей. Оно также должно иметь обтяжку натуральной кожей премиум класса с эффектом патины, иметь высокую спинку и подголовник. К особенным требованиям второго кресла для руководителя является способность выдерживать нагрузку не в 150, а в 200 кг. Подлокотники кресла должны быть деревянные, с мягкими накладками из натуральной кожи.

Средняя начальная цена одного кресла для переговоров в мэрии Черкесска составила 114 тыс. рублей, а стула для брифинга – 38,6 тыс. рублей.

Итоги аукциона на проведение закупки мебели для кабинетов мэрии были подведены 29 ноября. Победителем признана фирма «Стройград», предложившая поставить элитные кресла для чиновников за 420,75 тыс. рублей.

Ранее жители КЧР пожаловались на задержку выплаты социальных пособий. По их словам, проблема носит массовый характер.

5 συνήθειες που έχεις στο μπάνιο που (τελικά) κάνουν κακό στην υγεία σου

Αυτές τις συνήθειες που έχεις στο μπάνιο, πρέπει σιγά σιγά να τις αποβάλεις! Σου κάνουν κακό!

24 Nov 2019, 11:00 by neolaia.gr Team

Το μπάνιοείναι ένα μέρος που ενώ μας βοηθάει στην υγιεινή μας και στην καθαριότητα έχει και αρκετά μικρόβια. Επίσης, κάνουμε σε αυτό αρκετά πράγματα τα οποία δεν κάνουν καλό στην υγεία μας και στην υγιεινή του σώματος μας. Ξέρεις ποια είναι αυτά;

Είναι κάποιες συνήθειες που έχουμε όλοι και πρέπει να αρχίσουμε να τις αποφεύγουμε. Πάμε να δούμε ποιες είναι;

1. Κάποια σημεία του σώματός μας δεν τα πλένουμε

Είναι κάποια σημεία στην πλάτη μας, τα οποία όταν κάνουμε μπάνιο δεν τα πλένουμε. Και είναι λογικό γιατί, απλά, δεν τα φτάνουμε. Γι αυτό χρειάζεται να πάρουμε ειδικές βούρτσες ώστε να φτάνουμε τα σημεία αυτά και να καθαρίζουμε σωστά όλα μας το σώμα.

2. Αφήνεις την οδοντόβουρτσά σου στον πάγκο 

Το μπάνιο μας έχει αρκετά μικρόβια λόγο της υγρασίας που δημιουργείτε. Οπότε, για όποια κίνηση και να θέλεις να κάνεις, καλό είναι να μην ακουμπάς ούτε λεπτό την οδοντόβουρτσά σου στον πάγκο. Κράτα τη, ή βρες ένα ασφαλές μέρος για να την ακουμπάς.

3. Παίρνουμε διάφορα gadget στο μπάνιο

Ναι, μπορεί να πλένεις τα χέρια σου μετά, αλλά τα μικρόβια έχουν, ήδη, μεταφερθεί στο κινητό, το tablet ή οτιδήποτε άλλο έχεις μαζί σου στην τουαλέτα. Γι αυτό, απλά να έχεις στο μπάνιο σου περιοδικά, sudoku ή ότι άλλο θέλεις για να περάσει η ώρα σου. Κι όχι οποιοδήποτε gadget που μετά θα το μεταφέρεις και σε άλλο μέρος του σπιτιού.

4. Χρησιμοποιείς πετσέτα για το πρόσωπό σου

Ειδικά, τα άτομα που έχουν προβλήματα στο δέρμα τους πρέπει να το αποφεύγουν αυτό. Καλό είναι να χρησιμοποιούμε πετσέτες μίας χρήσης. Έτσι, θα αποφύγουμε την μεταφορά μικροβίων στο πρόσωπό μας, αλλά και θα έχουμε μία πιο καθαρή επιδερμίδα.

5. Αφήνουμε βρεγμένες πετσέτες στο μπάνιο

Είναι το μεγαλύτερο λάθος που κάνουμε ΟΛΟΙ. Τις πετσέτες μας πρέπει να τις βγάζουμε και να τις αερίζουμε. Να τις αφήνουμε να στεγνώνουν και μετά να τις βάζουμε στη θέση τους. Έτσι θα αποφύγουμε τη δημιουργία μικροβίων στις υγρές επιφάνειες τους.

Δες ακόμα: Τελικά κάθε πότε πρέπει να κάνουμε μπάνιο;

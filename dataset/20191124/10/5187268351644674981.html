Oracle lança o VirtualBox 6.1 RC 1 com suporte para o Kernel Linux 5.4

24 Nov 2019, 10:00 by Pedro Pinto

O VirtualBox é uma das plataformas gratuitas mais populares no campo da virtualização. Esta ferramenta permite aos utilizadores criar máquinas virtuais de uma forma muito simples.

A Oracle lançou agora o VirtualBox 6.1 RC 1 que chega com suporte para o Kernel Linux 5.4 e melhorias na interface gráfica.

Uma das tecnologias que tem marcado os últimos anos da computação é a virtualização. Esta tecnologia permite-nos, de uma forma simples, ter vários sistemas operativos a correr (simultaneamente) na mesma máquina, sem qualquer tipo de alterações.

Com o Virtualbox é possível virtualizar sistemas como o Windows, Linux e até mesmo MacOS. Em termos de funcionalidades, destaque para a possibilidade de realizar snapshots, partilhar pastas entre o sistema virtual e o nativo, suporte para drag and drop e a possibilidade de clonar facilmente máquinas virtuais.

Novidades do VirtualBox 6.1 RC

A primeira Release Candidate (RC) da versão 6.1 desta ferramenta já está disponível. O principal destaque vai para o suporte para o Kernel Linux 5.4. Além disso, a Oracle fez também pequenos ajustes ao nível da interface gráfica. Há muitas outras novidades que podem ser consultadas aqui.

A Oracle refere que não devem usar esta versão para máquina virtuais em produção. Esta é uma versão bleeding-edge, que ainda se encontra em atualização e correção de bugs. No entanto, podem sempre experimentar nas vossas máquinas e tirar partido das novidades.

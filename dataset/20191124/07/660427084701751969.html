Anggota RELA wajar dibekal senjata api

Cota tidak memadai untuk jaga keselamatan diri dan kampung

24 Nov 2019, 07:24 by ainulillia

BANGI, 27 Rabiulawal 1441H, Ahad – Timbalan Menteri Dalam Negeri, Datuk Mohd Azis Jamman berkata, kementeriannya menyokong cadangan membekalkan senjata api kepada anggota Jabatan Sukarelawan Malaysia (RELA).

Beliau berkata, pasukan RELA wajar diperkasakan dengan membenarkan anggotanya membawa senjata api apabila terbabit dalam operasi berisiko atau ditugaskan di kawasan yang dikenal pasti berdepan ancaman keselamatan.

Katanya, cota atau belantan yang digunakan anggota RELA terbabit operasi di kawasan berisiko tidak sesuai, berbanding ancaman keselamatan dihadapi mereka.

“Saya dimaklumkan setiap tahun anggota RELA diberi latihan dalam mengendalikan senjata api kerana ada operasi yang mengkehendaki mereka membawa senjata. Namun, kita perlu perkasakan lagi RELA, contoh dalam operasi berisiko.

“Mereka jangan dibekalkan cota saja tetapi juga senjata bukan saja menjaga keselamatan penduduk, malah untuk menjaga keselamatan diri mereka juga kerana cota saja tidak cukup,” katanya seperti dilapor BH Online

Beliau berkata demikian selepas program Bakti RELA dan Dialog Bersama Timbalan Menteri Dalam Negeri di Kampung Tatahan, Inanam, dekat sini, hari ini.

Mohd Azis berkata, beliau pernah menyentuh mengenai keperluan anggota RELA bertugas di Zon Keselamatan Khas Pantai Timur Sabah (ESSZONE).

“Saya sudah sentuh juga sebelum ini mengenai RELA di ESSZONE. Kawasan itu masih berbahaya. Sebab itu ada perintah berkurung di sana. 

“Jadi, anggota RELA di sana wajar dibekalkan senjata, bukan saja untuk menjaga keselamatan kampung, malah diri mereka,” katanya.

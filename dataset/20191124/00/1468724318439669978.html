ഇന്നത്തെ നക്ഷത്ര ഫലം (24-11-2019)

24 Nov 2019

# ബ്രഹ്മശ്രീ കാണിപ്പയ്യൂർ നാരായണൻ നമ്പൂതിരിപ്പാട്

അ​ശ്വ​തി : അ​ധി​കാ​ര​പ​രി​ധി വ​ർ​ധി​യ്ക്കും. വി​വി​ധ​ങ്ങ​ളാ​യ പ്ര​വ​ര്‍ത്ത​ന​ങ്ങ​ള്‍ ഏ​റ്റെ​ടു​ക്കും. പൂ​ര്‍വ്വി​ക​സ്വ​ത്ത് ഭാ​ഗം​വെ​യ്ക്കു​ന്ന​തി​ല്‍ വി​ട്ടു​വീ​ഴ്ച​യ്ക്കു ത​യ്യാ​റാ​കും.

ഭ​ര​ണി : വി​ദേ​ശ​യാ​ത്ര​യ്ക്ക് അ​നു​മ​തി ല​ഭി​യ്ക്കും. പ​റ​യു​ന്ന വാ​ക്കു​ക​ളും ചെ​യ്യു​ന്ന പ്ര ​വൃ​ത്തി​ക​ളും ഫ​ല​പ്ര​ദ​മാ​യി​ത്തീ​രും. ഭാ​വ​ന​ക​ള്‍ യാ​ഥാ​ര്‍ത്ഥ്യ​മാ​കും.

കാ​ര്‍ത്തി​ക : ആ​ത്മ​വി​ശ്വാ​സം വ​ർ​ധി​യ്ക്കും. വ്യ​വ​സ്ഥ​ക​ള്‍ പാ​ലി​യ്ക്കും. വാ​ക്കു​ക​ള്‍ ഫ ​ല​പ്ര​ദ​മാ​യി​ത്തീ​രും. സ​ജ്ജ​ന​സം​സ​ര്‍ഗ്ഗ​മു​ണ്ടാ​കും.

രോ​ഹി​ണി : വ്യ​വ​സ്ഥ​ക​ള്‍ക്ക​തീ​ത​മാ​യി പ്ര​വ​ര്‍ത്തി​യ്ക്കും. സ​ങ്കീ​ര്‍ണ്ണ​മാ​യ പ്ര​ശ്ന​ങ്ങ​ള്‍ക്ക് ശാ​ശ്വ​ത​പ​രി​ഹാ​രം ക​ണ്ടെ​ത്തും. വ​രു​മാ​നം വ​ർ​ധി​യ്ക്കും.

മ​ക​യി​രം : ന്യാ​യ​മാ​യ ആ​വ​ശ്യ​ങ്ങ​ള്‍ പ​രി​ഗ​ണി​യ്ക്ക​പ്പെ​ടും. പ്ര​വ​ര്‍ത്ത​ന​രം​ഗം മെ​ച്ച​പ്പെ​ടും. കാ​ലാ​ഹ​ര​ണ​പ്പെ​ട്ട ഗൃ​ഹോ​പ​ക​ര​ണ​ങ്ങ​ള്‍ മാ​റ്റി​വാ​ങ്ങു​വാ​നി​ട​വ​രും.

തി​രു​വാ​തി​ര : ഭാ​ര​വാ​ഹി​ത്വം ഏ​റ്റെ​ടു​ക്കു​വാ​ന്‍ നി​ര്‍ബ​ദ്ധി​ത​നാ​കും. ജാ​മ്യം നി​ല്‍ക്ക​രു​ത്. ദു​ശ്ശീ​ല​ങ്ങ​ള്‍ ഉ​പേ​ക്ഷി​യ്ക്കും. യാ​ത്രാ​വേ​ള​യി​ല്‍ പ​ണം ന​ഷ്ട​പ്പെ​ടാ​തെ സൂ​ക്ഷി​യ്ക്ക​ണം.

പു​ണ​ര്‍തം : വി​ട്ടു​വീ​ഴ്ചാ​മ​നോ​ഭാ​വ​ത്താ​ല്‍ ദാ​മ്പ​ത്യ​സൗ​ഖ്യ​മു​ണ്ടാ​കും. കു​ടും​ബ​മേ​ള​യി​ല്‍ പ​ങ്കെ​ടു​ക്കും. അ​മി​ത​വേ​ഗ​ത ഉ​പേ​ക്ഷി​യ്ക്ക​ണം.

പൂ​യം : ആ​ഗ്ര​ഹ​സാ​ഫ​ല്യ​മു​ണ്ടാ​കും. വി​ജ്ഞാ​ന​പ്ര​ദ​മാ​യ ച​ര്‍ച്ച​ക​ള്‍ ചെ​യ്യും. വി​ദ്യാ​ർ​ഥി ക​ള്‍ക്ക് അ​നു​കൂ​ല​സാ​ഹ​ച​ര്യം വ​ന്നു​ചേ​രും. ഓ​ര്‍മ്മ​ശ​ക്തി വ​ർ​ധി​യ്ക്കും.

ആ​യി​ല്യം : കു​ടും​ബ​ത്തി​ല്‍ സ്വ​സ്ഥ​ത​യും സ​മാ​ധാ​ന​വും ദാ​മ്പ​ത്യ​ഐ​ക്യ​വും ഉ​ണ്ടാ​കും. അ​സു​ഖ​മു​ണ്ടെ​ന്ന തോ​ന്ന​ല്‍ ഉ​പേ​ക്ഷി​യ്ക്ക​ണം. ആ​ത്മ​വി​ശ്വാ​സം കു​റ​യും.

മ​കം : ബ​ന്ധു​ഗൃ​ഹ​ത്തി​ലേ​യ്ക്ക് വി​രു​ന്നു​പോ​കും. മ​ത്സ​ര​രം​ഗ​ങ്ങ​ളി​ല്‍ വി​ജ​യി​യ്ക്കും. മാ​താ​പി​താ​ക്ക​ളു​ടെ ആ​ഗ്ര​ഹ​ങ്ങ​ള്‍ സാ​ധി​പ്പി​യ്ക്കും.

പൂ​രം : ആ​രോ​ഗ്യം തൃ​പ്തി​ക​ര​മാ​യി​രി​യ്ക്കും. ക​ഠി​ന​പ്ര​യ​ത്ന​ത്താ​ല്‍ കാ​ര്യ​സാ​ദ്ധ്യ​മു​ണ്ടാ കും. ​യാ​ത്രാ​ക്ലേ​ശ​ത്താ​ല്‍ അ​സ്വാ​സ്ഥ്യ​മ​നു​ഭ​വ​പ്പെ​ടും.

ഉ​ത്രം : സ​ന്താ​ന​സം​ര​ക്ഷ​ണ​ത്താ​ല്‍ സ​മാ​ധാ​ന​മു​ണ്ടാ​കും. പു​തി​യ വി​പ​ണ​ന-​വി​ത​ര​ണ ങ്ങ​ള്‍ ഏ​റ്റെ​ടു​ക്കും. സേ​വ​ന​സാ​മ​ര്‍ത്ഥ്യ​ത്താ​ല്‍ ആ​ഗ്ര​ഹ​നി​വൃ​ത്തി​യു​ണ്ടാ​കും.

അ​ത്തം : സൗ​ഹൃ​സം​ഭാ​ഷ​ണ​ത്തി​ല്‍ പു​തി​യ ആ​ശ​യ​ങ്ങ​ള്‍ ഉ​ത്ഭ​വി​യ്ക്കും. ആ​രോ​ഗ്യം തൃ​പ്തി​ക​ര​മാ​യി​രി​യ്ക്കും. ആ​ഗ്ര​ഹി​യ്ക്കു​ന്ന കാ​ര്യ​ങ്ങ​ള്‍ സാ​ധി​യ്ക്കും.

ചി​ത്ര : നി​സാ​ര​കാ​ര്യ​ങ്ങ​ള്‍ക്കു​പോ​ലും ത​ട​സം അ​നു​ഭ​വ​പ്പെ​ടും. ഭ​ക്ഷ്യ​വി​ഷ​ബാ​ധ​യേ​ല്‍ ക്കാ​തെ സൂ​ക്ഷി​യ്ക്ക​ണം. വാ​ഗ്വാ​ദ​ങ്ങ​ളി​ല്‍ നി​ന്നും ഒ​ഴി​ഞ്ഞു​മാ​റ​ണം.

ചോ​തി : ക​രാ​റു​ജോ​ലി​യി​ല്‍ ഒ​പ്പു​വെ​യ്ക്കും. ഏ​റ്റെ​ടു​ത്ത ചു​മ​ത​ല​ക​ള്‍ മ​ന​സം​തൃ​പ്തി യോ​ടു​കൂ​ടി പൂ​ര്‍ത്തീ​ക​രി​യ്ക്കും. പ്ര​വ​ര്‍ത്ത​ന​രം​ഗം മെ​ച്ച​പ്പെ​ടും.

വി​ശാ​ഖം : ധ​ര്‍മ്മ​പ്ര​വ​ര്‍ത്തി​ക​ള്‍ ,പ​രോ​പ​കാ​രം,ദാ​ന​ങ്ങ​ള്‍,പു​ണ്യ​പ്ര​വ​ര്‍ത്തി​ക​ള്‍ ഇ​വ​യി​ല്‍ സ​ര്‍വാ​ത്മ​ന സ​ഹ​ക​രി​യ്ക്കും. വാ​ഹ​നം മാ​റ്റി​വാ​ങ്ങു​വാ​ന്‍ തീ​രു​മാ​നി​യ്ക്കും.

അ​നി​ഴം : അ​ഭി​ലാ​ഷ​ങ്ങ​ള്‍സ​ഫ​ല​മാ​കും. വി​ശേ​ഷ​പ്പെ​ട്ട ദേ​വാ​ല​യ​ദ​ര്‍ശം ന​ട​ത്തു​വാ​നി​ട വ​രും. വി​ദേ​ശ​യാ​ത്രാ​നു​മ​തി ല​ഭി​യ്ക്കും. പൂ​ര്‍വി​ക​സ്വ​ത്ത് രേ​ഖാ​പ​ര​മാ​യി ല​ഭി​യ്ക്കും.

തൃ​ക്കേ​ട്ട : വ്യ​വ​സ്ഥ​ക​ള്‍ പൂ​ര്‍ത്തീ​ക​രി​യ്ക്കു​വാ​ന്‍ സാ​ധി​യ്ക്കു​ക​യി​ല്ല. ക​ര്‍ക്ക​ശ ബു​ദ്ധി ഒ​ഴി​വാ​ക്ക​ണം. വി​ട്ടു​വീ​ഴ്ച​യ്ക്കു ത​യ്യാ​റാ​കും.

മൂ​ലം : കു​ടും​ബ​ത്തി​ല്‍ സ്വ​സ്ഥ​ത​യും സ​മാ​ധാ​ന​വും ദാ​മ്പ​ത്യ​ഐ​ക്യ​ത​യും ഭ​ക്ഷ​ണ​സു​ഖ വും ​സു​ഖ​സു​ഷു​പ്തി​യും ക​ര്‍മ്മ​മേ​ഖ​ല​ക​ളി​ല്‍ പ്ര​വ​ര്‍ത്ത​ന​വി​ജ​യ​വും ഉ​ണ്ടാ​കും.

പൂ​രാ​ടം : പു​തി​യ പ്ര​വ​ര്‍ത്ത​ന​ങ്ങ​ള്‍ക്ക് തു​ട​ക്കം കു​റി​യ്ക്കും. വി​ത​ര​ണ​രം​ഗം വി​പു​ല​മാ ക്കും. ​അ​വ​ധി​യാ​ണെ​ങ്കി​ലും ജോ​ലി​ചെ​യ്യേ​ണ്ട​താ​യി​വ​രും. സ​ന്താ​ന​ങ്ങ​ളു​ടെ ആ​വ​ശ്യ​ങ്ങ​ള്‍ സാ​ധി​പ്പി​യ്ക്കും.

ഉ​ത്രാ​ടം : ധ​ര്‍മ്മ​പ്ര​വൃ​ത്തി​ക​ള്‍ക്കാ​യി പ​ണം ചി​ല​വ​ഴി​യ്ക്കും. സാ​മ്പ​ത്തി​ക​വി​ഭാ​ഗ​ത്തി​ല്‍ കൂ​ടു​ത​ല്‍ ശ്ര​ദ്ധ​കേ​ന്ദ്രീ​ക​രി​യ്ക്കു​വാ​ന്‍ നി​ര്‍ബ​ന്ധി​ത​നാ​കും. വി​ദേ​ശ ഉ​ദ്യോ​ഗ​ത്തി​ന് നി​യ​മ​നാ​നു​മ​തി ല​ഭി​യ്ക്കും.

തി​രു​വോ​ണം : പൊ​തു​പ്ര​വ​ര്‍ത്ത​ന​ങ്ങ​ള്‍ക്ക് നേ​തൃ​ത്വം ന​ല്‍കും. ച​ര്‍ച്ച​യി​ല്‍ വി​ജ​യി യ്ക്കും. ​പു​തി​യ ഉ​ദ്യോ​ഗ​ത്തി​ല്‍ പ്ര​വേ​ശി​യ്ക്കു​വാ​ന്‍ ദൂ​ര​യാ​ത്ര​പു​റ​പ്പെ​ടും.

അ​വി​ട്ടം : പു​തി​യ പ്ര​വ​ര്‍ത്ത​ന​ങ്ങ​ളെ​പ്പ​റ്റി ച​ര്‍ച്ച​ചെ​യ്യും. മം​ഗ​ള​വേ​ള​യി​ല്‍ പ​ങ്കെ​ടു​ക്കും. വി​ശ്വാ​സ​യോ​ഗ്യ​മാ​യ വ്യാ​പ​ര​ത്തി​ല്‍ പ​ണം മു​ട​ക്കും.

ച​ത​യം : കു​ടും​ബാം​ഗ​ങ്ങ​ളോ​ടൊ​പ്പം ഉ​ല്ലാ​സ​യാ​ത്ര​യ്ക്ക​വ​സ​ര​മു​ണ്ടാ​കും. മം​ഗ​ള​ക​ര്‍മ്മ ങ്ങ​ളി​ല്‍ പ​ങ്കെ​ടു​ക്കും. പ്രാ​യോ​ഗി​ക​വ​ശം ചി​ന്തി​ച്ചു പ്ര​വ​ര്‍ത്തി​യ്ക്കും.

പൂ​രോ​രു​ട്ടാ​തി : നി​ബ​ന്ധ​ക​ള്‍ പാ​ലി​യ്ക്കു​വാ​ന്‍ സാ​ധി​യ്ക്കും. പ്ര​വ​ര്‍ത്ത​ന​ക്ഷ​മ​ത വ​ർ​ധി യ്ക്കും. ​വാ​ക്കും പ്ര​വൃ​ത്തി​യും ഫ​ല​പ്ര​ദ​മാ​യി​ത്തീ​രും.

ഉ​ത്ര​ട്ടാ​തി : സൗ​മ്യ​സ​മീ​പ​ന​ത്താ​ല്‍ പ്ര​തി​കൂ​ല​സാ​ഹ​ച​ര്യ​ങ്ങ​ളെ അ​തി​ജീ​വി​യ്ക്കും. മം​ഗ ള​ക​ര്‍മ്മ​ങ്ങ​ള്‍ക്ക് നേ​തൃ​ത്വം ന​ല്‍കും. സ​മ്മാ​ന​പ​ദ്ധ​തി​ക​ളി​ല്‍ വി​ജ​യി​യ്ക്കും.

രേ​വ​തി : ആ​ത്മ​വി​ശ്വാ​സ​ത്തോ​ടു​കൂ​ടി പ​രീ​ക്ഷ, ഇ​ന്‍റ​ര്‍വ്യൂ തു​ട​ങ്ങി​യ​വ അ​വ​ത​രി​പ്പി​യ്ക്കു വാ​നാ​കും. ഉ​ദ്ദേ​ശി​ച്ച വി​ഷ​യ​ത്തി​ല്‍ ഉ​പ​രി​പ​ഠ​ന​ത്തി​ന് പ്ര​വേ​ശ​നം ല​ഭി​യ്ക്കും.

Bitcoin Core обновился до версии 0.19.0

24 Nov 2019 by LetKnow News

Разработчики наиболее популярного клиента для сети биткоина Bitcoin Core сегодня, 24 ноября, представили новую версию ПО. Об этом они сообщили в Twitter.

По теме:

Версия 0.19.0.1 уже доступна для скачивания. Она совместима с операционными системами Linux kernel, macOS 10.10 (+ более новые версии) и Windows 7 (+ более новые версии). В Bitcoin Core подчеркнули, что отныне клиент не поддерживает версию macOS старше 10.10, но в нем еще не добавлена поддержка темной темы в iOs.

В версию 0.19.0 включена функция reduce memory, которая улучшает конфигурацию клиента для запуска на устройстве с ограниченной памятью.

Разработчики добавили три удаленных вызова процедур (RPC):

- getbalances — для проверки подтвержденных и неподтвержденных балансов вместо использования разных RPC;

- setwalletflag — для включения и отключения свойств конкретного кошелька;

- getblockfilter — реализация фильтра BIP158 для указанного блока с использованием конфигурации blockfilterindex.

Среди новых функций — getblockfilter, обеспечивающий фильтры для блокчейна в соответствии с BIP158.

В новой версии GUI генерирует адреса формата bech32 по умолчанию, но пользователь может вручную изменить тип адреса. Также по умолчанию отключена поддержка BIP70.

Стоит отметить, что разработчики Bitcoin Core ранее пожаловались, что им недоплачивают.

Также они предупреждали, что откажутся от протокола BIP70.

Помимо этого, BitMEX выдала грант разработчику Bitcoin Core.

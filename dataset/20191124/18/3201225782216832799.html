Росбанк ​запустил услугу резервирования номера счета для предпринимателей​

24 Nov 2019, 18:16

Резервирование осуществляется по телефону после заявки на обратный звонок, которую можно оставить на сайте банка.

Росбанк запустил новую услугу для предпринимателей — резервирование расчетного счета. Теперь будущий клиент может заранее узнать номер счета в выбранном отделении, следует из релиза кредитной организации.

Резервирование осуществляется по телефону после заявки на обратный звонок, которую можно оставить на сайте банка, заполнив одно поле — контактный номер телефона. После чего менеджер «бронирует» номер счета для клиента и выбирает удобное отделение для подписания документов.

«Благодаря новой услуге по резервированию расчетного счета клиенту больше не придется ждать его открытия для получения реквизитов: непосредственно сразу после того, как счет будет забронирован, клиент получит его реквизиты по СМС и сможет направить их контрагентам. Затем клиенту останется только в любое удобное время подписать договор с банком и в этот же день начать полноценно работать в нашей системе дистанционного банковского обслуживания PRO ONLINE», — прокомментировал директор департамента малого бизнеса Росбанка Дмитрий Смирнов.

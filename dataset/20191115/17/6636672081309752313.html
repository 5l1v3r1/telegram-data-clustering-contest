Kitronik ARCADE brings a programmable gamepad to kids hands

15 Nov 2019, 17:04 by Alun Williams

This one looks impressive – the Kitronik ARCADE, encouraging kids to code their own game that they can play on the handheld console. Think of it as Nintendo Switch meets Scratch….

For the programming it uses Microsoft’s MakeCode, which is a kid-friendly, colour-themed graphical block-structured approach to coding blocks. You could think of it as Redmond’s answer to MIT’s Scratch.

It is what is used on the BBC Micro:bit devices, and Kitronik has lots of experience around the Micro:bit (see here and here, for example).

Specifically, it uses the MakeCode Arcade Block Editor to host and edit the code, which is downloaded to the device. Although you can also use Javascript and a Python editor is promised soon.

Give it a go, to see how easy arm intuitive it is: arcade.makecode.com

The device features:

  • A full colour LCD wide viewing angle screen
  • A clear case – all electronics are on show but protected from everyday knocks
  • A piezo sounder for audio feedback
  • 6 gamer input buttons
  • A menu button
  • A reset button and an on/off switch
  • Vibration motor for Haptic feedback
  • Software volume control
  • A USB programming port and two expansion ports for expert level users

It’s powered by either 3xAA batteries or via the micro USB connector

You can find details of the device on the Kitronik website, with the full data sheet here (it runs on an Atmel Atmel SAMD51J19A, the LED screen is 160 x 128 and there are two expansion ports, for example).

Pricing starts from £29.95 (£35.94 including VAT).

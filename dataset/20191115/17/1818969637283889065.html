Для Windows 10 Mobile неожиданно вышло новое обновление

Несмотря на то, что Microsoft уже назвала дату "похорон" мобильной версии Windows, компания выпустила обновление программного обеспечения для смартфонов.

15 Nov 2019, 17:51 by 24tv.ua

Заметим, апдейт задел практически все ключевые элементы прошивки.

Что известно об обновлении?

Новый патч под кодовым номером KB4522811 включая "заплатки" для устранения критических уязвимостей в операционной системе Windows 10 Mobile и некоторых системных приложениях, сообщает Wccftech.

Обновления затрагивает службы Windows App Platform, Microsoft Graphics Component, Windows Input and Composition, Windows Fundamentals, Windows Cryptography и Windows Virtualization, браузер Microsoft Edge, а также другие сервисы. 

Когда похоронят мобильную Windows?

Скорее всего, апдейт станет последним для операционной системы – поддержка Windows 10 Mobile истекает 10 декабря 2019 года. После этого срока компания прекратит выпуск апдейтов системы безопасности и регулярных пакетов обновлений мобильной ОС.

Что будет с магазином приложений?

Ранее Microsoft сообщала, что магазин приложений для Windows-смартфонов продолжит функционировать и после истечения указанного срока.

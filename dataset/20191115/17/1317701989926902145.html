.NET Core 3.1 Preview 3 und Visual Studio 2019 Preview 5 stehen zum Download bereit

15 Nov 2019, 17:05 by moinmoin

Noch kurz nachgereicht: Gestern wurde die .NET Core 3.1 Preview 3 bereitgestellt. Sie enthält ein paar Verbesserungen in Bezug auf Blazor und Windows Desktop, die in der .NET Core 3.0 eingeführt wurden.

Lange dauert es nun nicht mehr bis die finale Version von .NET Core 3.1 erscheinen wird. Im Dezember soll es dann soweit sein. Wenn denn nichts dazwischen kommt. Auch das Visual Studio 2019 hat ein Update auf die Preview 5 erhalten. Die Liste der behobenen Probleme ist hier um einiges länger.

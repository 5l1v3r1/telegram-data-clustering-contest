Facebook is making donations easier for nonprofit organisations by rolling out three new features

15 Nov 2019, 11:42

There are crowdfunding platforms out there like Kickstarter and Crowdfund which are doing a good job helping many nonprofit organisations to start up on their own. These platforms have made donations and fundraising easier to access online and organisations have gained reputation through them as well. Facebook has now added some of the new features for donations and fundraising across its platforms.

There are three new features for Facebook which are a donation button for Instagram, fundraising stickers on Facebook, and Charity Live Streaming Tools for Facebook Gaming Creators. These features will come useful when organisations want to gain funds on these widely-used social media platforms.

Instagram's new donate button

The donation button for Instagram will be added into Instagram profile and you can locate it next to the contact button. This is said to be the change that would generate the most donations among these three new features. 

Instagram used to allow users to add ‘donation stickers’ in their Stories, but the Stories will disappear after 24 hours so it is not that effective for users who want to do a long term fundraising campaign. The ‘donation stickers’ will now be coming to Facebook as well. 

Screenshot of how Charity Live Streaming Tools work.

Another update is to provide a wider rollout of fundraising for gaming live streams on Facebook. The Charity Live Streaming Tools will allow gaming creators to set a donation goal to raise funds from their fans. They are also able to choose from a list of charities available in their regions to start fundraising events.

These features have made donations easier for both fundraisers and users across Facebook’s various platforms. There is also proof that can show these features are working well as 25 gaming creators have raised more than 180,000 USD (~RM747,090) for charity last year in the US while live streaming. With more of these features rolling out, raising funds for good causes would be fast and easy to access.

Microsoft создала резервную копию GitHub на случай апокалипсиса

15 Nov 2019, 07:30 by Руслан Фадеев

О предполагаемой судьбе IT-индустрии в случае наступления апокалипсиса задумались не только энтузиасты. Компания Microsoft решила сохранить всё содержимое принадлежащего ей веб-сервиса GitHub в специальном бункере, в теории способном пережить даже конец света.

Резервную копию GitHub корпорация разместит в норвежском хранилище Arctic World Archive, расположенном в заброшенной угольной шахте на архипелаге Шпицберген. С 2017-го там хранятся бэкапы другой информации — архивов Ватикана, фильмов и прочих данных, которые могут представлять интерес для человечества, если тому придётся пережить глобальную катастрофу.

Поскольку цифровые носители в постапокалиптическом мире в теории окажутся бесполезны, всё содержимое GitHub, включая код Linux, Android и тысяч различных приложений, было записано на специально разработанную плёнку. Прочесть данные можно будет с помощью обычного увеличительного стекла. Ожидается, что даже при утере всех жёстких дисков на планете такой архив сможет восполнить пропажу ценной технической информации.

Утверждены субсидируемые авиарейсы из Екатеринбурга

15 Nov 2019, 08:33 by Евгения Скачкова

Росавиация утвердила список субсидируемых авиарейсов на 2020 год. В перечне - 20 направлений из Екатеринбурга. 

Согласно опубликованному приказу, условия будут действовать с первых дней января и до конца 2020 года. Путешественникам будут доступны следующие субсидируемые полёты:

  • Екатеринбург - Белоярский (авиакомпания "РусЛайн", количество мест - 50, количество рейсов в неделю - 2)
  • Екатеринбург - Горно-Алтайск (авиакомпания "РусЛайн", количество мест - 50, количество рейсов в неделю - 2)
  • Екатеринбург - Калуга (авиакомпания "РусЛайн", количество мест - 50, количество рейсов в неделю - 2)
  • Екатеринбург - Надым (авиакомпания "Ямал", количество мест - 100, количество рейсов в неделю - 2)
  • Екатеринбург - Нарьян-Мар (авиакомпания "РусЛайн", количество мест - 50, количество рейсов в неделю - 2)
  • Екатеринбург - Нижневартовск (авиакомпания "РусЛайн", количество мест - 50, количество рейсов в неделю - 2)
  • Екатеринбург - Нижний Новгород (авиакомпания "ИрАэро", количество мест - 103, количество рейсов в неделю - 3)
  • Екатеринбург - Новокузнецк (авиакомпания "ЮВТ АЭРО", количество мест - 50, количество рейсов в неделю - 2)
  • Екатеринбург - Новый Уренгой (авиакомпания "Ямал", количество мест - 100, количество рейсов в неделю - 2)
  • Екатеринбург - Ноябрьск (авиакомпания "Ямал", количество мест - 100, количество рейсов в неделю - 2)
  • Екатеринбург - Нягань (авиакомпания "Ямал", количество мест - 50, количество рейсов в неделю - 2)
  • Екатеринбург - Омск (авиакомпания Utair, количество мест - 70, количество рейсов в неделю - 2)
  • Екатеринбург - Оренбург (аэропорт Оренбург, количество мест - 17, количество рейсов в неделю - 3)
  • Екатеринбург - Салехард (авиакомпания "Ямал", количество мест - 100, количество рейсов в неделю - 2)
  • Екатеринбург - Саранск (авиакомпания "РусЛайн", количество мест - 50, количество рейсов в неделю - 2)
  • Екатеринбург - Саратов (авиакомпания "ИрАэро", количество мест - 103, количество рейсов в неделю - 3)
  • Екатеринбург - Сыктывкар (авиакомпания "Руслайн", количество мест - 50, количество рейсов в неделю - 3)
  • Екатеринбург - Тамбов (авиакомпания "Руслайн", количество мест - 50, количество рейсов в неделю - 2)

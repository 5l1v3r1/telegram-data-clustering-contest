Microsoft несподівано оновила 'мертву' Windows Mobile - Радіо Максимум

15 Nov 2019, 22:54 by Лазур Назарій

Попри оголошену раніше дату "похорону" мобільної версії Windows, компанія Microsoft випустила оновлення програмного забезпечення для смартфонів під управлінням операційної системи, яка завершає свій життєвий цикл.

Апдейт торкнувся практично всіх ключових елементів прошивки.

І ще:

Новий патч під кодовим номером KB4522811 містить "заплатки" для усунення критичних вразливостей в операційній системі Windows 10 Mobile і деяких системних додатках.

Оновлення зачіпає служби Windows App Platform, Microsoft Graphics Component, Windows Input and Composition, Windows Fundamentals, Windows Cryptography і Windows Virtualization, браузер Microsoft Edge, а також інші сервіси.

Найімовірніше, апдейт стане останнім для операційної системи – підтримка Windows 10 Mobile закінчується 10 грудня поточного року. Після цього терміну компанія припинить випуск апдейтів системи безпеки та регулярних пакетів оновлень мобільної ОС.

Раніше Microsoft повідомляла, що магазин додатків для Windows-смартфонів продовжить функціонувати і після закінчення зазначеного терміну.

Самые популярные языки программирования 2019 года — данные GitHub

11 Nov 2019, 09:45

Сервис GitHub выпустил ежегодный отчет State of the Octoverse, который включает в себя рейтинг популярности языков программирования. На работу обратил внимание Business Insider.

В рейтинг 2019 года вошли:

  1. JavaScript
  2. Python
  3. Java
  4. PHP
  5. C#
  6. C++
  7. TypeScript
  8. Shell
  9. C
  10. Ruby

В этом году C # и Shell поднялись на строчку вверх списка, а Python впервые обогнал Java. В GitHub отмечают, первенство JavaScript можно объяснить тем, что это один из основных языков, используемых для создания веб-страниц: с его помощью можно делать интерактивные сайты и игры.

Среди языков, которые быстрее всего набирают популярность — Dart, Rust, HCL.

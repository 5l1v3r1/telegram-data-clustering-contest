Fujitsu обновила линейку однопроцессорных серверов PRIMERGY

11 Nov 2019, 14:50

Fujitsu анонсировала обновленный модельный ряд серверов начального уровня Fujitsu PRIMERGY, разработанных для компаний малого и среднего бизнеса.

Новые модели поддерживают новейшие процессоры и увеличенный объем памяти, что делает однопроцессорные серверы PRIMERGY экономически эффективной альтернативой облачным решениям, особенно при обработке таких офисных задач, как общие базы данных, и для использования в граничных сетевых средах, где задержки и пропускная способность создают сложность для эффективной работы облачных решений.

Новейшие процессоры Intel Xeon E-2200 используются во всей линейке однопроцессорных систем, что значительно увеличивает их скорость работы. Максимальный объем памяти увеличен до 128 ГБ. Кристиан Лейтнер (Christian Leutner), директор по продажам компании Fujitsu в Европе, сказал: «Времена, когда для создания надежной универсальной системы необходимо было использовать двухпроцессорные серверы, остались в прошлом. Современные однопроцессорные серверы PRIMERGY обеспечивают высокую производительность и гибкость работы. Обновленная линейка систем Fujitsu PRIMERGY с расширенными возможностями для увеличения емкости памяти и повышения скорости обработки данных разработана для того, чтобы стать эффективным решением для развития бизнеса компаний любого масштаба, гарантируя, что у них есть технологии, способные справиться с любыми будущими задачами».

Ультракомпактные PRIMERGY TX1320 M4 представляют собой полнофункциональные серверы для обработки отдельных или виртуализированных рабочих нагрузок, которые можно установить практически в любом месте. Как и все однопроцессорные системы PRIMERGY, они отличаются бесшумной работой и поддерживают технологию Fujitsu Cool-safe Advanced Thermal Design для использования в различных температурных условиях.

Серверы PRIMERGY TX1330 M4 поддерживают расширенные возможности для увеличения объема памяти и емкости дисковой системы для эффективной обработки различных стандартных рабочих нагрузок. Они установили новый рекорд по результатам теста SPECpower_ssj 2008, который используется для определения характеристик производительности и потребления энергии вычислительного оборудования серверного класса.

Серверы PRIMERGY RX1330 M4 замыкают линейку однопроцессорных систем. Они предназначены для установки в стойку и поддерживают различные функции для соответствия требованиям будущего, сочетая в себе высокую производительность и доступную цену. Эта модель устанавливает новый рекорд по результатам теста VMmark, который используется для оценки производительности, масштабируемости и энергопотребления платформ виртуализации, и новый мировой рекорд по результатам теста SPEC CPU2017 для целочисленных вычислений по показателям скорости.

Все три модели серверов поддерживают пакет ПО Fujitsu Infrastructure Manager (ISM) Essential и iRMC S5 для простого и эффективного управления на протяжении всего жизненного цикла систем, включая развертывание, установку и администрирование.

Новые серверы PRIMERGY можно заказать у официальных дистрибуторов компании Fujitsu.

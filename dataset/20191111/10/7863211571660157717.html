Octoverse 2019: аудитория Github перевалила за 40 млн, Python впервые обошёл Java

11 Nov 2019, 10:49 by Александра Кукуть

Согласно свежему отчёту Octoverse, GitHub насчитывает более 40 млн разработчиков, причём в 2019 году к сервису присоединилось 10 млн, пишет ZDNet.

За последний год на GitHub появилось более 44 млн репозиториев, а свой первый репозиторий создало на 44 процента больше разработчиков, чем годом ранее. Также было принято свыше 87 млн пулл-реквестов, и на 28 процентов больше разработчиков, чем в 2018 году, отправили свой первый пулл-реквест.

Основные выводы отчёта:

  • Почти 80% пользователей GitHub — не из США. lodash/lodashexpressjs/expressvisionmedia/debug
  • Увеличилась популярность репозиториев по data science: глубокому обучению, обработке естественного языка и машинному обучению.
    Более половины среди наиболее популярных публичных репозиториев (по количеству звёзд) построены на numpy, многие — на базе scipy, scikit-learn, и TensorFlow; выросло число научных статей. Показатели использования блокнотов Jupyter Notebooks более чем удваиваются третий год подряд. Kotlin
  • Самые крупные проекты по числу участников — Microsoft Visual Studio Code (19,1 тысячи), Microsoft Azure (14 тысяч) и Flutter (13 тысяч).

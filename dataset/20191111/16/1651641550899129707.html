Microsoft now offers the full EMS E5 suite for US Government customers

11 Nov 2019, 16:15 by Pradeep

Microsoft was already offering some services which are part of its Enterprise Mobility + Security (EMS) E5 product suite for US Government customers. Microsoft today announced that Microsoft Cloud App Security and Azure Advanced Threat Protection (ATP) are now available for US Government GCC High customers. With this expanded availability, Microsoft now offers its full EMS E5 product suite for US Government customers. Microsoft also mentioned that Office 365 Cloud App Security and Cloud Discovery are also available for US Government customers. Read about Microsoft Cloud App Security and Azure Advanced Threat Protection below.

  • Microsoft Cloud App Security is a Cloud Access Security Broker (CASB) that supports various deployment modes including log collection and API connectors. It provides rich visibility, control over data travel, and sophisticated analytics to identify and combat cyberthreats across your Microsoft and third-party cloud services.
  • Azure ATP is a cloud-based security solution that leverages on-premises Active Directory signals to identify, detect, and investigate advanced threats, compromised identities, and malicious insider actions directed at your organization.

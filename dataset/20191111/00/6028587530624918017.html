Apple Watch

The latest generation of Apple Watch, Series 5, was released in September, so it’s the one model which may not receive big discounts this year. There are some, though. And this is certainly the best watch from Apple so far, with its always-on display the main upgrade this time.

Series 3, which premiered in 2017, is already a real bargain now – prices start from $199 – so there may not be huge discounts but any there are will make this already keenly priced gadget even more desirable.

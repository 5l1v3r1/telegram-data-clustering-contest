Esports: Berbagai Bonus dan Hadiah Menarik Diluncurkan PUBG Mobile dalam Harbolnas 2019

08 Nov 2019, 22:08 by Bola.net

Bola.net - PUBG Mobile meluncurkan program 11.11 untuk para pemain dan pencinta game bergenre battle royal terfavorit di Asia Tenggara. Program tersebut diluncurkan dalam rangka menyemarakkan Hari Belanja Online Nasional (Harbolnas) 11.11.

Beragam program dan bonus menarik disediakan oleh PUBG Mobile untuk para pemain dan pencinta game battle royal terlaris ini dengan tema Player Festival - #Fight4One, seperti discount lucky star, voucher log-in, reward giveaway, lalu Bonus berlipat setiap kali top-up. Untuk discount lucky star, program ini akan berlangsung pada 8-14 November.

Dalam program ini, para pemain bisa mendapatkan diskon-diskon menarik di lobby game event center dengan beragam besaran jumlah diskon yang hanya dapat digunakan pada hari tersebut saja. Lalu kedua, voucher log-in, yang digelar pada 8-11 November.

Untuk program voucher log-in, para pemain berkesempatan mendapatkan voucher hingga 250UC, serta PUBG Mobile akan merilis Outfit UC Bundle pada 11 November dan juga menyediakan kejutan telur emas untuk para pemain yang menghabiskan UC pada hari tersebut. Selain itu, PUBG Mobile juga mengadakan reward giveaway.

Reward giveaway ini merupakan pembagian hadiah dalam game menarik yang disediakan PUBG Mobile kepada para pemainnya. Pada 11 November, para pemain yang menggunakan 100UC berkesempatan untuk mendapatkan tiga telur emas, serta memiliki kesempatan untuk mendapatkan palu setelah menyelesaikan misi mingguan, dimana palu dan telur nantinya dapat digunakan untuk mendapatkan limited items.

PUBG Mobile juga mengadakan bonus berlipat setiap kali top-up di mana dalam program ini bekerjasama Midasbuy. PUBG Mobile memberikan kesempatan kepada para pemain untuk mendapatkan extra top-up dalam setiap kali pembelian top up 100UC maka para pemain akan mendapatkan bonus hingga 100UC dengan maksimal pembelian 100UC.

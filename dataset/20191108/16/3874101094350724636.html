Microsoft'tan Robot İşletim Sistemine Geliştirmeler

Microsoft, özel işletim sistemi ROS'a getirdiği yenilikleri resmi olarak duyurdu!

08 Nov 2019, 16:00 by CHIP Online

Microsoft ve Open Robotics'in robot işletim sistemi ROS, bu senenin başlarında kullanıma sunulmuştu. Şimdi ise Microsoft, ROS'un yeteneklerini geliştirdiğini duyurdu.

Duyuruya göre ROS, yeni bir Visual Studio Code uzantısı, Azure VM ROS şablonu ile geliştirildi. VS Code uzantısının yetenekleri arasında otomatik çevre yapılandırması, ROS çalışma zamanı durumunu izleme, başlatma ve durdurma, yapı görevlerinin otomatik olarak keşfedilmesi, tek tıklamayla ROS paketi oluşturma, Linux ROS geliştirme var.

Bunların yanında C++ ve Python uzantılarını kullanan ROS düğümleri de artık hata ayıklayabilecek.

Son olarak ROS ve ROS2 desteğinin Microsoft destekli ROS düğümleri sayesinde önemli ölçüde yükseltildiği belirtiliyor. Yeni teknolojiler hakkında daha fazla bilgi sahibi olmak isteyenler, Robot Operating System'e direk olarak buradan geçiş yapabilirler.

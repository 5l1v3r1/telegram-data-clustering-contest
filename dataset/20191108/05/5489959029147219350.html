Esports: PUBG Turut Menyemarakkan Harbolnas

08 Nov 2019, 05:21 by Bola.net

Bola.net - PUBG Mobile tak mau ketinggalan menyemarakkan program 11.11 dalam rangka Hari Belanja Online Nasional 11.11. Game bergenre battle royal terfavorit di Asia Tenggara itu meluncurkan program khusus bertajuk ”Fight4One”.

Dalam program tersebut, PUBG Mobile akan memberikan beragam bonus menarik untuk para pencinta permainan ini. Direktur Pemasaran PUBG Mobile Asia Tenggara, Oliver Ye menjelaskan, program 11.11 yang digagasnya itu merupakan bentuk rasa penghormatan kepada para pemain dan pencinta PUBG Mobile di Indonesia.

Selain bisa menikmati keseruan dalam memainkan game, para peserta juga dijanjikan berbagai hadiah menarik. Hal itu diyakini bakal menjadi tawaran menarik dari PUBG Mobile.

”Kami mengapresiasi serta mencoba memanjakan para pemain dan mengajak para pemain untuk memperkuat ikatan persahabat dengan squad dan juga pemain lainnya.” ujar Oliver Ye dalam rilis yang diterima Bola.net.

MediaTek MT6885 5G SoC will integrate the M70 baseband
3.7 (73.33%) 3 votes

According to a recent report, MediaTek will soon release its first 5G SoC which integrates the 5G baseband. From the reports, the chip model is MT6885 and it will integrate the M70 5G baseband. At the COMPUTEX2019 conference, MediaTek announced that its next 5G chip will integrate the 5G baseband.

According to reports, this chip uses the 7nm process technology. In addition, it is the world’s first smartphone chip that uses ARM’s latest and fastest Cortex A77 CPU and Mali-G77 GPU. More importantly, it integrates the Helio M70 5G modem with 4.7Gbps download speed and 2.5 Gbps upload speed. It further supports 2G, 3G, 4G, 5G connectivity as well as a dynamic power allocation.

This MediaTek 5G mobile platform is energy-efficient, and the design can achieve higher transmission speeds with lower power consumption. It creates a comprehensive ultra-high-speed 5G solution for terminal handset manufacturers. In addition, it uses a new AI architecture, equipped with a new independent AI processing unit APU which supports more advanced AI applications. The chip also supports image processing technology that eliminates image blur. Thus even when the object is highly unstable, the user can still take a wonderful photo.

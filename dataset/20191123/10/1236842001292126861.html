Microsoft ведёт переговоры о разработке эксклюзивов для стриминговой платформы Project xCloud

23 Nov 2019, 10:13 by Антон Самитов

Но на данный момент анонсировать нечего.

Карим Чаудри, вице-президент по облачным технологиям в Microsoft, подтвердил в интервью, что компания рассматривает возможность выпускать игры, эксклюзивные для сервиса Project xCloud.

Он подчеркнул, что не может рассказать о каких-то конкретных планах, так как переговоры со студиями-разработчиками только начались. Пока что библиотеку xCloud собираются расширять за счёт игр для Xbox, которые уже готовы для запуска в «облаке».

Что касается эксклюзивов, то сейчас идут первые этапы переговоров с внутренними и внешними студиями, но анонсировать нам пока нечего.

Новый контент по новым IP — это как минимум полтора-два года работы.Карим Чаудри
вице-президент по Project xCloud

Чаудри добавил, что в Xbox SDK уже внесли нужные изменения для работы с xCloud и добавили инструменты, которые позволят разработчикам адаптировать игры для стриминга. Это касается и смены размера шрифтов для разных экранов, и переработки сетевого кода под серверы облачного сервиса.

О расширении тестирования Project xCloud объявили во время X019 в Лондоне. Пока что оно доступно в США, Великобритании и Южной Корее, а в библиотеке доступно больше 50 игр. Полноценный запуск на ПК состоится в 2020 году.

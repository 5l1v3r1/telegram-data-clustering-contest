"คิม-หมาก" ขอข่าวลือแต่ง ดีกว่าข่าวลือเลิก จับมือเก็บเงินสร้างครอบครัว

23 Nov 2019, 03:01 by ผู้จัดการออนไลน์

“หมาก-คิม” ยืนยันยังไม่มีงานวิวาห์ในเร็ววันนี้ เปรยมีข่าวแต่งดีกว่าข่าวเลิก คุยเรื่องแต่งไม่จริงจัง ฝ่ายชายลั่นอนาคตสร้างเรือนหอต้องให้สบายทั้งลูก-เมีย

นานๆ จะได้เห็นคู่รักสุดหวานอย่างพระเอกหนุ่ม “หมาก ปริญ สุภารัตน์” กับนางเอกสาว “คิมเบอร์ลี่ แอน เทียมศิริ” ออกงานคู่กันสักที ยิ่งช่วงนี้มีข่าวหนาหูว่าอาจจะมีงานวิวาห์เร็วๆ นี้หรือเปล่า เพราะมีการลงทุนซื้อที่ด้วยกันแล้ว ล่าสุดทั้งคู่ก็มาร่วมงานเปิดตัว เดียร์ทัมมี่ อาณาจักรใหม่แห่ง ไลฟ์สไตล์ซูเปอร์มาร์เก็ต และ 7 สเปเชียลตี้สโตร์ ณ DEAR TUMMY ชั้น G ICONSIAM ก็พากันดับฝันแฟนๆ อีกรอบ โดยยืนยันว่ายังไม่มีข่าวดีในเร็วๆ นี้แน่นอน

คิมเบอร์ลี่ : “เรื่องข่าวลือว่าเราใกล้จะแต่งงาน มีออกมาเยอะมากค่ะ มีทั้งข่าวว่าเลิกกัน ข่าวว่าแต่งงาน แต่เอาอันนี้ดีกว่าเนอะ (หัวเราะ)”
หมาก : “ข่าวบอกว่ามีแพลนแต่งงานเป็นช่วงปีหน้า คือเราตอบได้แค่ว่าตอนนี้เราเสถียรอยู่ครับ เรื่อยๆ ไปก่อน ยังไม่รู้หรอกว่าเมื่อไหร่”
คิมเบอร์ลี่ : “ใช่ ก็รอลุ้นกันไปค่ะ จะปีหน้ามั้ยก็ยังไม่รู้เลยค่ะ อันนี้ก็ตอบไม่ได้จริงๆ แต่ที่รู้ๆ กันตอนนี้คือเรามีงานที่ต้องรับผิดชอบ ต้องทำงานของเรากันก่อนค่ะ”

เผยมีการพูดคุยกันจริง แต่แค่คุยกันเล่นๆ ยังไม่มีอะไรจริงจัง
หมาก : “เราก็คุยกันเท่านั้นเองครับ แพลนๆ ไว้คร่าวๆ เฉยๆ”
คิมเบอร์ลี่ : “ก็มีคุยไว้ถึงอนาคตค่ะ คุยกันเล่นๆ”
หมาก : “อย่างที่บอกไปเราก็ต้องทำงาน เก็บเงิน เลี้ยงครอบครัว เป็นหัวหน้าครอบครัว เราต้องรับผิดชอบตรงนี้ก่อนครับ ปลายปีหน้ามีลุ้นไหม ก็ยังตอบไม่ได้เลยครับ ผมดูเหมือนมีพิรุธเหรอ ไม่มี (เสียงสูง) ไม่มี (หัวเราะ)”
คิมเบอร์ลี่ : “ไม่มีหรอก เดี๋ยวปีหน้าค่อยมาถามใหม่ดีกว่า (หัวเราะ)”

บอกที่ไปหา “มาร์กี้ ราศรี บาเลนซิเอก้า จิราธิวัฒน์” บ่อยๆ เพราะรักหลาน
คิมเบอร์ลี่ : “รักหลานไง”
หมาก : “รักเด็กๆ”
คิมเบอร์ลี่ : “คือถามว่าปรึกษากี้ไหม มีแต่กี้นั่นแหละที่บอกให้รีบๆ ตามมาอะไรแบบนี้ กลัวลูกเหงา (หัวเราะ) แต่ถามว่าอยากมีเป็นของตัวเองไหม ก็คงยังไม่ใช่ตอนนี้ค่ะ ตอนนี้ต้องฟิตหุ่นก่อน อยากมีลูกแฝดไหม ก็ไม่ได้บังคับค่ะ คือยังไม่ได้คิดไว้ (หัวเราะ) ตอบหลายครั้งแล้วเหมือนกันนะ ไม่รู้จะตอบอะไรแล้ว (หัวเราะ)”
หมาก : “ถามว่าพอมีข่าวต่างๆ เข้ามา งานแต่งเริ่มเป็นรูปเป็นร่างขึ้นไหม ก็ยังครับ มันเป็นเรื่องของอนาคตจริงๆ”

บอกบ้านที่ซื้อไว้ด้วยกันยังไม่ใช่เรือนหอแน่ๆ เพราะคิดจะเอาไว้เป็นที่พักตากอากาศเฉยๆ
หมาก-คิม : “ยังไม่ใช่เรือนหอ (หัวเราะ)”
หมาก : “เราก็แค่คุยกันเฉยๆ ว่าพอเอาเข้าจริงๆ แล้วเราก็ต้องทำอะไรให้สะดวกสบายทั้งกับเราด้วย กับลูกด้วย เท่านั้นเองครับ”
คิมเบอร์ลี่ : “จะเริ่มหาที่สร้างเรือนหอจริงจังเมื่อไหร่ ยังไม่รู้ค่ะ ไม่รู้เลย ตอนนี้โฟกัสกับงานมากกว่าค่ะ”

หมาก : “จริงๆ เราไปดูที่ด้วยกันมา และตัดสินใจซื้อเลยเพราะชอบตรงนั้น แต่เราคงไม่ทำเป็นเรือนหอหรอก จะทำเป็นที่ที่เราไปพักผ่อนที่ต่างจังหวัดเท่านั้นเอง ส่วนที่กรุงเทพฯ ตอนนี้ยังดูไปเรื่อยๆ ยังไม่ได้ตัดสินใจเท่าไหร่ จะเอาบ้านหรือคอนโดยังไงดี (หันไปถามคิมเบอร์ลี่)”
คิมเบอร์ลี่ : “บ้านๆ ชอบบ้านค่ะ”

เผยปีใหม่นี้ได้เคาท์ดาวน์ด้วยกันแน่นอน
หมาก : “ผมว่าปลายปีตอนนี้เมืองไทยอากาศดีมากเลย ปีใหม่ก็อาจจะอยู่ไทยก็ได้ เดี๋ยวรอดูก่อน แต่เคาท์ดาวน์ด้วยกันแน่นอนครับ อยู่ด้วยกัน”
คิมเบอร์ลี่ : “ช่วงนั้นไม่ได้รับงานด้วยค่ะ พักผ่อน”
หมาก : “เขาถ่ายละครเหนื่อยมากนะคนนี้ ต้องพักผ่อนๆ ครับ”
คิมเบอร์ลี่ : “ถามว่าจะมีของขวัญพิเศษให้กันไหม ปีใหม่ไม่มีค่ะ เพราะอีกแป๊บเดียวก็วันเกิดแล้ว”
หมาก : “ก็ไม่ได้ขออะไรพิเศษนะครับ เพราะเพิ่งขอไปเมื่อวานเนี่ย ไม่มีเหตุผลอะไรเลย (หัวเราะ)”
คิมเบอร์ลี่ : “โห ไม่ค่อยจะได้ขออะไรเลย นี่ก็กรุบๆ ของเล็กๆ ถูกๆ ไม่แพง เราดูแล้ว ถ้ามากกว่านั้นเขาไม่ซื้อให้แน่นอน (หัวเราะ)”

ESPN’s Final Prediction For Ohio State-Penn State Game

23 Nov 2019, 13:49 by Andrew Holleran

ESPN’s Football Power Index has finalized its prediction for this afternoon’s Ohio State-Penn State game. The Buckeyes and the Nittany Lions are getting ready for kickoff in Columbus.

This is essentially a must-win game for both teams. Ohio State locks up a spot in the Big Ten Championship Game with a win, while Penn State would just need to beat Rutgers next week.

It’s possible the Buckeyes could make the College Football Playoff at 11-1, but it’s not likely. So this is a very massive contest taking place in Columbus today.

ESPN’s computer likes Ohio State – big.

ESPN’s Football Power Index is giving No. 2 Ohio State an 83.9 percent chance of winning today’s game. It gives Penn State just a 16.1 percent chance of pulling off the upset.

Ohio State and Penn State are set to kick off at noon E.T. inside Ohio Stadium.

The game will be on FOX.

Коврик для мыши — рабочая поверхность для манипулятора типа «мышь» и в данном обзоре мы рассмотрим лучшие коврики с Aliexpress.

Игровой коврик для мыши Iahead

реклама

Игровой коврик выполнен из качественной резины, сверху покрытой микро-текстурированной тканью. Данный коврик отлично походит для игр, а также для обычной домашней работы за компьютером. Резиновая основа коврика не скользит по столу, а большой размер обеспечит идеальную рабочую поверхность для мыши. Толщина коврика с подсветкой равна 4 мм. Верхний слой из текстурированного волокна, позволяет легко и просто управлять мышкой.

Главной особенностью этого коврика является наличие подсветки по контуру коврика. 13 режимов подсветки (Белый, красный, желтый, синий, фиолетовый, голубой, зеленый, переменная вспышка, переменная красная меняющаяся вспышка и др.) можно менять нажатием одной кнопки.

Подсветка подключается через USB кабель длиной 1,8 м.

Максимальный размер коврика составляет 900*400 мм. Другие варианты размеров:

реклама

- 800*300 мм.;

- 700*300 мм.;

- 350*250 мм.

Производителем предусмотрена возможность заказа коврика без подсветки. Толщина коврика без подсветки равна 2 мм.

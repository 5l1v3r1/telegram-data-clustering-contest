​Google Pixel 3 is starting to show the same battery problems as the Nexus 6P

14 Nov 2019, 08:30 by Jared Peters

If there’s one thing Google is good at, it’s repeating mistakes. Apparently the Pixel 3 is starting to show some of the same battery issues that plagued the Nexus 6P, which, in case you forgot, is a phone that ended its life as a class action lawsuit because it was pretty broken for tons of users. Not exactly a model citizen in smartphone history.

Same old battery problems

The issue happens when the Pixel 3 and Pixel 3 XL get below 30% of battery life. Sometimes users have huge drops in battery after that point, while others report a complete shutdown, almost as if the last third of the battery is unusable. This is what happened to cause the Nexus 6P’s early shutdown problems and forced Google to shell out a bunch of cash and replacements for dysfunctional phones.

These problems started appearing on Google’s own support forums back in April, and have only grown since then. Google’s solution for these affected users is to factory reset their device and try to boot into safe mode, except those don’t actually solve any of the problems. Even the RMA replacements that are being sent out exhibit some of the same issues, which means this might be a systemic problem with the Pixel 3’s battery.

This is going to be especially problematic as Pixel 3 batteries get older. Newer batteries naturally have fewer problems, but as they age these kinds of issues pop up more and more often. So far it doesn’t seem as common as the Nexus 6P’s issues, but this still isn’t very promising.

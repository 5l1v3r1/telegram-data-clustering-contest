Аналитики назвали новый инструмент мошенников, ворующих деньги с карт

14 Nov 2019, 09:30

Мошенники, которые похищают деньги с карт, представляясь сотрудниками банка, все чаще используют сервис переводов card2card для вывода чужих средств. Во время разговора аферисты просят назвать код из смс-сообщения.

С помощью полученного кода деньги с карты жертвы отправляются на виртуальные карты, привязанные к онлайн-кошелькам. Это усложняет возврат средств для потерпевшего и обеспечивает аферистам достаточную степень анонимности.

По данным Сбербанка, этот способ перевода используют 80% мошенников, которым удается войти в доверие клиентов банка с помощью методов социальной инженерии. Для перевода средств на виртуальный "пластик", который есть у "Яндекс.Денег", QIWI, Webmoney и других сервисов, нужно ввести данные карты человека, а затем подтвердить транзакцию кодом из смс. Уже с виртуальной карты мошенники могут переводить деньги на физический "пластик", с которого можно снять наличные.

Еще в 2018 году этот способ хищения денег был сравнительно редким, но в 2019 году мошенники стали регулярно прибегать к такому методу: в месяц банки фиксируют по меньшей мере несколько подобных случаев.

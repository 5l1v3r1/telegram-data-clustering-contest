GitHub перезапускает мобильное приложение

14 Nov 2019, 11:26 by Александра Кукуть

На конференции GitHub Universe в Сан-Франциско сервис анонсировал ряд обновлений, а также новое мобильное приложение, пишет VentureBeat.

Сейчас доступна бета-версия приложения для iOS, версия для Android появится в начале 2020 года. Дата полноценного релиза пока не известна.

Приложение не будет мобильной копией сайта, но пользователи смогут получать уведомления, просматривать и комментировать код, принимать пулл-реквесты. Со временем функционал приложения будет расширяться.

В 2012 году GitHub уже выпускал приложение для Android, но в 2015-м его удалили из Play Store, отмечает Android Police.

Помимо этого, GitHub объявил о всеобщей доступности функции GitHub Actions, позволяющей автоматизировать процесс разработки, и сервиса управления пакетами GitHub Packages. Также GitHub улучшил поиск кода (пока в бета-версии для ограниченного круга пользователей, для всех станет доступен в ближайшие месяцы) и оптимизировал работу с уведомлениями — изменения увидят сначала пользователи мобильного приложения, а через несколько месяцев — самого GitHub.com.

Расширена программа GitHub Sponsors, в рамках которой можно финансово поддержать создателей открытого софта: теперь она распространяется не только на отдельных разработчиков, но и на целые команды.

​Работа для разработчиков мобильных приложений с зарплатой до 3000$. ​

1. Заполните анонимную форму — 5 минут.
2. Укажите зарплатные (и другие) ожидания.
3. Выберите желаемую индустрию или область деятельности.
4. Получайте релевантные предложения​​.​​​​​​​​​​​​​​

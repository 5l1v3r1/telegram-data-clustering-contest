Realme 3 and 3i updated with Dark Mode and November security patch

13 Nov 2019, 11:57 by Marco Lancaster

Rate this post

Realme is in a roll this month when it comes to providing updates for their smartphones. After releasing a software update for the Realme 5 and the Realme 2 Pro yesterday, the company is now rolling out another OTA for the Realme 3 and 3i. Just like the previous updates, this update consists in bringing the latest additions to the ColorOS firmware. Therefore, there’s a new System-Wide Dark Mode, Redesigned Notifications Center and Clone App support for more third-party apps.

Interestingly enough, the update also bumps the security patch to the latest one available – November 2019. In addition, it includes a new charging animation. Furthermore, users will have a better way to monitor data usage with a customizable data usage reminder and the ability to check how much data connected devices are consuming. The usual rules for updates are valid for this one. It’s a gradual OTA release and Realme is rolling it out for a limited number of Realme 3 users. However, a broader rollout will happen if now critical bugs pop-out during the tests. Interestingly enough, these updates are rolling out before the revelation of ColorOS 7.0.

According to rumors, with ColorOS 7.0, the company will provide a custom branch specific for Realme devices. This custom branch will be close to stock Android.

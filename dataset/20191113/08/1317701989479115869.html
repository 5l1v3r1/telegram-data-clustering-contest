Media Feature Pack Windows 10 N 1909 (November 2019 Update)

13 Nov 2019, 08:27 by moinmoin

Wer eine Windows 10 N Version installiert hat, benötigt das Media Feature Pack damit der Windows Media Player, Groove Music, Skype und anderes auch korrekt funktioniert. Auch die Codecs müssen nachinstalliert werden.

Anders als bisher wird es das Media Feature Pack nicht mehr direkt zum Download angeboten. Es wird nur noch über die optionalen Funktionen in Windows 10 angeboten. Einstellungen -> Apps -> Apps und Feature. Hier die Optionalen Features anklicken und Feature hinzufügen auswählen.

Nach der Installation kann man nun noch die Mediencodecs aus dem Microsoft Store nachinstallieren.

  • VP9-Videoerweiterungen
  • Webmedienerweiterungen
  • HEIF-Bilderweiterungen
  • Nicht zu vergessen die HEVC-Videoerweiterung. Aber denkt daran. Davon gibt es eine kostenpflichtige und eine kostenlose Version. Siehe hier

Sollte es widererwarten doch noch einen Offline-Installer für das Media Feature Pack Windows 10 1909 geben, werden wir diesen dann hier nachträglich einfügen.

Microsoft Auto-Updating Windows 10 1803 Devices to May 2019 Update

13 Nov 2019, 21:08 by Sergiu Gatlan

Windows 10, version 1803, also known as the April 2018 Update, has now reached end of service for Home, Pro, Pro Education, and Pro for Workstations editions, and will no longer receive any future quality and security updates.

As Microsoft said in October when it warned users to update their Windows 10 1803 devices before end of support, customers who want to continue receiving quality updates should update to the latest version of Windows.

Redmond also states that an automatic feature update to May 2019 Update will be initiated on Windows 10 1803 devices that reached end of service to allow them to continue receiving updates.

For Windows 10 devices that are at, or within several months of reaching end of service, Windows Update will automatically initiate a feature update (with users having the ability to choose a convenient time); keeping those devices supported and receiving the monthly updates that are critical to device security and ecosystem health. - Microsoft

Some security updates might still trickle in

However, very rarely, Microsoft might still release security updates for critical vulnerabilities that could potentially have a major impact on no longer serviced Windows versions such as Windows 10 1803.

This already happened this year, for instance, when the company issued a security update for the no longer supported Windows XP and Windows Server 2003 to patch the wormable Remote Code Execution (RCE) vulnerability dubbed BlueKeep and tracked as CVE-2019-0708, a flaw discovered in the Remote Desktop Services (RDS) platform.

It is strongly recommended that users install the Windows 10 1903 update since "if a device is out of service, it is no longer receiving quality updates from Microsoft and may become more vulnerable to security risks and viruses," as Redmond said in September.

"A single exploited device can reveal other devices on a network, revealing otherwise secure information or spreading malware to other computers in the network."

The Enterprise and Education editions of Windows 10, version 1803 are the next editions to go out of service on November 10, 2020.

Windows 10 safeguard holds for some devices

However, as Microsoft's Windows lifecycle fact sheet notes that "Not all features in an update will work on all devices. A device may not be able to receive updates if the device hardware is incompatible, lacks current drivers, lacks sufficient storage space, or is otherwise outside the original equipment manufacturer’s (OEM) support period."

In these cases, the feature update will not be initiated on Windows 10, version 1803 devices that are not compatible with the latest version of Windows 10.

If Windows Update will not offer to upgrade their systems, customers will have to review the Windows 10 System Requirements and the Windows Processor Requirements to check if their devices match these requirements.

Users should also know that there still are four compatibility holds active preventing some devices to upgrade to Windows 10, version 1903:

• Unable to discover or connect to Bluetooth devices using some Realtek adapters (Mitigated)
• Safeguard on certain devices with some Intel and Broadcom Wi-Fi adapters (Resolved)
• Intermittent loss of Wi-Fi connectivity (Mitigated)
• AMD RAID driver incompatibility (on for older AMD RAID drivers)

Microsoft also provides Windows 10 customers with detailed information on how to troubleshoot update problems and fix Windows Update errors.

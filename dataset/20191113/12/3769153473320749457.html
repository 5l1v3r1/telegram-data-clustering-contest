"Highlight" - Addon for Syntax Highlighting Text Objects

13 Nov 2019, 12:45

Sreeraj R writes:

Recently while working on a personal project I had to find a way to animate a computer screen showing someone coding. So I thought why not make it to an addon, may be it will be useful to someone else.

Highlight is a Blender Addon to create syntax highlighting on text objects containing any type of code.

Currently it supports 430+ languages/language subsets.
Supports Blender 2.81.

Белгород попал в середину списка городов для лучшей жизни с детьми

Областной центр оказался на 111 позиции из 200 возможных

13 Nov 2019, 10:08 by Мария ШЕПЕЛЕВА

Белгород попал в середину рейтинга городов для лучшей жизни с детьми.

Такие данные эксперты получили с помощью опроса, в котором приняли участие 90 тысяч человек. Чтобы оценить уровень удовлетворенности инфраструктурой для детей в каждом из 200 городов, исследователи попросили их жителей оценить высказывание «В моем районе есть все необходимое для детей: игровая площадка, детский сад и поликлиника, школа и другое» по десятибалльной шкале. Затем для каждого населенного пункта был рассчитан средний балл. Белгород занял 111-ю строчку в списке с результатом 6,5 балла, чем ухудшил показатели по сравнению с прошлым годом. На 114 месте оказался Старый Оскол, результат которого составил 6,4 балла.

Отметим, что жители городов Белгородчины довольны инфраструктурой для детей меньше, чем в среднем по России. Лидером рейтинга стала Москва с оценкой 8 баллов, а аутсайдером – Каменск-Шахтинский, набравший лишь 4,3 балла.

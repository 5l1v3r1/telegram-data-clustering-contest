Rayakan Hari Jadi ke-25, Gigi Rilis Box Set

03 Nov 2019, 05:07 by Windy Eka Pramudya

UNTUK merayakan hari jadi ke-25, Gigi memberikan kado istimewa untuk para Gigikita, sebutan penggemar Gigi, Pada Sabtu, 2 November 2019. Gigi meluncurkan box set ekslusif sebagai bentuk perayaan ulang tahun mereka. Box set yang terdiri atas empat keping puzzle ini mewakili kisah empat personel Gigi, yaitu Armand Maulana (vokal), Dewa Budjana (gitar), Gusti Hendy (drum), dan Thomas Ramdhan (bas).

Box set ekslusif ini berisi memorabilia para personel Gigi. Setiap box set berisi kaus, DVD dokumenter yang berkisah tentang keseharian personel dalam judul A Day With, buku harian yang ditulis tangan para personel, serta kartu anggota Gigikita edisi setiap personel. Selain benda memorabilia Armand dkk, box set juga diisi mini album terbaru Gigi berjudul Silver dalam format kaset, kartu anggota 25 tahun Gigi, 10 kartu pos bergambar foto perjalanan 25 tahun Gigi, dan sertifikat kepemilikan box set.

Uniknya, mini album Silver memuat lima lagu baru Gigi, yang dua di antaranya tidak dikasih lirik. Di dua nomor ini, Armand hanya bergumam (humming) dengan iringan musik dari Budjana, Thomas, dan Hendy. Nantinya, siapapun bisa menulis lirik untuk dua lagu tersebut. Jika liriknya cocok, akan dijadikan lagu Gigi.

"Konsep humming ini sengaja kami buat untuk memberi tantangan kepada pendengar. Mereka dibebaskan untuk membuat lirik berdasarkan melodi yang sudah ada. Cara ini untuk membangun interaksi Gigi dengan pendengar," ungkap Armand di The Fifth, Jalan Kemang Raya, Jakarta.

Box set ekslusif 25 tahun Gigi ini merupakan ide bersama Gigi yang diwakili tim manajemen Gigi dan PT Beli Album Fisik milik Erix dari Endank Soekamti. Armand menyebutkan, hampir semua band yang bertahan selama puluhan tahun pasti pernah merilis sesuatu yang kreatif dan sifatnya personal. Untuk itulah, di tahun ke-25 ini mereka memutuskan untuk merilis box set ekslusif.

"Menurut Erix, penggemar itu terbagi tiga. Pertama yang die hard, kedua di tengah-tengah, dan ketiga, penggemar yang selewat. Box set ini menyasar ke tipe penggemar yang die hard. Walaupun harganya enggak murah, tapi pasti ada yang beli. Tujuannya memang bukan produk massal, tapi kalau banyak yang pesan, alhamdulillah," ungkap Armand.

Alasan lain Gigi merilis box set ekslusif adalah untuk merilis karya dalam bentuk fisik. Inilah sebabnya mereka membuat mini album Silver dalam format kaset. Bagi Gigi, merilis karya dalam bentuk fisik menjadi tantangan di tengah gempuran format musik digital.

"Kami ingin mengingatkan ke generasi sekarang, kalau di industri musik ada era fisik. Kebetulan sekarang kaset juga sedang booming lagi. Jadi, kami memilih untuk kembali ke kaset," kata Budjana.
Hendy menambahkan, sebagai band, Gigi telah banyak melahirkan karya. Pada hari jadi ke-15 mereka pernah merilis buku. Di tahun ini, mereka memutuskan merilis box set dan fokus ke kisah para personel. Hal ini dimaksudkan untuk mendekatkan awak Gigi lebih personal ke pendengar.

Sementara itu, bagi Thomas, mini album Silver dan box set menjadi pengingat untuk Gigi. Mereka harus terus bersyukur dan terus berkarya karena Tuhan sudah memberi kesempatan untuk Gigi terus berjalan.
"Kami adalah band yang lahir di era 1990-an. Kami merasakan era kaset sampai sekarang era digital. Di tahun ke-25 ini kami ingin kembali merasakan karya analog, makanya kami bikin kaset," ujar Thomas.

Konsep box set dan minialbum Silver dikerjakan sekitar satu bulan. Dari mini album itu, Gigi menawarkan lagu Pesan yang Singkat sebagai single. Lirik lagu ini ditulis Thomas, sedangkan aransemennya dikerjakan Gigi.Â 

"Semoga Gigi bisa terus ada, sampai kami tidak bisa berkarya lagi," ujar Armand yang diamini personel lain.***

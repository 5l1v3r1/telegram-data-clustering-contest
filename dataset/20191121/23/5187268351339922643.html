Zorin OS 15 Lite: Largue de uma vez por todas o Windows 7

21 Nov 2019, 23:02 by Pedro Pinto

Zorin OS é talvez a distribuição mais parecida ao Windows, muito por causa do seu visual e menu iniciar muito idêntico ao Windows 7. Esta distribuição é baseada no popular Ubuntu e traz como ambiente gráfico de trabalho o Zorin Desktop.

Recentemente foi lançada uma versão lite do Zorin OS 15. Conheçam as novidades.

Apesar de o Windows 10 ser o sistema operativo para Desktop mais usado do mundo, o Windows 7 continua a estar instalado em milhões de computadores. A Microsoft já alertou várias vezes para o facto de ser importante migrar para o Windows 10, mas a mensagem tem sido um pouco ignorada.

Migrar para o Windows 10? Experimentem o Zorin OS 15 Lite

É verdade que existem muitas distribuições Linux para se escolher. Alguns utilizadores acham que são demasiadas e que o know how de tantos programadores poderia ser direcionado distribuições mais completas e robustas.

No entanto, havendo tanta oferta existem também mais opções de escolha. Para quer quer algo parecido com o Windows, então experimentem o Zorin OS.

Como principal novidade, o Zorin OS 15 Lite vem com o Desktop Xfce 4.14 que traz várias melhorias à interface. Há novos temas que estão disponíveis em várias cores. Os temas podem ser escolhidos via app Zorin Appearance.

Esta versão é baseada no Ubuntu 18.04.3 LTS. Vem totalmente personalizada para tirar o melhor partido de máquinas com menos recursos. Se ainda usam o Windows 7, pensem em dar uma oportunidade ao novo Zorin OS 15 Lite.

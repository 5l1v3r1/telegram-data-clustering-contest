СМИ: Amazon готовит свой ответ Google Stadia

21 Nov 2019, 11:00 by Геннадий Воробьев

Крупные платформодержатели начали погоню за облачным геймингом: у Sony — PlayStation Now, у Google — Stadia, у Microsoft — Project xCloud. Как утверждает сайт CNET, скоро в гонку включится ещё одна корпорация — Amazon.

По сведениям портала, Amazon набирает команду соответствующих специалистов в подразделение Amazon Web Services — ветвь, которая занимается облачными технологиями компании. Amazon уже якобы переманила нескольких ценных сотрудников из Microsoft и других крупных игровых студий, чтобы подготовить запуск своей платформы для облачного гейминга.

Если верить CNET, Amazon собирается интегрировать облачный гейминг в Twitch. Быть может, грядущая площадка войдёт в список услуг подписки Amazon Prime, где уже есть ускоренная доставка посылок из интернет-магазина Amazon, доступ к библиотекам музыки, фильмов, сериалов и книг, а также игровые бонусы с Twitch.

Инсайдеры считают, что Amazon анонсирует и запустит свою площадку для облачного гейминга в 2020-м — так, чтобы составить конкуренцию следующему поколению консолей.

Apple 正在找工程師開發多媒體軟體，Windows 版 iTunes 也跟著要被汰換了？

21 Nov 2019, 02:45 by 愛范兒

雖然 Mac 系列每次更新都能成為焦點，但不可否認的是 Windows 電腦在個人市場的市占率依然有絕對優勢，所以蘋果也為 iTunes、iCloud 等軟體開發了 Windows 版本，而且今後可能還會有更多的蘋果應用出現在微軟的平台。

知名資訊網站 Neowin 發現蘋果發布在 LinkedIn 上的招募資訊中，出現了一個名為「高級軟體工程師──Windows 多媒體應用程式」的職位，補充說明中還寫道「為 Windows 開發下一代多媒體應用程式」。

該職位所屬部門為「Media Apps」（多媒體應用團隊），「如果你熱愛音樂並熱衷於寫代碼」的描述暗示了蘋果正在開發的可能是一款類似於 Apple Music 的 Windows 軟體，或許還包括 Apple TV。

在今年發表的 macOS Catalina 中，iTunes 已經被 Music、Podcasts 和 TV 3 個軟體取代，但在 Windows 上，iTunes 仍然是使用 Apple Music、iTunes Store 以及備份 iPhone 的主要方式。

眾所皆知，蘋果的軟體一向很少對其他平台開放，而且比起微軟在 iOS 和 Mac 平台上從 mail、行事曆到瀏覽器全方位發力，蘋果對 Windows 的支援可以說相當湊合，基本上屬於「又不是不能用」的態度，比如勸退很多 iPhone 和 iPod 使用者的桌面版 iTunes，操作之卡頓讓人難以相信是以使用者體驗著稱的蘋果的產品，直到去年蘋果為了相容 Windows 10 上線了的 UPW 版，體驗才有所改善。

然而，雖說自有平台內的生態是蘋果的競爭優勢所在，但隨著這幾年其增值服務的擴張，蘋果也開始重視起其他平台的用戶，比如 2015 年上線的 Android 版 Apple Music 體驗就相當不錯，遵循 Material Design 規範的設計也讓人看到了蘋果的誠意，Apple TV 現在也能在 Samsung TV、Roku 和亞馬遜 Fire TV 等第三方裝置上使用，今年 6 月蘋果還和微軟合作打造了新的 iCloud for Windows。既然功能臃腫的 iTunes 被淘汰已成定局，那麼將 Mac 上的娛樂三件套搬到 Windows 上也在情理之中。

此外，蘋果在這次招募要求中提到擁有 UWP（通用 Windows 平台）開發經驗是加分項，可以推測接下來在 Windows 上發表的大概是 UWP 應用軟體。9To5Mac  就指出，UWP 應用軟體同時適用於 Windows 電腦和 Xbox One，很多使用者把後者當作多媒體中心，因此在 UWP 平台推出多媒體應用對於 Apple TV+ 的擴張具有重要意義。

鑑於蘋果還在找工程師，要用上 Windows 版的應用軟體估計還要等好一段時間。

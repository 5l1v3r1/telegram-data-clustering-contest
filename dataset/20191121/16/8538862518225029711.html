Ghost Games has today released a fresh patch for Need for Speed Heat on PlayStation 4. Taking the game to version 1.03, this update makes some pretty decent improvements to the overall experience. You can download the patch right now -- it's just over 2GB in size.

But what does it do? One thing it aims to address is long loading times during night races. The visual effects on night race intros have been reduced as a short term fix while Ghost Games figures out a better long term solution. Elsewhere, various scenarios which might crash the game have been addressed with this update, which is definitely appreciated.

There are a few other bug fixes too, which you can read about on the Need for Speed subreddit. Are you still enjoying Need for Speed Heat on PS4? Have you run into any issues? Hit the gas in the comments below.

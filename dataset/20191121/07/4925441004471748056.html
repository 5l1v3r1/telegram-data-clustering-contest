В Search Console стали доступны данные об эффективности страниц с разметкой для товаров

21 Nov 2019, 07:37 by Анна Бондарь

Google добавил новый фильтр «Product results» в раздел «Вид в поиске» отчёта об эффективности. С его помощью можно просматривать данные по страницам с разметкой для товаров, которые отображаются в качестве расширенных результатов.

В русскоязычной версии Справки Search Console этот фильтр называется «Результаты поиска товаров».

Если страницы сайта отображаются как расширенные результаты, то теперь владелец ресурса сможет анализировать их эффективность, используя новый фильтр.

В Search Console будут доступны данные по показам, кликам, средней позиции и CTR. При желании их также можно будет сегментировать по устройствам, географии и запросам.

С помощью нового фильтра вебмастера смогут узнать, сколько трафика они получают благодаря таким данным, как цена товара и его наличие в результатах поиска. Они также смогут анализировать, как менялся трафик со временем, и по каким товарным запросам показывается сайт.

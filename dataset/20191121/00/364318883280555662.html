CAT มุ่งขยายตลาดด้วยบริการ เปิดแอปพลิเคชันใหม่ตอบสนองไลฟ์สไตล์ลูกค้า

21 Nov 2019 by Pornnapat Kaewbandit

เปิดตัว 2 แอปพลิเคชัน บนสมาร์ทดีไวซ์ C internet Mobile App ตอบโจทย์การให้บริการที่รวดเร็ว สะดวกสบายสำหรับผู้ใช้งาน และ C nema Mobile App ตอบสนองความต้องการคอนเทนท์ด้านความบันเทิง ภาพยนตร์ ซีรีส์ สารคดี ฯลฯ บนสมาร์ทดีไวซ์แบบ HD เพื่อประสบการณ์ที่ดีที่สุด และความคุ้มค่าในการเลือกใช้บริการ

ดร.ดนันท์ สุภัทรพันธุ์ รองกรรมการผู้จัดการใหญ่ สายงานการตลาดและบริการ บริษัท กสท โทรคมนาคม จำกัด (มหาชน) หรือ CAT เปิดเผยว่าเราให้ความสำคัญอย่างมากต่อการมอบประสบการณ์ที่ดีที่สุดให้กับลูกค้า ซึ่งเป็นส่วนหนึ่งของการสร้าง Customer Branding ตาม Brand DNA ของ CAT ด้วยการเข้าถึงลูกค้า เข้าใจความต้องการ และปรับตัวให้สอดรับกับพฤติกรรมที่เปลี่ยนไปของลูกค้าให้ได้มากที่สุด

“ปัจจุบันการใช้งานอินเทอร์เน็ตเป็นเหมือนปัจจัยที่ 5 ของทุกกลุ่มไปแล้ว โดยเฉพาะการใช้อินเทอร์เน็ตความเร็วสูงบนสมาร์ทดีไวซ์ต่างๆ ไม่ว่าจะเป็นโทรศัพท์สมาร์ทโฟน แท็บเล็ต หรือแม้แต่สมาร์ททีวี ผ่านการเชื่อมต่อด้วย Wi-Fi ดังนั้นความแรงและความเสถียรของสัญญาณ Wi-Fi จึงเป็นเรื่องสำคัญที่สำหรับลูกค้ามาก อย่างไรก็ตาม หลายต่อหลายครั้งที่ปัญหาการใช้งานอินเทอร์เน็ตของลูกค้าไม่ได้เกิดจากปัญหาของเครือข่าย แต่เกิดจากปัญหาการเชื่อมต่ออุปกรณ์ภายในบ้านเข้ากับเครือข่ายเช่น Wi-Fi หรือปัญหาอื่นๆ ที่อาจเกิดขึ้นได้ ซึ่ง CAT ต้องการจะให้บริการ และสนับสนุนการใช้งานของลูกค้าให้เกิดความพึงพอใจสูงสุด และช่วยแก้ปัญหาของลูกค้าได้อย่างรวดเร็วที่สุด CAT จึงได้พัฒนาแอปพลิเคชัน ‘C internet Mobile App’ ขึ้นเพื่อให้ลูกค้าได้รับประสบการณ์การใช้งานที่ดีที่สุดและราบรื่นที่สุด”

ดร. ดนันท์ เปิดเผยว่า จุดเด่นของบริการ C internet Mobile App คือการเป็น On Demand One-Stop-Service Application สำหรับลูกค้าผู้ใช้งานอินเทอร์เน็ตของ CAT เพื่อใช้เป็นช่องทางในการสื่อสารแบบสองทางกับผู้ใช้บริการ และยกระดับการให้บริการเพื่อความรวดเร็วและสะดวกสบายสูงสุดสำหรับลูกค้า โดยลูกค้าสามารถเลือกทำรายการ และรับบริการได้อย่างรวดเร็ว ไม่ว่าจะเป็นการตรวจสอบรายละเอียด แพ็กเกจให้บริการของลูกค้า การชำระค่าบริการผ่านระบบ Payment Gateway ที่มีความปลอดภัยสูงมาตรฐานสากล โดยสามารถเลือกชำระค่าบริการผ่านบัตรเครดิต หรือเดบิตได้และมีการตัดยอดบิลแบบ Real Time ซึ่งระบบจะหักล้างยอดบิลที่ค้างได้ทันทีทำให้ลูกค้าที่อยู่ในสถานะตัดสัญญาณชั่วคราวสามารถกลับมาใช้บริการได้ตามปรกติอย่างรวดเร็ว พร้อมบริการทดสอบความเร็วการใช้งานอินเทอร์เน็ต (Speed Test)

“C internet Mobile App ยังเพิ่มความสะดวกสบายด้วยการแจ้งซ่อมโดยส่งภาพถ่าย หรือคลิปวีดีโอเพื่อขอคำปรึกษาจากฝ่ายสนับสนุนทางเทคนิคได้โดยตรง ซึ่งจะช่วยตอบโจทย์การแก้ปัญหาของลูกค้าได้ตรงจุด และรวดเร็วกว่าการติดต่อฝ่ายสนับสนุนด้วยการอธิบายผ่านทางโทรศัพท์หรือรอช่างเข้าไปตรวจสอบปัญหาที่บ้าน โดย CAT มีแผนที่จะพัฒนาบริการด้านการสนับสนุนทางเทคนิคนี้จากการส่งภาพหรือคลิปวีดีโอ มาเป็นการ Live VDO Chat เพื่อแจ้งปัญหาและให้บริการได้รวดเร็วและตรงจุดยิ่งขึ้น นอกจากนี้ CAT ยังมีโปรโมชั่นพิเศษสำหรับผู้ใช้ คือทันทีที่ทำการติดตั้งแอปพลิเคชัน จะได้รับ 50 points ที่สามารถสะสมเพื่อใช้ Redeem หรือแลกของรางวัลต่างๆ ผ่านแอปพลิเคชัน หรือใช้ลุ้นรับของรางวัลมากมาย อาทิ มอเตอร์ไซค์ โทรทัศน์ และของรางวัลอื่นๆ ที่จะหมุนเวียน และเพิ่มเติมอย่างต่อเนื่อง”

ดร.ดนันท์ กล่าวเพิ่มเติมว่าผู้บริโภคมีแนวโน้มต้องการใช้งานอินเทอร์เน็ตที่มีความเร็วมากขึ้นเรื่อยๆ ซึ่งยิ่งผู้บริโภคเลือกใช้บริการอินเทอร์เน็ตความเร็วสูงมากขึ้นเท่าไหร่ ต้องยิ่งให้ความสำคัญกับการให้บริการที่ดีมากขึ้นเท่านั้น โดยปัจจุบัน ผู้บริโภคกว่า 50% ใช้บริการอินเทอร์เน็ตในการดูคอนเทนท์ต่างๆ เป็นหลัก

“พฤติกรรมของผู้บริโภคปัจจุบันคือการเลือกคอนเทนท์ในลักษณะของ On-Demand คือเลือกชมเรื่องที่ต้องการ ในเวลาที่ต้องการ บนดีไวซ์ที่ต้องการ ดังนั้นจะเห็นได้ว่าการรับชมรายการย้อนหลัง หรือการใช้บริการ Streaming เป็นที่นิยมมากขึ้นเรื่อยๆ CAT จึงต้องการมอบมูลค่าเพิ่มให้กับลูกค้าด้วยการเปิดบริการแอปพลิเคชัน C nema Mobile App ที่ตอบโจทย์ความต้องการด้านการบริโภคคอนเทนท์ให้กับลูกค้า โดยจุดเด่นของ C nema Mobile App อยู่ที่การเลือกรับชมภาพยนตร์ ซีรีย์ สารคดี ดิจิทัลทีวีแบบความคมชัดสูงได้ตามความต้องการ บนทุกดีไวซ์ที่เชื่อมต่อได้ โดยเรามีแผนที่จะเพิ่มเนื้อหาอื่นๆ นอกเหนือจากที่มีอยู่ในปัจจุบัน อาทิ เนื้อหาสำหรับเด็ก และสารคดีต่างๆ”

นอกจากนี้ ดร.ดนันท์ ยังได้ย้ำว่า CAT ให้ความสำคัญกับกลุ่มลูกค้าทุก Segment ไม่ว่าจะเป็นกลุ่มลูกค้าทั่วไป กลุ่มลูกค้าองค์กร กลุ่มลูกค้าธุรกิจ กลุ่มลูกค้า E-Commerce ไปจนถึงกลุ่มลูกค้าที่มีความต้องการเฉพาะ อาทิ กลุ่ม Gamer และ Caster โดยเรามุ่งมั่นที่จะพัฒนาบริการ และมอบมูลค่าเพิ่มให้กับลูกค้าทุกกลุ่ม รวมถึงนำเสนอ Package ที่เหมาะสมกับทุกกลุ่มความต้องการ ภายใต้แนวคิด Customer Centric ซึ่งเป็นหนึ่งใน C-A-T-A-L-Y-S-T ซึ่งเป็น Brand DNA ของ CAT อีกด้วย

“เรากำลังเตรียมทีม CAT Internet Expert เพื่อเข้าไปแนะนำบริการ การติดตั้ง ตรวจเช็คพร้อมวัดค่าสัญญาณในแต่ละจุด และแนะนำแพ็กเกจบริการให้กับลูกค้า เพื่อความพึงพอใจสูงสุดอีกด้วย” ดร.ดนันท์

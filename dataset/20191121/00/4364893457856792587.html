Api Berkobar, Dua Lansia dan Balita 10 Bulan Selamat

21 Nov 2019 by Manik Priyo Prabowo

Surabaya (beritajatim.com) – Kebakaran hebat kembali melanda Kota Surabaya, Kamis (21/11/2019). Kebakaran yang melahap dua rumah ini membuat warga kawasan ex lokalilasi Dolly gempar.

Mereka secara spontan berhamburan untuk menyelamatkan dan memadamkan api. Diketahui si jago merah melahap rumah milik Tajib (80) dan milik Fatur Rohman (32) di Jalan Putat Jaya Gang IV -C, Surabaya. Menurut informasi, rumah milik Tajib ini dihuni dua lansia, sedangkan rumah milik Fatur Rohman dihuni empat orang salah satunya balita 10 bulan yang sedang tertidur pulas.

“Kedua rumah terbakar habis. Menurut keterangan saksi awalnya api melahap rumah Tajib dan api merambat ke rumah Fatur,” jelas Kepala Dinas Pemadam Kebakaran Surabaya, Didik Irianto.

Awalnya, lanjut Didik, saat itu saksi  mencium bau asap seperti tanda-tanda kebakaran. Setelah dilakukan pengecekan keluar rumah, ternyata benar, warga melihat api melahap rumah Tajib. Usai melihat api membara, warga pun berusaha berteriak dan membantu penghuni dua rumah tersebut untuk menyelamatkan diri.

Usai kedua penghuni rumah keluar, warga yang sudah menghubungi 112 menunggu mobil pemadam dan berusaha memadamkan api. “Usai dapat laporan enam unit pemadam kita kerahkan untuk memadamkan api. Beruntung di dalam gang  tak ada mobil parkir. Jalannya juga lebar,” lanjutnya.

Tiba di lokasi, petugas melakukan pengepungan api guna menahan api merambat ke rumah lainnya. Bahkan, guna mempercepat pemadaman, petugas melakukan pembobolan tembok. “Kebakaran terjadi sekitar pukul 19.40 WIB, mobil pemadam tiba sekitar pukul 19.46 WIB. Api padam pukul 20.20 WIB, lalu pembasahan berakhir pukul 20.48 WIB. Kita terjunkan 10 unit mobil pemadam,” papar Didik.

Kini petugas dari tiga pilar yang mendatangi lokasi langsung memberikan pertolongan baik kebutuhan hunian dan alat tidur sementara. “Korban kebakaran disiapkan bantuan berupa kasur, bantal, selimut, pakaian, perlengkapan mandi dan makan 3x sehari selama 7 hari,” lanjut Bambang Vistadi, Kepala Bidang Operasional Dinas Kebakaran Kota Surabaya.

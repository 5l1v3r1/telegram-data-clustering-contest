As melhores placas gráficas para comprar nesta Black Friday

21 Nov 2019, 14:00 by Maria Inês Coelho

O investimento numa boa placa gráfica é considerável. Por isso, muitos optam por deixar esta compra para as épocas de promoção. Com a Black Friday à porta, nada melhor que começar a fazer a lista dos produtos essenciais onde podem entrar finalmente as placas gráficas.

Assim, selecionámos algumas das melhores para que possa fazer a sua escolha.

Asus AMD Radeon RX5700 XT

A Asus AMD Radeon RX5700 XT é considerada por muitos como uma das melhores placas gráficas na relação qualidade/preço. O seu preço situa-se então na casa dos 450/500 €.

Como especificações oferece:

  • Memória: 8 GB GDDR6

  • Clock GPU:

       •  Base Clock: 1605 MHz
       •  Game Clock: 1755 MHz
       •  Boost Clock: até 1905 MHz

  • Processadores de fluxo: 2560
  • Interface Memória: 256 Bits
  • TDP: 225W
  • Interfaces de ligação: 1 x HDMI 2.0b + 3 x DisplayPort 1.4 + Suporte HDCP

Asus Nvidia GeForce GTX 1660 SUPER TUF 3

Indicada para o mundo gaming e para trabalhos de alto desempenho sugerimos a Nvidia GeForce GTX 1660 SUPER. Este modelo já se encontra na casa dos 300€.

Especificações principais:

  • Memória: 6 GB GDDR6
  • CUDA Core: 1408

  • Clock GPU:

       •  Base Clock: 1530 MHz
       •  Boost Clock: até 1860 MHz

  • Interface Memória: 192 Bits
  • Interfaces de ligação: 1 x HDMI 2.0b + 1 x DisplayPort 1.4 + Suporte HDCP + 1 x DVI-D

Sapphire Pulse Radeon RX 580

Outra das opções populares, por cerca de 200 €, é a Sapphire Pulse Radeon RX 580. É uma das opções indicadas para o mundo gaming, a um preço já mais atrativo que, em Black Friday, poderá surpreender ainda mais.

As especificações:

  • Memória: 8 GB GDDR5
  • Processadores de fluxo: 2304
  • Boost Clock: até 1366 MHz
  • Clock de Memória: 2000 MHz
  • Interface Memória: 256 Bits
  • Interfaces de ligação: 2 x HDMI 2.0b + 2 x DisplayPort 1.4 + Suporte HDCP + 1 x DVI-D

Nvidia GeForce RTX 2080 Ti

Num patamar bem mais acima das sugestões anteriores, sugerirmos a Nvidia GeForce RTX 2080 Ti. Esta placa gráfica destaca-se por ser uma das melhores para trabalhar com conteúdos 4K, ideal para ambiente profissional. O preço sem dúvida que é extremamente elevado (1200/1300 €), mas poderá ser um ótimo investimento nesta altura.

Como especificações oferece:

  • Memória: 11 GB GDDR6

  • Clock GPU:

       •  Base Clock: 1350 MHz
       •  Game Clock: 1635 MHz
       •  Boost Clock: até 1650 MHz

  • Núcleos CUDA: 4352
  • Interface Memória: 352 Bits
  • Interfaces de ligação: 1 x HDMI 2.0b + 3 x DisplayPort 1.4 + 1 x USB Tipo-C

«Лаборатория Касперского» разработала защиту для проектов на блокчейне

21 Nov 2019, 12:00 by Юлия Степанова

«Лаборатория Касперского» запустила сервис Kaspersky Enterprise Blockchain Security для защиты корпоративных проектов на основе блокчейн-технологий, сообщили “Ъ” в компании. Новым сервисом заинтересовались несколько компаний нефтегазового сектора и строительной сферы в России и за рубежом, с которыми сейчас ведутся переговоры. По оценке «Лаборатории Касперского», средняя стоимость сервиса для крупных организаций, например, финансового сектора составит $50–70 тыс.

«В крупных компаниях уже пару лет ведется разработка блокчейн-приложений. Их начинают внедрять на уровне целой компании. С увеличением числа корпоративных блокчейн-приложений атаки на них, вероятно, участятся»,— поясняет эксперт «Лаборатории Касперского» по блокчейн-технологиям для крупного бизнеса, руководитель направления Blockchain Security Павел Покровский.

По прогнозу IDC, к 2022 году глобальные расходы на подобные проекты достигнут $11,7 млрд. При этом по результатам опроса «Лаборатории Касперского» в 2019 году только 15% руководителей подразделений по информационной безопасности по всему миру считают, что блокчейн-технология окажет существенное влияние на информационную безопасность организаций в будущем.

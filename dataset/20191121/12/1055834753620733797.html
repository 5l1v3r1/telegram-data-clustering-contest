Japan: Daemon X Machina free Code Geass: Lelouch of the Rebellion Set now available

21 Nov 2019, 12:25 by My Nintendo News Admin

Japanese owners of the action-packed Daemon X Machina for the Nintendo Switch can claim free downloadable content based on the Code Geass anime franchise. The Code Geass: Lelouch of the Rebellion Set is now the second large content based update for Daemon X Machina and it includes the following cosmetic items:

  • Arsenal: Lancelot
  • Outer Suit: Suzaku Kururugi
  • Outer Edit Parts: Suzaku Kururugi

Повышение пенсионного возраста не помеха: для пожилых краснодарцев сохранились льготы по земельному налогу

21 Nov 2019, 12:35

Льготы по уплате земельного налога для горожан будут предоставляться, как и раньше: для женщин с 55 лет, для мужчин с 60 лет, то есть до наступления «нового» пенсионного возраста. Эту инициативу поддержали депутаты Гордумы на 86 заседании. 

По словам главы города Евгения Первышова, эта льгота – важная поддержка для жителей Краснодара. Было решено оставить ее на бессрочной основе для горожан, зарегистрированных на территории кубанской столицы. 

– Это значит, что, как и раньше, один земельный участок каждой категории не будет облагаться налогом, для многих семей это важно, – добавил мэр. 

Пожилые краснодарцы имеют право: 

- не платить земельный налог на один участок каждого вида разрешенного использования (дачный участок, участок под ИЖС и т.д.);

- на какой именно участок можно не выплачивать налог, собственник выбирает самостоятельно;

- под льготу не попадают земельные участки, используемые для ведения бизнеса.

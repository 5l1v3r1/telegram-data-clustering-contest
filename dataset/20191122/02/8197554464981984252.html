Pelaku Penyiraman Cairan Kimia Racik Soda Api Sambil Minum Kopi

22 Nov 2019, 02:09 by Antara, Clara Maria Tjandra Dewi H.

TEMPO.CO, Jakarta - Pelaku penyiraman cairan kimia di Jakarta Barat, Findra Yunico, meracik cairan soda api sambil minum kopi di Polsek Kebon Jeruk. Pria yang bekerja sebagai tukang servis AC itu menjalani gelar rekonstruksi perkara di tiga wilayah di Jakarta Barat, Kamis.

Pada adegan 43B, Findra memeragakan saat dia memesan kopi, meracik soda api, lalu memasukkannya dalam botol.

"Kamu sempat ngopi?" tanya aparat polisi saat gelar rekonstruksi di dekat Polsek Kebon Jeruk Jakarta Barat, Kamis, 21 November 2019. Findra pun mengangguk.

Dalam adegan rekontruksi sebelumnya, Findra membeli di soda api di sebuah toko bangunan di Jalan Meruya Ilir. Usai membeli soda api, dia mengendarai motor menuju lokasi servis AC tempatnya bekerja, yang kira-kira berjarak 600 meter.

Di tempat kerja, Findra meracik cairan soda api. Setelah itu, ia pergi meracik kembali soda api sembari minum kopi.

Setelah racikannya jadi, Findra mengendarai motor sambil mencari target korban perempuan secara acak. Sedikitnya enam perempuan menjadi korban penyiraman cairan kimia soda api yang dilakukannya. Dari enam perempuan itu, 5 di antaranya adalah siswi SMP.

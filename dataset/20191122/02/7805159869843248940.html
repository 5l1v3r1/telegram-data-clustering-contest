Realme 5 Pro update brings front camera Nightscape while Realme X2 Pro update brings November 2019 patches

22 Nov 2019, 02:15 by Pranob Mehrotra

Chinese smartphone manufacturer Realme has been doing a great job of releasing software updates for its devices over the last couple of weeks. The company recently released updates for the Realme 3 Pro and the Realme X, featuring the new dark mode and Android security patches. It has also released updates for the Realme 5 and Realme 2 Pro, with the same features. Most recently, it rolled out updates for the Realme XT, Realme 3/3i, and Realme X2. Now, the company has started rolling out updates for the Realme 5 Pro and the recently launched Realme X2 Pro.

Realme 5 Pro XDA Forums || Realme X2 Pro XDA Forums

The latest OTA update for the Realme 5 Pro (vRMX1971EX_11_A.13) brings the nightscape feature to the front camera, which will allow users to click better selfies in low-light conditions. The update also includes the Android security patches for November 2019, along with a few minor improvements and bug fixes. Here’s the official changelog for the latest ColorOS 6.0 update for the Realme 5 Pro:

  • Security

    • Android security patch: November, 2019

  • Camera

    • Added nightscape of front camera
    • Optimize the noise in dark environments
    • Optimized camera quality reddish issue
    • Optimized camera clarity

  • Game

    • Optimized frame dropping of Arena of Valor

  • Power Consumption

    • Optimized the power consumption of partial video scenes

  • Known issue fixed

    • Fixed carrier network issues in partial areas
    • Fixed the full-screen display notch area shows the prompt information error issue

The company’s first true flagship, the Realme X2 Pro, has also received an update that brings the Android security patch for November 2019, along with some improvements to system stability. Here’s the official changelog for ColorOS 6.0 (vRMX1931EX_11.A.06) for the Realme X2 Pro:

  • Security

    • Android security patch: November, 2019

  • System

    • Fixed some known issues and improved system stability

The OTA updates have already started rolling out to users in Europe. In case you haven’t received the update on your device yet, you can download the update package from the links below and manually install it on your phone.

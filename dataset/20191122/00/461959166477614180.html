Названы имена лучших участковых и инспекторов по делам несовершеннолетних области

22 Nov 2019 by Ирина Птуха

Подведены итоги ежегодного областного конкурса на звание "Лучший участковый уполномоченный полиции Тамбовской области" и "Лучший инспектор по делам несовершеннолетних Тамбовской области" за 2019 год. Имена победителей озвучили на межведомственной комиссии по профилактике правонарушений.

Первое место в конкурсе "Лучший участковый уполномоченный полиции Тамбовской области" заняли капитан полиции Александр Шишкин из Рассказово и майор полиции Виктор Теселкин из Котовска. Второе место жюри присудили старшему лейтенанту полиции Ярославу Лагуткин из Первомайского района и старшему лейтенанту полиции Александру Протасову из Мичуринска. На третьем месте лейтенант полиции Денис Катышев из Никифоровского района, майор полиции Кирилл Родюков и лейтенант Сергей Корякин из Тамбова.

Лучшим инспектором по делам несовершеннолетних Тамбовской области стала старший лейтенант полиции Наталия Козырева из Котовска. Второе место заняла майор полиции из Рассказово Анна Антонова, на третьем капитан полиции из Тамбова Инна Родионова.

Награждение лауреатов конкурса будет проведено на расширенной коллегии УМВД Российской Федерации по Тамбовской области по итогам 2019 года.

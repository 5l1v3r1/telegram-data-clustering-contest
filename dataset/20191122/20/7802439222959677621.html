Cara Mengatasi Limit Google Drive di PC dan Android

22 Nov 2019, 20:42 by Virgina Maulita Putri - detikInet

Jakarta - Google Drive merupakan layanan penyimpanan cloud yang ditawarkan oleh Google. Layanan ini bisa digunakan untuk mengunduh, mengunggah, dan berbagi file ke teman, keluarga, rekan kerja atau publik.

Jika sering menggunakan Google Drive kalian pasti sering menemukan peringatan yang berbunyi "Quota exceeded. Sorry, you can't view or download this file at this time".


Jika menemukan peringatan tersebut, artinya kalian tidak bisa mengunduh file tersebut karena telah melewati batas kuota. Lalu bagaimana cara mengakalinya?

Berikut cara mengatasi Google Drive yang limit di PC dan Android yang dirangkum detikINET:

1. Arti Limit Google Drive

Setiap file yang diunggah ke Google Drive memiliki kuota yang terbatas. Jika terlalu banyak orang mengunduh file yang kalian bagikan, file tersebut akan diblokir ketika kuota telah melewati batas.

Alasannya karena Google ingin menghindari penyalahgunaan. Apalagi jika traffic ke file tersebut sangat tinggi, maka file akan terkunci dan tidak bisa di-download selama 24 jam sebelum kuota tersebut di-reset.

Jika file yang kalian bagikan tidak menerima traffic yang terlalu tinggi, periode terkuncinya bisa saja lebih pendek. Tapi jika file tersebut terus menerima traffic yang tinggi kalian bisa saja harus menunggu berhari-hari sebelum bisa mengunduhnya.

2. Mengatasi Google Drive Limit di PC

Sebelum mulai, ada baiknya pastikan kalian sudah memiliki akun Google Drive. Jika sudah, sign in ke akun Google Drive kalian lewat browser di PC.

Setelah itu, cari file yang ingin kalian ubah agar bisa kembali diunduh. Klik kanan di file yang ingin kalian unduh dan pilih opsi Get shareable link. URL file tersebut akan muncul, lalu copy dan paste URL tersebut di tab baru.

Lewati langkah di atas jika kalian sudah memiliki link file tersebut dari orang lain. Setelah melihat halaman preview, cari tombol Add to Drive dengan ikon Drive dengan tanda plus di sisi kanan atas.

Setelah tombol tersebut diklik, file tersebut akan ditambahkan ke akun Google Drive kalian. Setelah menambahkan file, tombol tersebut akan berganti ikon dan klik sekali lagi untuk masuk My Drive atau halaman depan akun Google Drive kalian.

Cari file yang sudah kalian tambahkan lalu klik kanan dan pilih opsi Make a copy. Google Drive akan membutuhkan waktu beberapa saat untuk menyalin file tersebut.

Setelah file selesai disalin, kalian bisa langsung mengunduh file dengan nama "Copy of xyz" tanpa masalah. Untuk mengunduh, klik kanan di file tersebut dan pilih opsi Download lalu klik Download Anyway.


3. Mengatasi Google Drive Limit di Android

Untuk mengatasi kuota Google Drive yang terlampaui di ponsel Android kalian bisa langsung menggunakan aplikasinya. Caranya pun serupa dengan langkah di atas dan tidak terlalu sulit.

Pertama buka link file yang ingin kalian unduh di browser pilihan, misalnya Google Chrome. Link tersebut akan dibuka dalam Google Drive versi mobile yang tidak memiliki banyak kontrol.

Untuk bisa menambahkan file ke akun Drive kalian, buka menu Google Chrome dengan ikon tiga titik vertikal. Setelah itu centang opsi Desktop site untuk membuka situs versi desktop.

Setelah situs desktop terbuka, ulangi cara menambah file ke akun Drive seperti yang telah dicontohkan untuk PC di atas. Jika file sudah berhasil ditambahkan, buka aplikasi Google Drive di ponsel dan cari file yang ingin diunduh.

Klik menu yang berada di bawah file dan pilih opsi Make a copy. Setelah itu pilih My Drive, pilih folder tempat menyimpan copy dan klik Select. Tunggu proses copy selesai, lama atau tidaknya tergantung pada ukuran file dan kecepatan internet.

Setelah proses copy selesai, klik kanan di file hasil copy dan pilih opsi Download. File kalian akan secara otomatis diunduh ke ponsel.

Из­ме­нил­ся ре­жим ра­бо­ты под­раз­де­ле­ний ни­же­го­род­ской ГАИ

22 Nov 2019, 20:03

Режим работы регистрационно-экзаменационных подразделений Госавтоинспекции по Нижегородской области по понедельникам и четвергам изменился, сообщает пресс-служба ведомства.

Прием граждан по вопросам выдачи водительских удостоверений организован по понедельникам:

  • с 08:00 до 16:00 в межрайонном регистрационно-экзаменационном отделе ГИБДД (МРЭО, ул. Стрелка, 4-б);
  • с 09:00 до 17:45 в регистрационно-экзаменационных подразделениях ГИБДД ОМВД России по г. Арзамасу, ГИБДД УМВД России по г. Дзержинску, ГИБДД МО МВД России «Городецкий», ГИБДД ОМВД России по Кстовскому району, ГИБДД ОМВД России по Павловскому району, ГИБДД ОМВД России по г. Бор, ГИБДД ОМВД России по г. о. Семеновский.

Кроме того, изменения коснулись работы регистрационно-экзаменационных подразделений ГИБДД по четвергам. Теперь в МРЭО ГИБДД (в операционном зале по адресу: ул. Стрелка, 4-б) прием граждан будет осуществляться по четвергам в следующие часы:

  • с 13:00 до 18:30 по вопросам регистрации транспорта;
  • с 13:00 до 18:00 по вопросам выдачи водительских удостоверений.

В регистрационно-экзаменационных подразделениях области прием граждан по четвергам организован следующим образом:

  • с 09:00 до 17:45 в ОТНи РА ГИБДД УМВД России по г. Н. Новгороду, РЭО ГИБДД ОМВД России по г. Арзамасу, РЭО ГИБДД УМВД России по г. Дзержинску, РЭО ГИБДД МО МВД России «Городецкий», РЭО ГИБДД ОМВД России по Кстовскому району, РЭО ГИБДД ОМВД России по Павловскому району, РЭО ГИБДД ОМВД России по г. Бор, РЭО ГИБДД МО МВД России «Сергачский», РЭО ГИБДД ОМВД России по г. Выкса, РЭО ГИБДД МУ МВД России по ЗАТО Саров, РЭО ГИБДД ОМВД России по г. о. Семеновский, РЭО ГИБДД ОМВД России по Володарскому району, РЭГ ГИБДД ОМВД России по Богородскому району;
  • с 13:00 до 17:45 в РЭО ГИБДД ОМВД России по Балахнинскому району, РЭО ГИБДД МО МВД России «Кулебакский», РЭО ГИБДД МО МВД России «Княгининский», РЭО ГИБДД ОМВД России по Лукояновскому району, РЭО ГИБДД ОМВД России по Лысковскому району, РЭО ГИБДД МО МВД России «Уренский», РЭО ГИБДД ОМВД России по г. Шахунья.

Более подробно о режиме работы подразделений ГИБДД можно узнать на официальном сайте.

Госавтоинспекция напоминает, что по вопросам осуществления государственных услуг (регистрация автомототранспортных средств, получение водительских удостоверений, и т. п.) можно обратиться на портал госуслуг.

Ζώδια: Οι αστρολογικές προβλέψεις για το Σαββατοκύριακο 23 και 24 Νοεμβρίου 2019

22 Nov 2019, 21:51

Zώδια: Έντονο αυτό το Σαββατοκύριακο, όπου όλα μπορεί να συμβούν. Δείτε αναλυτικά την πρόβλεψη για το κάθε ζώδιο, για το Σαββατοκύριακο 23 και 24 Νοεμβρίου 2019, από την αστρολόγο Αλεξάνδρα Καρτά.

ΚΡΙΟΣ

Αυτό το διήμερο θα υπάρξουν γεγονότα που θα σας ξαφνιάσουν, κατά την διάρκεια ιδίως της Κυριακής. Ο Άρης πλέον στο ζώδιο του Σκορπιού, θα έρθει σε αντίθεση με τον πλανήτη Ουρανό, όπου και θα πρέπει να περιμένετε οικονομικές και συναισθηματικές ανατροπές.

ΤΑΥΡΟΣ

Αυτό το Σαββατοκύριακο θα γίνετε ανυπόμονοι, εριστικοί και επαναστάτες, όπου και θα βρεθείτε να αντιμετωπίζετε κάποια προσωπικά θέματα, με λίγα αποθέματα υπομονής. Φροντίστε να απομακρύνετε από κοντά σας τοξικά πρόσωπα.

ΔΙΔΥΜΟΙ

Πρέπει να καταλάβετε τις ευκαιρίες που θα παρουσιαστούν μέσα στο Σάββατο, αλλά προσέξτε τα απρόοπτα γεγονότα την Κυριακή, όπου και θα σας δημιουργήσουν δυσκολίες και κακή συνεννόηση με αγαπημένα σας πρόσωπα.

ΚΑΡΚΙΝΟΣ

Αυτό το Σαββατοκύριακο, θα υπάρξουν καλές συνθήκες για να κάνετε κάποιες κατασκευές στο σπίτι σας, ή σε ένα χώρο που εργάζεστε και ζείτε την καθημερινότητα σας. Την Κυριακή προσοχή σε συναισθηματικές συγκρούσεις.

ΛΕΩΝ

Το Σάββατο θα είναι μία μέρα που θα έχει πολύ επικοινωνία και θα βρεθείτε με πρόσωπα από το οικογενειακό σας περιβάλλον. Την Κυριακή, θα πρέπει να αντιμετωπίσετε μία ξαφνική ανατροπή, που θα συμβεί στο οικογενειακό σας περιβάλλον.

ΠΑΡΘΕΝΟΣ

Το Σάββατο θα είστε γεμάτοι από έγνοιες που αφορούν τα οικονομικά σας και ίσως να κάνετε κάποιες επιτυχημένες αγορές, μετά από την συμβουλή δικών σας προσώπων. Την Κυριακή θα έχετε κάποια νέα που θα σας ταράξουν.

ΖΥΓΟΣ

Η Σελήνη το Σάββατο στο δικό σας ζώδιο, βοηθά εν μέρη κάποια θέματα που έχουν να κάνουν με την προσωπική σας ζωή, αλλά την Κυριακή συναντά τον Άρη στο ζώδιο του Σκορπιού, όπου και αναστατώνει τα συναισθήματα σας.

ΣΚΟΡΠΙΟΣ

Ένα ανήσυχο διήμερο θα είναι αυτό για σας, αγαπητοί μου Σκορπιοί. Η Σελήνη δεν θα βοηθήσει καθόλου στο να ηρεμήσετε, εφόσον όλα θα είναι τεταμένα, ανατρεπτικά και χωρίς σταθερότητα.

ΤΟΞΟΤΗΣ

Αυτό το διήμερο είναι αρκετά δύσκολο σε θέματα εργασιακά και υγείας. Μπορεί να κάνετε νέα διαβήματα, αλλά μην ξεχνάτε ότι το απρόβλεπτο θα παραμονεύει κάπου εκεί κοντά, για να σας κάνει χαλάστρα.

ΑΙΓΟΚΕΡΩΣ

Τα επαγγελματικά σας θα σας απασχολήσουν έντονα, αλλά την Κυριακή θα έχετε κάποια ένταση που θα προέρχεται από την συναισθηματική σας ζωή. Πολλά θέματα στο οικογενειακό σας περιβάλλον θα σας κουράσουν.

ΥΔΡΟΧΟΟΣ

Με μία ανέμελη συμπεριφορά θα έχετε πολλά πράγματα να αντιμετωπίσετε αυτό το διήμερο, αλλά η προσοχή σας θα πρέπει να στραφεί και σε κάποιες αποφάσεις σας, που χωρίς πολύ σκέψη έχετε ολοκληρώσει.

ΙΧΘΥΕΣ

Η ενέργειά σας, θα είναι πολύ ανεβασμένη, ίσως πάνω από το κανονικό εφόσον θα αψηφήσετε πολλά πράγματα για να φτάσετε στους ποθητούς σας στόχους. Η γοητεία σας μεγάλη και επικίνδυνη.

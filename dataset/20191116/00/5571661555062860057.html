Китай разрешил VeChain отслеживать поставки чая

16 Nov 2019 by LetKnow News

Правительство китайской провинции Шуанцзян опубликовало одобрение блокчейн-платформы для отслеживания цепочек поставок чая, разработанной компанией VeChain.

По теме:

VeChain заключила партнерство с лидирующими производителями чая в регионе и внедрила свою платформу для отслеживания цепочек поставок, с помощью которой можно проследить полный путь чая от сырья до потребителя. Покупатели смогут отсканировать QR-код на упаковке, либо использовать технологию NFC, чтобы убедиться в подлинности продукта.

Разработанная компанией VeChain блокчейн-платформа для отслеживания чая пуэр стала одной из первых в рамках программы Национальной системы по отслеживанию стратегических продуктов, запущенной правительством Китая.

«Мы придаем большое значение этой платформе, которая отслеживает каждую упаковку чая, ее происхождение и логистику, а также которая позволит увеличить репутацию и популярность чая Юннань по всему миру», – подчеркнул представитель Министерства торговли Лу Сютянь (Lu Xiutian).

В апреле сингапурская компания VeChain прошла регистрацию в соответствии с новыми правилами Китая о регулировании блокчейна.

В результате, международная платформа Foodgates на базе публичного блокчейна VeChain Thor будет использоваться для поставки пищевых продуктов из Франции в Китай.

Помимо этого, VeChain поможет отследить поставки вина через блокчейн. Стартап договорился о партнерстве с австралийским виноделом Penfolds, чтобы выпустить в продажу партию вина, отслеживаемую с помощью блокчейна.

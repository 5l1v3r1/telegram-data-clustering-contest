Доступна для установки сборка Windows 10 19025

16 Nov 2019, 07:04 by Scorpion81

Начислено вознаграждение
Эта новость написана посетителем сайта, и за неё начислено вознаграждение.

В пятницу компания Microsoft в канале тестирования «Ранний доступ» выпустила предварительную сборку системы Windows 10 под номером 19025. Таким образом, на этой неделе сборок было две. Как обычно, новые функциональные возможности в них отсутствуют. Есть только довольно протяжённый список исправлений, одно из которых относится к работе песочницы и WDAG.

реклама

Кроме того, начали нормально работать определённые модели сканеров отпечатков пальцев, перестали зависать настройки VPN, повышена скорость входа в систему. Исправлена ошибка авторизации Windows Hello, которая не работала с определёнными приложениями. Есть множество других исправлений.

реклама

Среди известных ошибок присутствует вечная с античитерским ПО BattlEye, есть проблема с запуском приложения «Параметры», драйверы принтера продолжают отображаться как доступные уже после их установки. Кроме того, зависают некоторые модели внешних накопителей с интерфейсом USB 3.0.

Cuidado! Aplicações com login do Twitter ou Facebook podem estar a roubar os teus dados

25 Nov 2019, 21:19

O Twitter publicou hoje, no seu site oficial, a descoberta de uma vulnerabilidade que estará a comprometer as informações pessoais dos seus utilizadores. Algo que se manifesta quando usamos as nossas credenciais de acesso à rede social para nos autenticarmos noutras aplicações para smartphones.

Este método de autenticação é cada vez mais utilizado pela comodidade que oferece. No entanto, tal como adianta o Twitter, algumas aplicações que usam o kit de desenvolvimento oneAudience estão a tirar proveito indevido dessas informações.

Dados que estão a ser roubados

  • Email
  • Nome de utilizador
  • Último tweet

O Facebook também já referiu ter sido vítima do mesmo esquema, portanto, os mesmos dados estão comprometidos para todos os que usam esta rede social. A empresa de Mark Zuckerberg referiu já ter cortado todos os acessos a este kit de desenvolvimento para que este esquema não se perpetue.

A origem do problema não está no Twitter ou Facebook

Tal como o Twitter avança, a vulnerabilidade que despoletou este roubo de dados não é da sua responsabilidade. O problema está do lado do oneAudience incluído em algumas aplicações para smartphone.

A rede social diz ter provas que este esquema foi usado em algumas aplicações para Android, mas nada lhes garante que tenham tido sucesso. Já no que respeita ao iOS, ainda não existem provas de que tenha sido afetado pelo mesmo problema, embora exista essa possibilidade.

É ainda referido que a Google e Apple já foram notificadas do problema para que possam tomar as medidas necessárias. Para já, nenhuma delas se pronunciou sobre o assunto.

O que podes fazer para segurar os teus dados pessoais

Não há muito que possas fazer a não ser eliminar do teu smartphone aplicações que possam ser suspeitas. Sobretudo aquelas que tenhas descarregado fora da Play Store ou App Store.

Na impossibilidade de saber se os teus dados foram efetivamente comprometidos, o melhor é mudares as tuas passwords. De preferência deves ter uma password diferente para cada serviço.

Esta notícia deve servir de aviso para o uso desmedido dos logins do Twitter ou Facebook para entrar noutros serviços. Quando o fizeres, deves ter a consciência de que estás a fornecer as tuas credenciais a outros serviços. Portanto, só o deves fazer quando tiveres total confiança.

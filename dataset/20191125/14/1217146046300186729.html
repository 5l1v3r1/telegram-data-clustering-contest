Microsoft Fixes Windows 10 Qualcomm Wi-Fi Driver Update Block

25 Nov 2019, 14:23 by Lawrence Abrams

Microsoft has removed a compatibility hold that prevents users from upgrading to Windows 10 1903 and Windows 1909 if they are using older Qualcomm Wi-Fi drivers.

Redmond first applied this update block on May 21, 2019, after users were reporting that they were experiencing sporadic loss of Wi-Fi connectivity after upgrading to Windows 10 1903.

Those affected by the hold would no longer be offered the May 2019 Update (Windows 10 1903) and now the November 2019 Update (Windows 10 1909) when checking for new updates.

Users would also be shown a message if they attempted to use the Windows 10 Upgrade Assistant or the Media Creation Tool explaining that they need to install a new version of the Qualcomm wifi drive in order to continue.

To resolve this issue, Microsoft has started pushing out an updated Qualcomm Wi-Fi driver to affected users that will allow them to install the latest Windows 10 feature updates.

"This issue was resolved with an updated Qualcomm Wifi driver and the safeguard hold has been removed. "

While Microsoft has removed the the safeguard hold, customers should know that "it can take up to 48 hours before you can update to offered Windows 10, version 1903 or Windows Server, version 1903."

If users are not automatically offered the driver, you can try the following steps as another method to receive the updated wifi driver.

  1. Select Start or the search box in the task bar.
  2. Type "device manager" in the search field and select Device Manager when it appears.
  3. Within Device Manager, find Network adapters and expand it to see all the currently installed adapters.
  4. Right-click the Qualcomm adapter and select Update driver.
  5. When prompted, select Search automatically for updated drivers on the dialog that opens.
  6. Wait for the update fo finish. If it says, "The best driver for your device are already installed." then the driver was not updated.  If the driver was updated, you should now be able to Check for Updates and install the newer version of Windows 10.

If this still doesn't work, you can read this guide from Microsoft to learn about an alternative method of receiving the Qualcomm driver.

This past week, Microsoft also placed a compatibility hold on users of older versions of Avast and AVG Antivirus. To resolve that safeguard, users are required to upgraded to newer versions of the antivirus software.

NetApp и Google представили облачные решения

25 Nov 2019, 17:45

Компании NetApp и Google объявили ряд инициатив. В частности представлены NetApp Cloud Volumes Service, NetApp Cloud Volumes ONTAP для Google Cloud и поддержке Anthos на NetApp HCI, что должно помочь организациям сосредоточиться на инновациях в любых гибридных средах.

Отмечается, что предприятия стремятся в полной мере воспользоваться преимуществами гибкости и эффективности, которую предоставляет облако, но им необходимо быть уверенными в том, что их данные находятся в безопасности и в надежном месте, что сервис предлагается по разумной цене и может при необходимости масштабироваться. Для достижения этих целей бизнес строит свои Data Fabrics, которые обеспечивают максимальную гибкость при переходе в гибридные мультиоблачные среды. NetApp и Google Cloud помогают клиентам раскрыть весь потенциал своих данных на основе глубоко интегрированных решений корпоративного класса для работы с данными, оптимизированных и сертифицированных для Google Cloud.

«Исследования IDC показывают, что все больший процент развертываемых в облачных средах корпоративных рабочих нагрузок зависит от производительности систем хранения данных и что все больший приоритет отдается инструментам, обеспечивающим управление, автоматизацию и мобильность данных, — отметил Дипак Мохан (Deepak Mohan), директор по исследованиям Cloud Infrastructure Services, IDC. — Благодаря тому, что корпоративные сервисы по управлению данными NetApp предоставляются из Google Cloud, компании получают возможность достичь уровня производительности и функциональности, необходимого для поддержки наиболее интенсивных рабочих нагрузок, и немедленно внедрять инновации в локальных и публичных облачных средах».

«Мы с радостью расширяем партнерство с NetApp, позволяющее переводить данные клиентов в гибридные мультиоблачные среды и сделать решения NetApp общедоступными для Google Cloud, — прокомментировал анонс Кевин Ичхпурани (Kevin Ichhpurani), корпоративный вице-президент Global Ecosystem в Google Cloud. — Кроме того, у клиентов теперь есть возможность развернуть Anthos на сертифицированных HCI от NetApp, предоставляя клиентам больший выбор в аппаратных on-prem решениях, на которых они могут запускать Anthos».

Сервисы и решения по управлению данными NetApp помогают заказчикам эффективно управлять переходом от on-prem к гибридным и мультиоблачным средам.

 

«Наша миссия — помочь клиентам реализовать потенциал публичного облака и дать им возможность развивать инновации и свой бизнес с помощью облачных технологий — гармонично сочетается с подходами, которых придерживаются в Google Cloud как on-, так и off-prem, — заявил Энтони Лай (Anthony Lye), старший вице-президент и генеральный менеджер Cloud Data Services, NetApp. — Благодаря мощным корпоративным дата-сервисам и решениям NetApp, глубоко интегрированным в среды Google Cloud, бизнес получает возможность перейти на новый уровень управления данными и добиться беспрецедентной производительности, позволяющей в полной мере воспользоваться инновационным лидерством Google Cloud в разработке приложений, аналитике и машинном обучении».

NetApp Cloud Volumes Service для Google Cloud — это облачный файловый сервис, интегрированный в Google Cloud и обеспечивающий производительность, доступность и безопасность, необходимые для эффективной работы критически важных для бизнеса приложений. Кроме того, клиенты получили возможность c уверенностью запускать приложения в Google Cloud. Cloud Volumes Service для Google Cloud также стал доступен в Великобритании.

Общедоступным на глобальном уровне стал NetApp Cloud Volumes ONTAP для Google Cloud, максимально эффективное решение для управления данными, ускоряющее для предприятий переход к облачным решениям путем обеспечения повышенной защиты данных, эффективности хранения и мобильности в Google Cloud и гибридных мультиоблачных средах.

NetApp HCI — это масштабируемая локальная гибридная облачная инфраструктура, которая трансформирует частные облака в один из регионов мультиоблачной среды. NetApp HCI теперь сертифицирован для работы с Anthos Google Cloud, что дает клиентам возможность спокойно работать, зная, что в их распоряжении находится настоящая платформа корпоративного класса, сертифицированная Anthos для надежного развертывания и запуска своих приложений.

С помощью NetApp Trident, полностью поддерживаемого проекта с открытым исходным кодом, компании получают возможность оркестрировать сложную гибридную облачную среду, которая на локальном уровне объединяет Google Cloud, Anthos и NetApp HCI. Предприятия имеют возможность по мере необходимости масштабировать операции, добавляя вычислительные ресурсы или мощности для удовлетворения требований своего бизнеса. Выбор из широкого спектра ведущих платформ для приложений, включая Anthos, Kubernetes, VMware и NetApp Kubernetes Service.

Sicherheits-Support für Windows 7 wird eingestellt: Warum Sie jetzt handeln sollten

Microsofts Betriebssystem Windows 7 erhält bald keine Sicherheits-Updates mehr und auch dessen kurzlebiger Nachfolger, Windows 8.1, wird ab diesem Zeitpunkt nicht mehr mit neuen Funktionsupdates versorgt.

Microsoft selbst schreibt: "Der Support für Windows 7 endet am 14. Januar 2020. Anschließend sind weder technische Unterstützung noch Softwareupdates über Windows Update zum Schutz des PCs verfügbar. Microsoft empfiehlt dringend, vor Januar 2020 zu Windows 10 zu wechseln, um Situationen zu vermeiden, in denen Sie Service oder Support benötigen, der nicht mehr verfügbar ist."

Ab diesem Tag kann man davon ausgehen, dass es verstärkt Angriffe auf bestehende Sicherheitslücken geben wird.

Es ist also ratsam, auf ein anderes Betriebssystem zu wechseln. Mithilfe des offiziellen und kostenlosen Windows Media-Creation-Tools geht das Update von Windows 7 auf Windows 10 schnell und einfach: Sie brauchen dazu nur eine gültige Windows-10-Lizenz.

Windows 8.1, soll laut Microsoft noch bis 10. Januar 2023 mit Sicherheitsupdates versorgt werden - wird aber in Sachen neuer Funktionen mittlerweile links liegengelassen. Auch hier ist Windows 10 für eigentlich alle Nutzer die wesentlich bessere Wahl.

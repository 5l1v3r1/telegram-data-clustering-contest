Selain Mode Zombie, Ini Update Terbaru Game eSports COD Mobile

25 Nov 2019, 20:00 by Martini, Matheus Elmerio Giovanni

INDOSPORT.COM - Garena Indonesia akhirnya merilis update terbaru untuk game eSports Call of Duty (CoD): Mobile. Tak hanya menambahkan mode Zombie, ada beberapa fitur lain yang juga telah disempurnakan.

Untuk mode Zombie sendiri, makhluk setengah mayat itu akan terus bermunculan pada setiap wave di mode terbaru Call of Duty (CoD) Mobile ini. Untuk mengalahkannya, pemain diharuskan saling bekerja sama dengan tiga teman lainnya.

Kemudian, Garena juga menghadirkan map baru bernama Summit di dalam mode multiplayer. Map bernuansa salju ini merupakan map ikonik yang sebelumnya sudah sering dimainkan oleh para pecinta game seri Call of Duty sebelumnya.

Selain menambahkan mode baru ke dalam game CoD Mobile, Garena juga menambahkan fitur baru dan melakukan optimalisasi terhadap fitur yang sudah ada sehingga menambah pengalaman bermain para Soldiers.

Fitur baru berupa Bonus EXP memungkinkan pemain untuk menerima tambahan exp ketika bermain bersama teman. Pemain juga dimudahkan untuk bertemu lawan yang seimbang, menerima sistem skoring yang lebih adil, serta mendapatkan match dengan lebih cepat.

Perlu diketahui, seluruh rank pemain pada season sebelumnya akan direset pada hari ini, Senin (25/11/19). Maka, jangan lupa untuk update game CoD: Mobile masing-masing pemain.

Terakhir, Garena juga merilis karakter baru Alex Mason yang bisa dimiliki para pemain melalui Premium Pass Season 2. 

Para pemain Call of Duty (CoD) Mobile juga akan mendapatkan berbagai item in-game bernuansa salju mulai dari skin senjata AK-47 Glacier, Arctic.50 Glacier, skin kendaraan ATV - Alaska, dan Backpack - Aurora.

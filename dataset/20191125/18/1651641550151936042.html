Microsoft releases new firmware updates for Surface Pro X and Surface Laptop 3 with several improvements

25 Nov 2019, 18:48 by Pradeep

Microsoft has now released new firmware updates for the new Surface Pro X and Surface Laptop 3. These updates come with several improvements. For example, the updates for Surface Pro X improves Surface Pen charging performance, improved WiFi speed, improved single external monitor performance and more. The updates for Surface Laptop 3 improve system stability, audio performance, Cortana speech experience and more. Find the full change log below.

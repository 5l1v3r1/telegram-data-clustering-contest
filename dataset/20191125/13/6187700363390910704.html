Linus Torvalds anuncia el kernel de Linux 5.4: la última gran actualización del año trae soporte para el exFAT de Microsoft

25 Nov 2019, 13:41 by Gabriela González

Este domingo, Linus Torvalds anunció el lanzamiento del kernel de Linux 5.4, nombre clave es "Kleptomaniac Octopus". Esta vendría a ser la última gran actualización de 2019 y llega con varias novedades interesante.

Linux 5.4 trae el soporte para el sistema de archivos exFAT, propiedad de Microsoft, gráficos Radeon mucho más rápidos, soporte para Intel Tiger Lake, mejoras en Raspberry Pi, soporte para mucho hardware nuevo, y numerosas actualizaciones de seguridad.

Lo más interesante en Linux 5.4

Entre las novedades más destacadas podemos mencionar:

  • Soporte para las GPU AMD Arcturus
  • Soporte para las GPUs Navi 12 y Navi 14
  • Soporte para las gráficas Intel Tiger Lage "Gen12"
  • Soporte para el SoC Qualcomm Snapdragon 855
  • Soporte para Intel Ice Lake Thunderbolt
  • Reporte de temperatura para la serie de procesadores AMD Ryzen 3000
  • Mejoras de rendimiento para Raspberry Pi SPI
  • Soporte para el receptor Logitech Lightspeed
  • Correcciones para juegos nuevos que se ejecutan con Wine/Proton
  • La función de seguridad Linux LOCKDOWN LSM se incorporó finalmente

Cabe destacar que el soporte para exFAT llegó gracias a que Microsoft finalmente publicó la especificación y dejó de objetar que el soporte se añadiera al kernel de Linux. exFAT es bastante popular como sistema de archivos en tarjetas SD y otros dispositivos.

Tovarlds escribió: "No ha pasado mucho esta última semana, que es justo como me gusta. Y como era de esperar, la mayoría de las solicitudes de pull que recibí eran para la ventana de fusión 5.5, en la que obviamente comenzaré a trabajar mañana. Lo poco que hay aquí son principalmente algunas actualizaciones de red (mezcla de controladores de red y redes básicas), y algunas actualizaciones menores de controladores de GPU".

Linux 5.5 será el primer lanzamiento de 2020 y se espera para finales de enero o principios de febrero. Esa versión del kernel traerá novedades como el "live-patching" que monitorizará el sistema para manejar mejor los parches, soporte para overcloking para las GPUs Navi de AMD, mejoras en la autonomía de portátiles con gráficas Intel o NVIDIA discretas, y mucho más.

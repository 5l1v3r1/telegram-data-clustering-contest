Windows 10: atualização de novembro traz novo bug que trava pesquisa do sistema

25 Nov 2019

A Microsoft já iniciou a distribuição da Atualização de Novembro para a maioria dos usuários do Windows 10. No entanto, muita gente resolveu aguardar um pouco mais para saber se o software trazia algum bug importante, algo que está se tornando rotina na história do sistema.

Como não tivemos nenhum relato nas últimas semanas, o update já está presente em milhares de PCs. Contudo, apesar de não trazer problemas de grande escala, a atualização de novembro (19H2) acaba de ter o seu primeiro bug relatado por alguns usuários.

Isso porque a versão 19H2 está quebrando a pesquisa do File Explorer. Veja abaixo que o software agora exibe mais resultados quando o usuário digita algum termo na pequena barra suspensa. Essa está sendo considerada uma das maiores alterações da atualização de novembro.

Segundo relatos de algumas pessoas, ao digitar algum termo na barra de pesquisa, o sistema acaba congelando por um longo período. A única solução disponível é reiniciar o processo do Windows Explorer:

Na verdade, às vezes a caixa de pesquisa está ficando totalmente congelada. No Windows Explorer (pastas do processo do Windows Explorer), não é possível clicar com o botão direito nem com o botão esquerdo até que você force o reinício do Windows Explorer. Isso o corrige por um tempo, mas tudo acaba travando novamente

Por enquanto, apesar da crescente quantidade de usuários relatando o mesmo problema, a Microsoft ainda não se manifestou oficialmente sobre o assunto. Com isso, a única solução disponível é usar o Gerenciador de Tarefas para descongelar o Windows Explorer.

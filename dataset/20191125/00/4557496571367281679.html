Google將於2021年終止Android雲端列印服務

25 Nov 2019 by 克勞德

Google 日前宣布旗下 Cloud Print 雲端列印服務，將於 2021 年元旦正式退場，同時表示 Chrome OS 對於印表機的支援已趨完善，市面上也有不少印表機品牌，可直接透過 Android、iOS 等裝置使用雲端列印服務，因此認定 Cloud Print 不再具有其必要性。此外，Google 也保留約一年的時間，好讓用戶在服務關閉前，能盡早做出對應準備。

Google Cloud Print 雲端列印服務自 2010 年 4 月推出，最早是針對 Chromebook 的列印需求而設，使用者可透過 Wi-Fi 連線進行印表機操作，且無需安裝驅動程式即可使用；之後，更於 Android 與 iOS 平台的 Chrome 瀏覽器導入此功能，大幅度提高了 Cloud Print 雲端列印服務的實用性，也影響了後續十年的雲端列印發展。

▲Google Cloud Print 雲端列印服務自 2010 年推出以來，一直維持著 Beta 測試版的字樣。

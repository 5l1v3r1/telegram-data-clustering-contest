Scythe Fuma 2, Un refrigerador de CPU con ventiladores de giro inverso

25 Nov 2019, 00:56 by Gustavo Gamarra

El Fuma 2 de Scythe se aparta del anterior refrigerador de CPU Fuma, que tiene un sistema de doble torre con seis Heatpipes, y ventiladores dobles para ayudar a disipar el calor, junto con el novedoso sistema Reverse Jet Flow para disipar el calor en el área circundante.

Scythe Fuma 2, Un refrigerador de CPU con ventiladores de giro inverso

El sistema ‘Reverse Jet Flow’ hace que cada ventilador gire al revés, lo que permite una mayor presión estática y un flujo de aire más estable que permite que la CPU se enfríe con mayor eficacia.

El disipador térmico asimétrico permite una mejor compatibilidad con los modulos RAM más altos, especialmente cuando hay dos ranuras de RAM a cada lado de la CPU o una configuración de memoria de cuatro canales. El Scythe Fuma 2 mide 137 mm de ancho, 154,5 mm de alto y 131 mm de profundidad, lo que hace que este refrigerador de CPU sea muy grande en comparación con los otros refrigeradores que estamos viendo.

Junto con el disipador de calor asimétrico, los ventiladores incluidos son dos modelos diferentes, con dos tipos de espesores, uno de 17mm incluyendo las almohadillas de goma anti-vibración y el otro de 27mm con las almohadillas de goma anti-vibración. El ventilador más delgado permite la compatibilidad con la memoria de canales de calidad, aunque si está dispuesto a renunciar a la memoria de cuatro canales, se puede conectar un tercer ventilador, aunque no otorgaría un aumento real de rendimiento térmico. Estos ventiladores giran a una velocidad variable, dependiendo del calor/carga que está generando actualmente la propia CPU, pero pueden girar entre 200 y 1200RPM cada uno.

Los sockets en los que se montará este refrigerador de CPU son el LGA 775/ 115x/ 1366/ 2011(V3)/2066 que incluye los sockets más recientes. En el lado AMD, soporta desde el socket FM1 hasta el socket actual AM4.

El Fuma 2 de Scythe tiene un costo de 59.99 USD en Amazon.

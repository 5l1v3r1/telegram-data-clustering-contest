Update for Realme 5 Pro in Europe brings Nightscape for the selfie cam

25 Nov 2019 by Peter

The Realme 5 Pro is receiving a camera-focused update. Version RMX1971EX_11_A.13 is currently rolling out in Europe after arriving in India last week. Here’s what’s new.

The selfie camera gains Nightscape mode. The rear camera should also perform better in the dark thanks to some additional tweaking (it already had Nightscape). The image clarity in general should be improved and if you’ve experienced a reddish tint in some of your photos, that should be fixed now too.

Playing Arena of Valor should result in fewer dropped frames with this update, which also reduces power usage while watching video.

This update will move your Realme 5 Pro up to the November 2019 security patch. The software is still based on Color OS 6.0, though, the plan is to skip 6.1 and jump straight to v7.0.

You should be getting a notification for the over-the-air update soon. If can’t wait, you can follow the Source link where you’ll find a manual download and instructions on how to install the update.

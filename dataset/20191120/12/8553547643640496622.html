Резервные копии Veeam Backup для Office 365 теперь можно хранить в облаке GigaCloud

20 Nov 2019, 12:56

Компания GigaCloud сообщает о запуске Veeam Backup для Microsoft Office 365 – инструмента для резервного копирования данных, которые хранятся в облаке Microsoft: электронной почты в Exchange Online, документов OneDrive для бизнеса, Microsoft Teams и узлов SharePoint Online.

Облако Microsoft обеспечивает высокую доступность данных, но уязвима к человеческим ошибкам и действий злоумышленников. Microsoft оставляет сохранения данных в зоне ответственности пользователя. Поэтому GigaCloud предоставляет клиентам возможность создавать и сохранять резервные копии данных в репозитории облака GigaCloud, что гарантирует их надежную защиту.

Инструмент предназначен для компаний с большим штатом, которые используют или планируют использовать облачный Office 365 для работы.

Что дает Veeam Backup для Microsoft Office 365?

  • Получение из облака Microsoft данных вашего облачного экземпляра MS Office 365 (Exchange Online, SharePoint Online, OneDrive для бизнеса и Microsoft Teams).
  • Создание в облаке GigaСloud защищенного архива резервных копий в формате базы данных Jet Blue (еженедельно/ ежедневно/ ежечасно несколько раз – в случае необходимости).
  • Быстрый поиск хранимых данных и гранулярное восстановления отдельных объектов с облачного архива GigaСloud (возможно восстановление данных отдельных пользователей, отдельных листов и файлов).

Сохранение резервной копии в репозитории реализуется на стороне облачного оператора. В облаке развернуто решение Veeam backup and replication server с компонентом, который позволяет создавать резервные копии конкретного пользователя из облака Office 365. Для авторизации в облаке Office 365 необходимые пароль и логин, предоставляемых клиентом. Далее, по требованию клиента, техническими специалистами оператора конфигурируется бэкап-план. В нем указывают учетные записи которых пользователей необходимо резервировать, с какой периодичностью делать полные и инкрементальные бэкапы.

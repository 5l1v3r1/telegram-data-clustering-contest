В Отделении ПФР по Тверской области состоится видеосеминар для преподавателей курсов компьютерной грамотности

20 Nov 2019, 14:54

Фото: budemdobree.ru

Завтра, 21 ноября, с 9.30 до 12.30 Пенсионный фонд Российской Федерации совместно с ПАО «Ростелеком» по селекторной видеосвязи ПФР проводит пятый дистанционный обучающий семинар по теме: «Особенности организации курсов компьютерной грамотности для пенсионеров на основе программы «Азбука Интернета», обмен практиками». 

В видеосеминаре, который будет транслироваться в Отделении ПФР по Тверской области, по адресу, Тверь, ул. Вагжанова, д. 9, стр. 1, примут участие сотрудники социальных учреждений.

Как «Азбука Интернета» поможет преподавателям в их работе с пожилыми людьми Тверской области? На этот и другие вопросы ответят участники семинара.

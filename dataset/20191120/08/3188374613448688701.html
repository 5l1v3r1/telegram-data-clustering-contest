Microsoft почала поширювати велике оновлення Windows 10

20 Nov 2019, 08:48 by natali

Минулого тижня компанія Microsoft зробила доступним велике піврічне оновлення Windows 10 November 2019 Update (версія 1909) для всіх, хто охочих. Тепер же користувачам стали доступні додаткові варіанти.

Раніше користувачі могли вручну перевірити наявність нової версії, після чого завантажувати й установлювати Windows 10 November 2019 Update. Менш нетерплячі могли почекати автоматичного оновлення.

Вчора Microsoft оновила Windows 10 Update Assistant (Помічник з оновлення Windows 10), так що тепер оновлення Windows 10 November 2019 Update відбудеться автоматично після установки «Помічника».

Крім того, оновлення тепер доступне через інструмент Media Creation Tool, який дозволяє створити та завантажити образ диска з Windows 10 (файл ISO). Файл ISO можна використовувати для встановлення або перевстановлення Windows 10, а також для створення інсталяційної флешки або диска.

Tor Project принимает пожертвования через Lightning Network

20 Nov 2019 by LetKnow News

Некоммерческая организация Tor Project, занимающаяся разработкой решений для установления анонимного сетевого соединения, во вторник объявила о начале приема пожертвований в биткоинах через решение по масштабированию второго уровня Lightning Network.

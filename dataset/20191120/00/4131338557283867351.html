Повышение удобства установки расширений в браузерах при работе с веб-клиентом

20 Nov 2019

Данная статья является анонсом новой функциональности.
Не рекомендуется использовать содержание данной статьи для освоения новой функциональности.
Полное описание новой функциональности будет приведено в документации к соответствующей версии.
Полный список изменений в новой версии приводится в файле v8Update.htm.

Планируется в версии 8.3.17

Для полноценной работы веб-клиента 1С иногда может потребоваться установить дополнительные программы: расширение для браузера и/или расширения для работы с файлами, криптографией, программой «1С:Предприятие – оповещения и запуск», внешние компоненты. Механизмы безопасности браузеров не дают сделать это полностью автоматически и требуют участия пользователя на некоторых фазах установки; в некоторых случаях может потребоваться перезапуск браузера.

Раньше процесс установки не был интуитивно понятен: диалоги могли содержать много информации (не всегда актуальной), пользователю в процессе установки нужно было выполнять несколько действий параллельно (устанавливать расширение для браузера и расширения для работы с файлами, криптографией и т.д.). Кроме того, в Firefox не всегда было понятно, что произошло скачивания файла.

В версии 8.3.17 мы переработаем интерфейс установки расширений, сделаем его более удобным для пользователей, кроме того:

  • Пользователи Chrome смогут продолжить работу сразу после установки расширения для браузера, без перезагрузки браузера.
  • Пользователи Firefox будут устанавливать расширение из магазина расширений Firefox

Таким образом, количество действий пользователя сведено до минимума, убраны лишние диалоги.

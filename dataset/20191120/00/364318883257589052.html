เผยรายชื่อเกมเข้าชิงสาขาต่างๆ ในงาน The Game Awards 2019

20 Nov 2019 by Jeeranart Ruengsub

ตลอดปีที่ผ่านมา มีเกมต่างๆ ออกมาให้เล่นกันมากมาย หลากหลายแนว พอเข้าสู่ช่วงปลายปี วงการเกมก็มีการมอบรางวัลเช่นกัน เหมือนกับวงการบันเทิง The Game Awards 2019 งานประกาศรางวัลของคนในวงการเกม ทั้งเบื้องหน้าและเบื้องหลัง ตอนนี้มีการประกาศเกมที่เข้าชิงสาขาต่างๆ ใครรักเกมไหน ชอบเกมไหน สามารถเข้าไปลงคะแนนกันได้ผ่านเว็บไซต์ The Game Awards 2019 คลิกที่นี่ และ Google คลิกที่นี่

The Game Awards 2019 จะประกาศรางวัลวันที่ 12 ธันวาคม 2019 ผ่านทาง YouTube, Twitch, Mixer, Facebook, Twitter

Damkar Jakut: Api di Gedung Hailai Sudah Padam, Sedang Proses Pendinginan

05 Nov 2019, 07:51 by Indra Komara

Jakarta - Si jago merah yang melalap Gedung Hailai, Ancol, Jakarta Utara, akhirnya sudah padam. Saat ini masih dilakukan proses pendinginan.

"Api padam pukul 03.00 WIB. Sekarang dalam proses pendinginan," kata Kasudin Gulkarmat Jakarta Utara Satriadi Gunawan, saat dikonfirmasi, Selasa (5/11/2019).

Menurut dia, tidak ada korban akibat kebakaran tersebut. Penyebab kebakaran masih dilakukan penyelidikan, termasuk total kerugian.

"Penyebab dan kerugian masih di selidiki," ujarnya.

Sebelumnya, kebakaran di Gedung Hailai terjadi pada Selasa (4/11) malam pukul 22.50 WIB. Api cepat menyambar hampir ke seluruh bangunan gedung.

Sekretaris Kampung Siaga Bencana (KSB) Ancol, Amar menjelaskan, api dengan cepat melalap Gedung Hailai diduga karena angin kencang dan material mudah terbakar yang ada di dalam gedung. Amar mengatakan, gedung tersebut sudah dalam tahap proses pembongkaran.

"Jadi saat ini sedang ada tahap pembongkaran oleh kontraktor yang bongkar, jadi konstruksi ada proses pengelasan. Kalau sumber api diduga dari hasil pekerjaan las itu. Cuma waktu kejadian sudah tidak ada waktu pengerjaan, pengerjaan berhenti pukul 17.00 WIB. Jadi ini kayaknya sisa besi masih panas, terus di bawah juga banyak material busa, kayak busa sofa, jadi mungkin angin juga agak kenceng jadi tiba-tiba besar (api)," jelas Amar saat dihubungi terpisah.

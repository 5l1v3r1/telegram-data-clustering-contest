7 главных анонсов конференции Microsoft Ignite

05 Nov 2019, 09:26 by Александра Кукуть

Издание TechCrunch выбрало самые важные новости с ежегодной конференции Microsoft Ignite, которая в этот раз проходит 4–8 ноября в Орландо. Всего Microsoft сделала более 175 анонсов.

Azure Arc

Azure Arc — новый набор технологий, который позволяет компаниям использовать сервисы и инструменты Azure для управления ресурсами в любой облачной инфраструктуре, включая Amazon AWS и Google Cloud, на локальных серверах и граничных устройствах. Пока доступна превью-версия Azure Arc.

Project Cortex

Первый новый сервис Microsoft 365 со времени запуска Microsoft Teams. По словам компании, этот AI-инструмент создаёт интерактивную «сеть знаний» организации и представляет собой нечто среднее между классификатором корпоративных данных и цифровым ассистентом. Сервис автоматически находит, сортирует по темам информацию из документов в различных сервисах Microsoft и в нужный момент высвечивает ссылки на них, к примеру, в приложениях Office, Outlook и Teams. Публичный доступ откроется в первой половине 2020 года, сейчас проходит закрытое тестирование Project Cortex.

Endpoint Manager

Microsoft объединила сервисы ConfigMgr и Intune, которые помогают компаниям управлять ПК, ноутбуками и мобильными устройствами сотрудников под брендом Endpoint Manager, а также запустила ряд новых инструментов и рекомендаций для компаний.

Microsoft Edge

Публичный доступ к браузеру Microsoft Edge на базе движка Chromium для Windows и macOS будет открыт 15 января 2020 года на более чем 90 языках. С сегодняшнего дня можно скачать ознакомительную версию.

Веб-версия Visual Studio Online

Microsoft анонсировала превью-доступ к веб-версии среды разработки Visual Studio, которая позволяет работать с кодом и редактировать его с любого устройства через браузер. В мае Visual Studio Online стала доступна ограниченному кругу разработчиков, а теперь открыта для всех.

Power Virtual Agents

Power Virtual Agents — новый бескодовый/малокодовый инструмент для создания чат-ботов, использующий технологии и инструменты машинного обучения Azure. Инструмент является частью платформы Power Platform, в состав которой также входят средство автоматизации процессов Microsoft Flow (теперь Power Automate) и Power BI.

Cortana

Microsoft интегрировала голосовой ассистент Cortana интегрирован в Outlook на iOS (интеграция с Android будет позже), чтобы повысить продуктивность сервиса. Cortana сможет зачитывать входящие сообщения, назначать встречи и делать рассылку участникам, создавать ежедневную подборку запланированных в календаре событий и напоминать ответить на важные письма. Также ассистент теперь «заговорит» мужским голосом.

Бонус

  • Project Silica: Microsoft совместно с Warner Bros. записала фильм «Супермен» 1978 года на кварцевом стекле. По словам компании, проект использует последние достижения в области лазерной оптики и AI для хранения данных.
  • Решение Azure Blockchain Tokens позволяет токенизировать физические или цифровые активы, управлять ими и использовать на блокчейн-платформах различного типа. Сервис предоставляет доступ к набору открытых API для взаимодействия с токенами. Как пример использования подобных токенов в бизнесе — цифровизация цепочки поставок, программ лояльности, документооборота. Решение доступно в предварительной версии.
  • Улучшена запись голоса в Microsoft Stream: в сервисе теперь используется машинное обучение для обнаружения и устранения нежелательного фонового шума в любом видео одним щелчком мыши.
  • Кнопка чатов Teams в Outlook позволяет удобно вести переписку и переключаться между цепочками писем в Outlook и чатами в Teams.

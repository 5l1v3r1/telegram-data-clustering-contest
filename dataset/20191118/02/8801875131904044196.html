微軟 Game Stack 橫空出世！全球遊戲開發生態系蔚然成形

18 Nov 2019, 02:00 by TechNews

遊戲已然成為全球現象，在全球 76 億人口中，光是遊戲玩家就高達 20 億人，約占 Internet 使用者的一半，而整個遊戲產業規模達 1,500 億美元。為了搶占遊戲市場大餅，微軟推出結合 Azure 雲端，同時整合全面性各類型遊戲開發解決方案的 Game Stack，進而打造出跨平台、跨裝置的全球遊戲開發生態系。

從遊戲設計到發布，再到營運獲利的全方位工具與服務

微軟 Game Stack 的最大特色，就是將全面性各類型的遊戲開發工具、平台、服務與解決方案全部集結在一起，以便讓開發者能夠用來開發出從遊樂器主機、PC 到智慧型手機，再到 Web 等各類型更多元與更完美的遊戲。

台灣微軟行銷暨營運事業部資深雲端產品經理蕭博仁表示，Game Stack 就是把過去供 Xbox 使用的 Azure 運算、儲存等 IaaS 雲端基礎架構及 PaaS 服務，轉而提供給遊戲開發商使用的解決方案。嚴格說來，Game Stack 就是 Azure，它是採行 B2B 商業模式並屬於遊戲開發商專屬版本的 Azure。

不論如何，Game Stack 為開發者提供了可運行在各種裝置與平台上極其包羅萬象的開發資源，遊戲開發商甚至可以在設計、建立、測試、發布、獲利變現到營運等不同階段選擇最佳、最適合的 Game Stack 解決方案，進而滿足最完美妥善的遊戲開發生命週期管理要求。

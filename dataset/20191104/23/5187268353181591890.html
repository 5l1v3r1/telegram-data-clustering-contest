Dropbox Transfer copia WeTransfer: Partilhe até 100GB gratuitamente

04 Nov 2019, 23:00 by Pedro Pinto

Com o lançamento de outros serviços, o Dropbox tem vindo a perder popularidade. Nesse sentido, é preciso atacar a concorrência oferecendo as melhores funcionalidades. Em julho o Dropbox anunciou a nova funcionalidade Dropbox Transfer.

Hoje o Dropbox Transfer passou a estar disponível para quem tem planos subscritos no Dropbox.

Dropbox Transfer – send-it-and-forget-it

Chama-se Dropbox Transfer e é uma funcionalidade que faz confundir o Dropbox com o WeTransfer. De acordo com o que foi anunciado, o Dropbox é normalmente um serviço direcionado para trabalho colaborativo. Este novo serviço é uma funcionalidade “send-it-and-forget-it”.

O limite no volume dos ficheiros a partilhar pode ser até 100 GB.

Para usar este serviço basta selecionar os ficheiros do Desktop ou da própria conta Dropbox. Depois é gerado um link que pode ser facilmente partilhado com outras pessoas.

É ainda possível proteger o conteúdo com uma password ou definir um período de expiração. Depois do link partilhado, é possível consultar um conjunto de estatísticas.

Atualmente o Transfer pode ser usado via Web, via app para Desktop e via iOS. Para usar o serviço basta que aceda aqui.

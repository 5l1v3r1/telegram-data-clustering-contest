Программисты из Нарьян-Мара разработали приложение «Мои Документы» для iOS

17 Nov 2019, 21:42 by Анжелика Галанова

В Ненецком автономном округе для платформы iOS по заказу от МФЦ округа разработали удобное для жителей приложение. По нему можно отследить готовность услуги.

В Ненецком округе начало работать приложение «Мои Документы» для гаджетов на платформе iOS, сообщает сайт администрации округа.

Приложение «Мои документы» заказал и оплатил Многофункциональный центр государственных и муниципальных услуг. С заказом справились местные программисты из Нарьян-Мара. На Android подобное приложение уже было разработано, теперь потребовалось аналогичное для платформы iOS.

С помощью приложения можно узнать информацию о предоставляемых в МФЦ услугах, сделать предварительную запись на приём, узнать адрес нужного офиса, статус заявления, оценить работу МФЦ.

Скачать приложение можно бесплатно в App Store, набрав в поисковике «Мои Документы НАО». Кстати, МФЦ Ненецкого округа предоставляет более 200 различных услуг.

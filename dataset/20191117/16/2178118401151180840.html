В «Хранилище судного дня» при помощи Microsoft поместят весь открытый код с GitHub

17 Nov 2019, 16:42 by Kirill Sergeev

Компания Microsoft планирует сохранить весь открытый исходный код с GitHub в «Арктическом мировом архиве» на острове Шпицберген. Туда уже поместили первую специальную катушку со 120 ГБ информации.

Для хранения кода Microsoft использует сверхстойкую плёнку, покрытую порошком из оксида железа. По словам компании-производителя, она способна хранить данные до 750 лет в нормальных условиях и до 2 тысяч лет в холодной и сухой пещере с низким содержанием кислорода. При этом чтение информации с нее осуществляется при помощи лупы.

Первая бобина была помещена в «Хранилище судного дня» 24 октября. Любопытно, что доставил ее на Шпицберген лично CEO GitHub Нат Фридман. «Соседями» катушки с исходным кодом Linux, Android и 6000 важных open-source приложений стали ватиканские архивы, записи о недвижимости в Бразилии, итальянские фильмы и рецепт секретного соуса для неназванного бургера.

В общей сложности Microsoft и GitHub планируют разместить в «Хранилище судного дня» 200 катушек, каждая из которых будет содержать 120 гигабайт ПО с открытым исходным кодом.

«Арктический мировой архив» был открыт в марте 2017 года рядом с «Всемирным семяхранилищем» в заброшенной угольной шахте. В данный момент он представляет из себя обычный металлический шкаф.

O Chrome θα υποδεικνύει τις πιο αργές ιστοσελίδες

12 Nov 2019, 22:59 by Τάσος Χριστοδούλου

Τις ιστοσελίδες που φορτώνουν αργά λόγω κακού προγραμματισμού ενδέχεται να υποδεικνύει σύντομα ο δημοφιλής browser της Google, Chrome.
 
 
Σε σχετικό blog post η εταιρεία ανέφερε πως δουλεύει πάνω σε μια σειρά από «speed badging» συστήματα που θα ενημερώνουν τους χρήστες γιατί μια σελίδα αργεί να φορτώσει. Ωστόσο, δεν αποκαλύφθηκαν λεπτομέρειες σχετικά με το πότε η λειτουργία αυτή θα εισαχθεί στον Chrome.
 
Στο εν λόγω blog post η Google αναφέρει πως άρχισε το project αυτό για να «βοηθήσει τους χρήστες να καταλαβαίνουν πότε μια σελίδα μπορεί να φορτώνει αργά, ενώ παράλληλα ανταμείβει τα sites που παρέχουν γρήγορες εμπειρίες». Όπως συμπληρώνεται, δουλεύει παράλληλα πάνω σε στοιχεία και metrics που θα δείχνουν πότε οι ιστοσελίδα είναι στημένες με τρόπο που τις καθιστά γενικά αργές.
 
Οι ενδείξεις αυτές θα μπορούσαν να περιλαμβάνουν μηνύματα κειμένου που ενημερώνουν πως μια σελίδα «συνήθως φορτώνει αργά» ή να αλλάζουν το χρώμα μιας μπάρας έτσι ώστε ο χρήστης να βλέπει ότι η σελίδα καθυστερεί.
 
Στο blog αναφέρεται πως το σχέδιο αυτό για την «αξιολόγηση» των sites θα εξελίσσεται με το πέρασμα του χρόνου, και απώτερος στόχος είναι η βελτίωση του τρόπου με τον οποίο δημιουργούνται οι ιστοσελίδες. «Εξετάζουμε πολύ στενά την προσέγγισή μας όσον αφορά στο πού βάζουμε τον πήχυ για αυτό που θεωρείται καλή εμπειρία χρήστη και ελπίζουμε να φτάσουμε σε κάτι που θα είναι πρακτικά επιτεύξιμο από όλους τους developers» σημειώνεται σχετικά.
 
«Το σχέδιό μας για τον εντοπισμό sites που είναι γρήγορα ή αργά θα υλοποιηθεί σταδιακά, με ολοένα και αυστηρότερα κριτήρια. Ο μακροπρόθεσμος στόχος μας είναι να ορίσουμε το badging (κατάδειξη) για εμπειρέις υψηλής ποιότητας, που μπορεί να περιλαμβάνουν σήματα πέρα απλά από την ταχύτητα» αναφέρεται στηο blog post.

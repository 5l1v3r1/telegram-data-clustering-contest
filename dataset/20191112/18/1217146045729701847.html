Windows 10 November 2019 Update Is Now Available

12 Nov 2019, 18:28 by Mayank Parmar

Microsoft has officially released the November 2019 Update for Windows 10 and is in the process of rolling out to users. This update will upgrade the operating system to Windows 10 1909 and is an optional feature update that will not be installed automatically for users running newer versions of Windows.

As previously reported, Windows 10 November 2019 Update is not a big release and it doesn't come with any major improvements.  Instead this should be considered a service pack that includes bug fixes and mostly feature improvements.

The main improvements from this build include:

  • You can now create an event directly from the Calendar flyout on the Taskbar.
  • Action Center is getting a new button at the top and Microsoft is also updating notifications settings with a new option that will help you sort notifications by most recently shown. The notification toast has been updated with a new button to quickly mute notifications from that particular app.
  • File Explorer updated with new Search experience and OneDrive integration.
  • The navigation pane on the Start menu now expands when you hover over it with your mouse
  • Third-party digital assistants like Alexa will work from the Lock screen.

Microsoft says it's rolling out the update now and it will show up on PCs gradually. For users who wish to install it right now, they need to open the Settings app and navigate to Update & Security page > Windows Update, and then click on 'Check for updates'.

If the update is available for your PC, you will see a new 'Download and install now' option underneath the check for updates button. November 2019 Update will be labeled as an optional update, so it won't be installed automatically.

Microsoft says you cannot download the Windows 10 version 1909 with the Update Assistant tool until November 18. Instead you can use one of the following two methods.

Getting the November 2019 Update via Windows Update

To get the November 2019 Update using Windows Update, please follow these steps:

  1. Open the Start menu and click on Settings icons.
  2. Navigate to Update & Security.
  3. Open Windows Update page.
  4. Click 'Check for updates'.
  5. A new section "Feature update to Windows 10, version 1909" will appear.
  6. Click 'Download and install now' to begin the download.

Downloading the November 2019 Update ISO files

If you don't want to wait until the update is released for your device, you can also grab ISO images and manually force the update.

  1. Open Windows 10 download page in Chrome or Firefox.
  2. Open the Developer tools.
  3. Toggle device bar and select iPhone, iPad or a Pixel phone. It can be any platform other than Windows.
  4. Refresh the page and you have the direct download links on your screen.
  5. Select the language and system version (32 or 64-bit) and download the file. Remember that the link will expire in 24 hours.

Automatic update for older versions

Microsoft plans to automatically initiate Windows 10 feature update download on PCs running a version of Windows 10 that will be reaching the end of service in a few weeks or months. Users will still have the ability to choose a convenient time.

Today, Windows 10 April 2018 Update has reached end of service for Home and Pro editions, and such devices should receive the latest feature update.

Что нового в ноябрьском обновлении Windows 10: гибкая работа с уведомлениями и поиск по OneDrive в «Проводнике»

12 Nov 2019, 18:02 by Евгений Делюкин

Новых функций немного: в основном Microsoft улучшает производительность и стабильность системы.

12 ноября Microsoft выпустила второе «большое» годовое обновление Windows 10. В июле компания предупреждала, что до 2020 года сконцентрируется на улучшении производительности и качестве работы системы, а не на новых функциях.

Улучшение производительности и исправление ошибок

В обновлении Microsoft улучшит распределение работы ядер процессора, что должно повысить производительность многоядерных систем. На некоторых системах с определёнными процессорами улучшится время работы от аккумулятора — подробности Microsoft не приводит.

Также снизится время отклика пера для более точной работы. Ранее в Windows 10 время отклика автоматически устанавливалось системой, и производители не могли его снизить.

Онлайн-поиск в «Проводнике»

Теперь «Проводник» ищет файлы не только на устройстве, но и в личном облачном хранилище OneDrive. Визуально поиск тоже изменился: при вводе запроса отображается меню с предлагаемыми файлами.

Голосовые помощники на экране блокировки

Windows 10 позволит вызывать сторонние голосовые ассистенты: Alexa или «Алису» — без разблокировки системы. Ранее функция работала только с Cortana.

Создание событий из панели задач

Пользователи смогут добавлять новые задачи и события прямо с панели задач, не открывая «Календарь». Для этого достаточно нажать на иконку времени и даты.

Улучшение управления уведомлениями

Теперь уведомления сортируются по последнему отображаемому уведомлению, а не имени приложения, а их звуковой сигнал можно отключить при получении. Это поможет найти самые «активные» приложения и при необходимости отключить звуковые оповещения.

Также уведомления теперь можно настроить из полученного сообщения, а в панели быстрых действий добавилась кнопка перехода в настройки уведомлений.

Другие незначительные изменения

  • Меню «Пуск» автоматически раскрывается при наведении на кнопки «Настройки», «Документы», «Изображения» и «Выключение».
  • Экранный диктор и другие инструменты «Специальных возможностей» начнут считывать состояние клавиши Fn на клавиатуре.
  • Производители оборудования получат больше возможностей отладки для процессоров Intel.
  • Добавлена защита Windows Defender Credential Guard для ARM64-устройств. Она поможет уберечь от кражи учётные данные в компаниях, где используются такие устройства.
  • Добавлена возможность изменить политики Windows 10 S в организациях и разрешить Win32-приложения из Microsoft Intune.

Установка Windows 10 November 2019 Update будет похожа на установку еженедельного обновления системы через «Центр обновления Windows», а не тяжеловесных крупных обновлений.

Оно должно быть небольшим и быстро устанавливаться. Если на устройстве установлено майское обновление, пользователям не нужно перезагружать компьютер, чтобы получить новые функции.

Существенные изменения в Windows 10 появятся весной 2020 года — в следующем полноценном обновлении. Microsoft добавит полноценное ядро Linux с подсистемой Windows для Linux (WSL2) и возможность переустанавливать систему из облака.

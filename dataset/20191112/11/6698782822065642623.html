GitHub: количество реальных учетных записей дошло до 40 миллионов

12 Nov 2019, 11:47

По оценкам IDC, в мире насчитывается порядка 5 млн разработчиков ПО, работающих на условиях полной занятости, 7 млн частично занятых и еще 7 млн разработчиков, работающих бесплатно.

В очередном докладе компании GitHub State of the Octoverse говорится, что число учетных записей пользователей на сайте GitHub (за исключением фальшивых спамерских записей) дошло до 40 миллионов. Ясно, впрочем, что не все эти пользователи являются разработчиками программ. В GitHub подсчитывают общее число учетных записей, независимо от их активности и участия в разработке проектов, исходные коды которых размещены на сайте. Учетные записи на GitHub создают не только разработчики, но и сотрудники маркетинговых отделов и руководители по продуктам, отмечает обозреватель InfoWorld, а многие разработчики заводят по несколько учетных записей. У GitHub должны быть данные об активности пользователей, но компания не публикует подобных сводных отчетов.

По оценкам IDC, в мире насчитывается порядка 5 млн разработчиков программ, работающих на условиях полной занятости, 7 млн частично занятых и еще 7 млн разработчиков, работающих бесплатно — всего 24,2 млн.

В докладе GitHub приведены и другие интересные данные: например, 80% правок исходного кода в проекты поступает из-за пределов США, а 36% частных репозиториев кода приходится на страны Азии.

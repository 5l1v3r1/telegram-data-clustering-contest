5 Apps Android de produtividade para instalar no seu smartphone

12 Nov 2019, 16:01 by Maria Inês Coelho

O smartphone é hoje uma importante ferramenta de trabalho, como tal, há que se munir das melhores soluções para aumentar a sua produtividade.

Para tal, sugerimos 5 aplicações que não devem faltar no seu Android.

Apps Android de produtividade

Solid Explorer File Manager

Com uma versão de testes gratuita por 15 dias, a Solid Explorer File Manager é uma completa app de gestão de ficheiros para o seu smartphone. Além disso, é ideal para gerir os ficheiros associados às várias contas cloud, seja Dropbox, Google Drive, OneDrive, entre muitas outras.

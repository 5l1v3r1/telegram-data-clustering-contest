Regal Portrait Painted Entirely With HTML & CSS Looks Different On Every Browser

12 Nov 2019 by Thanussha Priyah

There is no need for a double take. It is an actual painting done wholly through strings of codes. Artist, UI engineer, and web developer Diana Adrianne Smith spent a couple of weekends to create this Flemish and baroque-inspired portrait entirely with HTML and CSS.
Smith, whose work previously took the internet by storm, creates such coded portraits completely by hand, leaving one to guess how painstakingly each character entered had led to a masterpiece.
There is no magic trick involved— just codes.

Bus di Area Bandara Ngurah Rai Bali Kembali Keluarkan Api dan Berasap

12 Nov 2019, 21:31 by Aditya Mardiastuti

Denpasar - Kejadian bus angkutan penumpang di area Bandara I Gusti Ngurah Rai, Bali, mengeluarkan percikan api dan berasap kembali terjadi. Percikan api ini muncul dari bagian mesin.

"Pukul 18.55 WITA, kami menerima laporan bahwa telah terjadi insiden 1 buah bus milik Lion bermasalah pada bagian mesin yang menimbulkan percikan api dan asap pada bagian mesin. Bus dalam keadaan kosong," kata Kepala Otoritas Bandara Wilayah IV Bali-Nusra Elfi Amir saat dimintai konfirmasi, Selasa (12/11/2019).

Elfi mengatakan posisi bus tersebut ada di bay 2. Api berhasil dipadamkan dengan alat pemadam kebakaran (firex) dan tim Aircraft Rescue and Firefighting (ARFF) juga sudah ke lokasi.

"Api segera dipadamkan dengan firex dan Tim ARFF sudah berada di lokasi. Penanganan sementara bus ditarik ke parkir bus apron sebelah barat menggunakan BTT," tuturnya.

Elfi mengatakan saat ini pihaknya masih menginvestigasi penyebab terjadinya insiden tersebut. Sopir bus juga sedang dimintai keterangan polisi bandara.

"Saat ini driver bus tersebut sudah berada di Polsek Kawasan Udara Ngurah Rai untuk dimintai keterangan," jelasnya.

Microsoft removes the safeguard hold for Windows 10 users with Realtek Bluetooth radio

19 Nov 2019, 11:30 by Anmol

A couple of days ago, we reported that users with old Realtek Bluetooth Radio drivers have been blocked from receiving the Windows 10 November 2019 Update. Now, it looks like Microsoft has finally resolved the issue with Realtek as the updates are unblocked for users.

According to Microsoft’s support document (via The WinCentral), Windows 10 PCs with old Realtek Bluetooth Radio drivers should expect November updates to show up within 48 hours. Earlier today, Microsoft made the November 2019 Update available for both Update Assistant and Media Creation Tool.

This issue was resolved with an updated driver for the affected Realtek Bluetooth radio and the safeguard hold has been removed. Please note, it can take up to 48 hours before you can update to offered Windows 10, version 1909 or Windows 10, version 1903.

With the issue sorted out, users can freely upgrade to November 2019 Update as there are no known issues affecting the update at the moment.

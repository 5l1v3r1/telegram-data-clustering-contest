Microsoft is now rolling out Windows 10 November Update to more devices

19 Nov 2019, 00:12 by Anmol

Microsoft announced the Windows 10 November Update earlier this year and the update started rolling out to the consumers earlier this month. Unlike the other Windows 10 updates, the November 2019 Update doesn’t come with major features or changes.

Instead, Microsoft is saving those for 20H1 and is releasing 19H2 with a set of minor improvements and changes. Microsoft has now updated its support article (via Windows Latest) confirm the availability of November 2019 Update via Media Creation Tool. In case you didn’t know, Media Creation Tool allows customers to create a Windows 10 ISO which can be used to upgrade any Windows 10 PC.

However, you should be careful if using the Media Creation Tool to install the update on another PC. Since the tool can’t run a prerequisite check, it won’t be able to determine if the update is compatible with the third-party software. As of now, Microsoft is blocking the November 2019 Update on some Realtek Bluetooth Radio drivers. Apart from the old Realtek drivers, no issues have been reported so you can go ahead and install the update on other machines.

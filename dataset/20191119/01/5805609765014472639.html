How to Hide Items from the Control Panel in Windows

19 Nov 2019, 01:15 by Karanpreet Singh

If you have been using Windows 10 for a while, then you might be well aware of the Control Panel. It’s a centralized hub from where you can excess different Windows settings. Not just settings, but adding/removing programs were also done through the Control Panel.

Although you can uninstall programs from the app drop-down list on Windows 10, still Control Panel is the easiest way to add/remove new programs. If your computer is shared with others, then you may not want users to access certain items of Control Panel as through the control panel, one can modify the registry, modify system files, etc.

How to Hide Items from the Control Panel in Windows

So, if your computer is used by others also, then you can always stop others from changing certain settings on the Control Panel. In this article, we are going to share a working method that would let you hide Control Panel Settings on Windows 10 operating system.

Hide Items from the Control Panel in Windows:

Step 1. First of all, open Run dialog box on your Windows 10 operating system.

Step 2. On the RUN dialog box, enter ‘gpedit.msc’ and hit enter.

Step 3. This will open the Local Group Policy Editor. There you need to browse to the User Configuration > Administrative Templates > Control Panel. There double click on the ‘Hide specified control panel items’ from the right-hand side panel.

Step 4. In the next window, you need to enable the ‘Hide Specified Control Panel Items’ and clock on the ‘Show’ button near the List of disallowed control panel items.

Step 5. In the next window, you need to add the Canonical name of the Control Panel item which you want to hide. Below we have shared a list of Canonical name for any specific control panel item.

Step 6. Once added, close the opened window and restart your computer.

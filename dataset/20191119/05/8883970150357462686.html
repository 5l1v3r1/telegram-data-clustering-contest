Microsoft เตรียมถอดแอปฯ Cortana หลังมีผู้ใช้น้อยลง

19 Nov 2019, 05:37 by ZNeze

Microsoft เตรียมถอดระบบช่วยเหลือด้วยเสียง Voice Assistant หรือ Cortana เวอร์ชั่นแอพพลิเคชั่นบน iOS และ Android ออกจากหลายประเทศ คือ อังกฤษ, ออสเตรเลีย, เยอรมนี, เม็กซิโก, จีน, สเปน, แคนาดา และ อินเดีย ในวันที่ 31 มกราคม 63 จากยอดผู้ใช้งานแอปลดลง

จากที่เคยพยายามผลักดันออกมา ให้ผู้ใช้งานต่างๆ โดยเฉพาะกับบนระบบ Windows ในเครื่องคอมพิวเตอร์ PC จนมาถึงแอปในสมาร์ทโฟน แต่ก็ดูว่าจะไม่ค่อยเกิดเท่าที่ควรสักเท่าไหร่ เพราะว่ายอดก็ไม่ได้เป็นไปตามเป้าหมายที่วางเอาไว้เลย

แต่แม้ว่าจะยกเลิกบริการไปในหลายประเทศ แต่ Microsoft ยังไม่ได้เตรียมปิดอย่างถาวร อย่างในอเมริกาเอง ก็ยังเปิดให้ใช้งานแอปได้ต่อไป

ส่วนผู้ใช้งานในประเทศที่จะถูกยกเลิก ยังสามารถเข้าดูข้อมูล Reminder หรือ List ต่างๆ ที่สร้างเอาไว้ในแอป Cortana เดิมได้ ผ่านแอป To Do ที่มีการซิงก์ข้อมูลเหมือนกันแบบอัตโนมัติ

เรื่องนี้ไม่น่าเป็นที่แปลกใจสักเท่าไหร่ เพราะมีสัญญาณมาให้เห็นก่อนหน้านี้แล้ว ตั้งแต่ Satya Nadella ตำแหน่ง CEO ของ Microsoft คนปัจจุบัน ได้บอกเอาไว้ว่าไม่ได้มอง Cortana เป็นคู่แข่งกับทั้ง Amazon Alexa หรือ Google Assistant เลย และยังมีการลาออกของ Javier Soltero ที่เห็นหัวเรือใหญ่ทีม Cortana

และจากการถอดออก ก็เปิดรับเอา Alexa ของ Amazon เข้ามาแทน ซึ่งเริ่มเห็นได้จากช่วงปีที่แล้ว และยังมีการขายสินค้าของ Amazon ทั้ง Echo และ Echo Dot ผ่านร้านและช่องทางจำหน่ายสินค้าของ Microsoft เองด้วย

สำหรับการติดตั้งระบบ Windows 10 ที่มีการเปิดระบบตัวนี้ช่วยเอาไว้ Microsoft ก็เตรียมที่จะปิดฟังชั่นนี้ลงด้วยเช่นเดียวกัน ไม่ได้ตั้งเป็นค่า Default มาตรฐานเหมือนเดิมอีกต่อไป และจะถอดออกจากแถบ Search Bar ด้วย

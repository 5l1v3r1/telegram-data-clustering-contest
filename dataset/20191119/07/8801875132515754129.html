GitHub 推出手機 App，程式設計師隨時都可開工了

19 Nov 2019, 07:57 by Unwire HK

據最新報告，GitHub 在全世界擁有 4,000 萬使用者，本週又宣布多項新發展，包括推出 GitHub 手機測試版，讓使用者隨時隨地接收通知、提交新項目及在儲存庫加入新程式碼等；GitHub Actions 與 GitHub Packages 也推出正式版，GitHub 更發表「北極開源碼檔案櫃」（GitHub Arctic Code Vault），稱可將重要的程式碼保存超過 1 千年。

GitHub for mobile 為 GitHub 首個手機 App，讓開發者與團隊更緊密聯繫，例如發表程式設計的討論串，讓其他開發者參與討論；檢查程式碼並註釋，也能為程式碼合併變更。不過目前 iOS 版還是測試版，Android 版更要等到 2020 年才會上線。GitHub 執行長 Nat Friedman 打賭說，「 5 年內，GitHub 的互動將有一半以上會轉移到行動裝置」。另外，GitHub 認為平板電腦版 App 可讓使用者有更多空間瀏覽程式碼，帶來更好的體驗。

GitHub 也加入不少新功能，之一是提升通知功能的使用體驗。一些開發者會不斷收到大量通知，GitHub 集中到一個 Inbox，然後使用者可設定過濾方式，避免開發者信箱被雜亂通知塞滿。GitHub 還加入程式碼導覽、程式碼搜尋、程式碼檢查任務指派、預定提醒以及功能預覽等。

2018 年推出、用做建立自動化的軟體開發流程的測試版功能 GitHub Actions ，亦終於推出正式版，開發者可透過 GitHub Marketplace 取得。另一項程式碼打包工具 GitHub Package Registry，現在也成為正式版 GitHub Package。GitHub 表示，每個開發者帳號都可以使用以上兩項工具，且所有公共儲存庫皆可免費使用。

除了功能改善，GitHub 也宣布 GitHub Archive Program 封存計畫，當作保存開源碼。GitHub 表示，開源碼軟體是現代文明的隱藏基石，也是全人類共享的遺產，他們將把不同複本以不同資料格式存放在各處，希望將這些重要開源資產，在北極開源碼檔案櫃儲存至少 1 千年。

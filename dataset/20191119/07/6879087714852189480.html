Доход харьковского айтишника превысил 2000 долларов - исследование

19 Nov 2019, 07:48

Среднемесячный доход  харьковского айтишника составил 2 тысячи 75 долларов. Об этом сообщили в Харьковском IT-кластере по итогам исследования, проведенного в Харьковской области. В то же время средняя зарплата молодого специалиста, только начинающего работать в отрасли (джуниор), составляет 600 долл.

Как сообщили в IT-кластере, харьковкие айтишники владеют:

  • SQL – 60%;
  • Javascript – 45%;
  • Java – 32%;
  • C++ - 28%;
  • Python – 26%.

Было отмечено, что доля работающих с Javascript в сравнении с прошлым годом уменьшилась, а работающих с Python – увеличилась.

72% харьковских айтишников - мужчины, остальные - женщины.

Справка SQ. Харьковский IT-кластер создан в 2015 г. В него входят 70 участников, что составляет 30% всех крупных айтикомпаний региона.

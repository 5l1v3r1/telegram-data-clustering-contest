AMD Radeon Pro 5000M 系列行動 GPU 為蘋果新款 16 吋 MacBook Pro 帶來高效能 RDNA 架構

19 Nov 2019, 06:45 by TechNews

AMD 宣布蘋果最新推出的 MacBook Pro 搭載全新 AMD Radeon Pro 5500M 及 5300M 行動 GPU。AMD Radeon Pro 5000M 系列為首款針對專業使用者打造的 7 奈米製程獨立行動 GPU，採用領先業界的製程技術及強大的 AMD RDNA 架構，為影片編輯、3D 內容創作及基於 macOS 的遊戲開發帶來突破性的繪圖效能。

AMD Radeon Pro 5000M 系列行動 GPU 採用全新 AMD RDNA 架構，全新設計旨在打造卓越的效能、可擴充性以及高功耗效率。新款 GPU 內含高達 8GB 的高速 GDDR6 記憶體，擁有高達 192 GB/s 的記憶體頻寬，為 GDDR5 的 2 倍，並提供高達 4.0 TFLOPS 的單精度浮點數（FP32）效能。此外，新款 GPU 針對現今頂尖專業內容創作應用及運算密集的影像渲染工作負載進行優化，在 MacBook Pro Retina 螢幕上呈現令人讚嘆的超流暢及高畫面更新率的視覺效果。

AMD 全球副總裁暨 Radeon 繪圖技術事業群總經理 Scott Herkelman 表示，現今的高階專業創意工作者希望不論在辦公室或路上都能將腦中創意化為作品，而不受技術局限。Radeon Pro 5000M 系列 GPU 為 MacBook Pro 使用者帶來強勁的效能及視覺逼真度，讓他們能專注發揮創意，創造出令人驚豔的作品。

AMD Radeon Pro 5000M 是 2019 年 10 月所發表的 AMD Radeon RX 5500 系列顯示卡的新成員。其他筆電製造商也將推出搭載 AMD Radeon RX 5500M 遊戲 GPU 及新款 AMD Radeon RX 5300M 遊戲 GPU 的新系統，預計於本季上市。

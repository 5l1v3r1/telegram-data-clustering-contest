Σταματήστε να φορτίζετε το κινητό σας με αυτόν τον επικίνδυνο τρόπο

Μια δωρεάν φόρτιση μπορεί να καταλήξει σε αποστράγγιση του τραπεζικού σας λογαριασμού

19 Nov 2019, 22:50 by Metrosport.gr Team

Eνα τηλέφωνο ή φορητός υπολογιστής που είναι έτοιμα να κλείσουν λόγω χαμηλής μπαταρίας είναι ικανά να οδηγήσουν κάποιον σε μια απελπισμένη αναζήτηση προκειμένου να βρει έναν τρόπο να φορτίσει τη συσκευή του.

Ωστόσο, θα πρέπει κανείς να το σκεφτεί δύο και τρεις φορές, προτού καταφύγει στο πρώτο καλώδιο που θα βρει μπροστά του σε έναν σταθμό φόρτισης αεροδρομίου ή προτού συνδεθεί με τη θύρα USB του ξενοδοχείου - οι χάκερ παραμονεύουν.

Καθώς πλησιάζει η περίοδος των χειμερινών διακοπών, οι ειδικοί προειδοποιούν τους ταξιδιώτες για μια απάτη με φορτιστές USB.

Μια δωρεάν φόρτιση θα μπορούσε να καταλήξει σε αποστράγγιση του τραπεζικού σας λογαριασμού. Η κομπίνα γίνεται όταν οι ανυποψίαστοι χρήστες συνδέουν τις ηλεκτρονικές τους συσκευές σε θύρες USB ή χρησιμοποιούν καλώδια USB που φέρουν κρυφά με κακόβουλο λογισμικό.

Το κακόβουλο λογισμικό μολύνει τις συσκευές, δίνοντας στους χάκερ πρόσβαση σε αυτές. Οι «εισβολείς» μπορούν στη συνέχεια να διαβάσουν και να εξάγουν τα δεδομένα, συμπεριλαμβανομένων των κωδικών πρόσβασης, ακόμα και να κλειδώσουν προγράμματα ή εξαρτήματα, κάνοντάς τα άχρηστα.

Ασφαλείς τρόποι για να προστατεύσετε τον εαυτό σας και τις συσκευές σας είναι να έχετε τα δικά σας καλώδια φόρτισης, φορτίζοντας μόνο απευθείας από μια ηλεκτρική πρίζα και χρησιμοποιώντας φορητές μπαταρίες που αγοράστηκαν από αξιόπιστους πωλητές.

Αλλά δεν είναι μόνο τα καλώδια που θέτουν σε κίνδυνο τους καταναλωτές τεχνολογίας. Είναι και οι θύρες.

Οπως οι απατεώνες υποκλέπτουν κωδικούς χρεωστικών καρτών με την τοποθέτηση παράνομων συσκευών στα ΑΤΜ, οι χάκερ μπορούν εύκολα να βγάλουν τις θύρες USB και να τις αντικαταστήσουν τους με το δικό τους κακόβουλο υλικό.

Αν και οι ειδικοί δεν είναι σίγουροι για το πόσο συχνά συμβαίνουν αυτές οι επιθέσεις από χάκερ, η αυξανόμενη χρήση θυρών USB σε μέρη όπως ξενοδοχεία, αεροδρόμια και δημόσιες συγκοινωνίες έχει αυξήσει και τον κίνδυνο να πέσουν θύματα ανυποψίαστοι πολίτες.

Ενας ακόμη τρόπος με τον οποίον οι καταναλωτές θα μπορούσαν να προστατευτούν είναι η χρήση ειδικών προστατευτικών συσκευών που προσαρτώνται σε καλώδια USB - γνωστά ως «προφυλακτικά USB».

Με τη συσκευή αυτή το κινητό ή το λάπτοπ θα φορτίσει αλλά το καλώδιο δεν θα μπορεί να στείλει ή να λάβει δεδομένα.

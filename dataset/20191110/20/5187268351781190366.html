Dica: Como instalar o macOS Catalina em Macs não suportados

10 Nov 2019, 20:00 by Vítor M.

A Apple lançou ao público no início de outubro o macOS Catalina. Este é o novo sistema operativos para os computadores da Apple. Apesar der trazer novidades de desempenho, alguns computadores mais antigos ficaram de fora da atualização, tal como demos a conhecer. Nesse sentido, algumas máquinas ficaram de fora.

O que trazemos hoje pode ser uma “salvação” para estes casos que ficaram de fora. Vamos mostrar como instalar o macOS Catalina em Macs não suportados

macOS Catalina em Macs não suportados

Provavelmente tem uma máquina que não está na lista de atualização e até queria ter o macOS Catalina. Sabendo disso, procuramos uma solução e encontramos. Assim, trazemos uma solução de terceiros que permite umas afinações para “orientar” o instalador do macOS Catalina a este funcionar Macs não compatíveis.

Tenho um Macbook que não está na lista, devo atualizar na mesma?

Se deve ou não instalar o macOS Catalina no seu Mac, isso é outro assunto. Na verdade, se a Apple não o colocou na lista é porque as novidades apresentadas e o desempenho no geral ficam aquém nestes Macs. Por exemplo, o Catalina traz um recurso interessante, o Sidecar. Com este serviço, nem todos os que receberam a atualização o irão poder usar. Inclusive deixamos uma lista com os dispositivos compatíveis.

Apesar disso, o utilizador pode querer avançar porque está interessado em executar o macOS 10.15 num hardware não suportado. Será utilizado um “patch” para o fazer.

Se quer avançar, por sua conta e risco, veja no botão a seguir o utilitário DosDude Catalina Patcher. Veja no final um vídeo para o ajudar passo a passo.

Disable MacBook Pro Dedicated GPU

Important Notes:

    • I have recently come up with a MUCH better, PERMANENT solution to this issue, which I’d recommend using instead. More info can be foundhere.
  • After performing the steps below, sleep mode and brightness control will NOT work properly on your system.

– First, ensure your machine is completely shut down (You can press Control + Option + Shift + Power keys to reset the SMC).

– Now, boot your machine while holding the Command + S keys. As the system boots, it should display white text on a black screen, and should eventually stop at a prompt.

– At this prompt, enter the following command and press Return: nvram fa4ce28d-b62f-4c99-9cc3-6815686e30f9:gpu-power-prefs=%01%00%00%00

– Next, enter reboot, and press Return.

– Immediately after running the reboot command, hold down Command + R (if you’re running OS X 10.11 or later) to boot into Recovery Mode. If you don’t have a Recovery Partition, you can boot from a Mac OS installer USB drive.

– Once booted in Recovery Mode, open Terminal, and run the following command: csrutil disable

– Reboot the system.

– Once the system reboots, it should now successfully boot into your install of OS X. Now, you’ll need to download the program found here.

– Run the program, follow the prompts, and enter your password when prompted.

– Once the tool finishes running, your system should now run without the dedicated graphics, and should work completely normally.

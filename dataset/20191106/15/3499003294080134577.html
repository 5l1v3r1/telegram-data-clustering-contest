На юге столицы задержан подозреваемый в хищении оружия

06 Nov 2019, 15:11
Оперуполномоченные уголовного розыска ОМВД России по району Царицыно совместно с оперативниками УВД по ЮАО задержали приезжего мужчину, подозреваемого в хищении оружия.

В ОМВД России по району Царицыно обратился житель столицы. По словам заявителя в одном из ресторанов быстрого питания на Кантимировской улице неизвестный тайно похитил из его сумки, оставленной без присмотра, травматический пистолет и магазин с 10 патронами, после чего скрылся.
В результате проведения оперативно-розыскных мероприятий оперуполномоченные уголовного розыска ОМВД России по району Царицыно совместно с оперативниками УВД по ЮАО задержали подозреваемого. Им оказался 26-летний приезжий. Похищенное изъято.
По данному факту следствием территориального ОМВД возбуждено уголовное дело по признакам преступления, предусмотренного статьей 226 УК РФ «Хищение либо вымогательство оружия, боеприпасов, взрывчатых веществ и взрывных устройств». Подозреваемый задержан в порядке статьи 91 УПК РФ.

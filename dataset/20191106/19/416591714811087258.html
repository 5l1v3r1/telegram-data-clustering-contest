Saiba como ativar o modo escuro no WhatsApp Web

Acompanhando a tendência entre os aplicativos, o WhatsApp Web recebe sua versão de modo noturno

06 Nov 2019, 19:58 by Vinicius Szafran

O modo escuro vem sendo uma tendência entre as empresas. Twitter, Instagram e Google já disponibilizaram o modo, e agora será possível usá-lo também no WhatsApp Web. A crescente popularização do modo noturno pode ser explicada pelo fato de ele ser mais suave aos olhos do usuário, principalmente depois de muito tempo utilizando o computador ou celular.

Para ativar o modo no WhatsApp Web, alguns componentes são essenciais. Com eles, suas conversas pelo computador ficarão menos cansativas. Este método funciona apenas nos navegadores Mozilla Firefox e Google Chrome, pois depende de algumas extensões disponíveis apenas nesses browsers. Veja abaixo o processo para configurar o modo escuro.

Importante

Esta não é uma versão oficial do modo escuro, e sim uma alternativa para implementá-lo. O WhatsApp ainda está desenvolvendo seu tema noturno, mas isso não quer dizer que não existe uma maneira de evitar desconforto ao usar a ferramenta. Será necessário o download da extensão Stylus. Confira se o seu navegador está atualizado, porque a extensão pode apresentar problemas de compatibilidade com versões mais antigas do browser.

Como ativar o modo escuro no WhatsApp Web

O processo é bem simples e rápido de se executar. Como o WhatsApp ainda não implementou a opção, este guia traz uma versão alternativa, porém funcional. Confira:

  1. Para começar, é necessário instalar o navegador. Pode ser tanto o Google Chrome quanto o Mozilla Firefox.
  2. Caso tenha optado pelo Chrome, acesse este endereço para instalar o Stylus, uma extensão que permite aplicar temas para sites específicos. Se sua preferência é pelo Mozilla, então siga este link
  3. Quando a extensão estiver corretamente instalada, será necessário acrescentar o tema que o Stylus usa no WhatsApp Web. O tema está disponível aqui. O site também conta com outros diferentes temas, caso queira incrementar ainda mais seu navegador.
  4. Clique em "Install Style", em seguida abra ou recarregue a página do WhatsApp Web se for necessário.

Pronto! Agora você já sabe como ativar o modo escuro para o WhatsApp Web.

موعد مباراة المنتخب الودية أمام ليبيريا والقنوات الناقلة

07 Nov 2019, 02:21 by ياسمين تيسير

ياسمين تيسير تنطلق مساء اليوم الخميس مباراة المنتخب الأول الودية أمام ليبيريا استعدادا لخوض مباراتي الجولة الأولى والثانية للتصفيات المؤهلة لبطولة كأس الأمم الأفريقية 2020 أمام جزر القمر وكينيا.

وتقام المباراة في تمام الساعة السادسة مساء على استاد برج العرب و ستذاع المباراة عبر قناة "أون سبورت".

ويستعد حاليا منتخب مصر لـ مباراتي كينيا وجزر القمر في التصفيات المؤهلة لكأس الأمم الأفريقية 2021 بالكاميرون.

وأعلن البدرى ضم 32 لاعبا لصفوف المنتخب وجاءت كالآتى : محمد الشناوي (الأهلي ) محمد عوّاد (الزمالك ) محمد بسام (طلائع الجيش) أحمد فتحي (الأهلي ) رجب بكار (بيراميدز) محمد هاني (الأهلي ) عبدالله جمعة( الزمالك ) محمود وحيد (الأهلي ) محمد حمدي (بيراميدز ) محمود علاء (الزمالك ) رامي ربيعة (الأهلي ) باهر المحمدي (الإسماعيلي ) أحمد أيمن منصور (بيراميدز) طارق حامد (الزمالك) عمرو السولية (الأهلي) حمدي فتحي (الأهلي) عمر جابر(بيراميدز) عبدالله السعيد (بيراميدز) محمد مجدي أفشة (الأهلي) حسين الشحات (الأهلي) أحمد السيد زيزو (الزمالك) محمد فاروق (بيراميدز) اسلام عيسي (بيراميدز ) حسام حسن (سموحة ) مروان حمدي (وادي دجله) احمد جمعه (المصري ) أحمد حجازي (وست بروميتش ألبيون الإنجليزي) ومحمد صلاح (ليفربول الإنجليزي ) محمد النني (بشكتاش التركي) أحمد حسن كوكا (براجا البرتغالي) محمود حسن تريزيجيه (أستون فيلا الإنجليزي ) محمود عبدالمنعم كهربا (ديسبورتيفو أفيس البرتغالي ).

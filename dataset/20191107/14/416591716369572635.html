Nova versão do Chrome apresenta melhorias de navegação e integração Android

O novo sistema permite trocar informações com dispositivos Android de forma fácil e rápida. Além disso, é possível configurar os dois dispositivos no mesmo lugar, mas mantendo suas particularidades

07 Nov 2019, 14:39 by Luiz Nogueira

Após alguns testes, a versão 78 do Chrome OS já está disponível. Dentre os principais recursos da atualização, está a opção de configurar separadamente a versão de navegador e do Android, opções de compartilhamento entre a versão de desktop e móvel, além do suporte para as chamadas "Virtual Desks".

Compartilhamento de informações

Em sua nova versão, o Chrome recebeu uma nova opção de compartilhamento entre dispositivos. Agora é possível enviar informações diretamente entre a versão de navegador e de celular.

Por exemplo, quando o usuário visualizar um número de telefone para o qual deseja ligar, basta selecionar com o botão direito a opção "clique para ligar" que o número será direcionado para o dispositivo Android. Reralizar chamadas é mais rápido dessa maneira do queinserindo manualmente os dígitos.

Configurações separadas

Com o Chrome OS 78, os usuários podem ter configurações separadas para o navegador na versão de computador e Android. O primeiro pode ser acessado diretamente pelas configurações padrões disponíveis no navegador em si. Porém, uma nova opção descrita como "Chrome OS Settings" abre uma guia em que é possível definir preferências do dispositivo como Wi-Fi, Bluetooth e informações como tema utilizado.

Áreas de trabalho virtuais

Para a opção de "Virtual Desks", o Google implementou novos atalhos para facilitar sua criação e navegação. Para criar uma nova máquina virtual, basta selecionar os comandos Shift> Pesquisar> +; para alternar entre as máquinas, os comandos são os seguintes: Pesquisar> ]; para mover as abas pelas máquinas basta apertar Shift> Pesquisar> ].

YouTube

O YouTube para Android também recebeu algumas melhorias com o Chrome OS 78. Depois de iniciar um vídeo no Android e realizar alguma ação como mudar de janela ou minimizá-lo, será aberto um pequeno player no canto inferior direito em que o vídeo continuará passando.

Os controles disponíveis neste momento seriam apenas os de alternar o áudio, reproduzir/pausar e pular para o próximo vídeo. Ele também pode ser enviado para o Chromebook para continuar a reprodução por lá.

Personalização

O Chrome OS não recebeu todas as opções de personalização disponíveis para as outras plataformas, mas o Google incluiu uma aba "personalizar" para que algumas mudanças possam ser realizadas.

Agora é possível definir quais atalhos serão apresentados abaixo da aba de pesquisa do navegador. Os usuários podem decidir entre atalhos sugeridos com base nos sites mais visitados e ocultar os atalhos.

Também é possível mudar a cor e o tema do navegador – que permite escolher entre vinte temas predefinidos ou iniciar um seletor de cores para criar o seu próprio.

Preferências de impressão

O Chrome 78 simplificou a preferência de impressão. O sistema agora lista automaticamente todas as impressoras compatíveis sem a necessidade de configuração prévia. O Google também diminuiu o número de etapas necessárias para salvas as impressoras usadas com maior frequência.

Linux

A nova versão do sistema também apresenta melhorias para o Linux no Chrome OS. Agora é possível fazer backups de aplicativos e arquivos do Linux salvando-os diretamente no armazenamento local do dispositivo, em uma unidade externa ou no Google Drive. Essa cópia pode ser restaurada ao configurar um novo computador.

O suporte à GPU Crostini por padrão oferece uma "experiência nítida e com menor latência". Além disso, os usuários serão avisados se um aplicativo Linux não suportar o teclado virtual do modo tablet dos Chromebooks.

O Chrome OS 78 deve chegar a todos os Chromebooks nas próximas semanas.

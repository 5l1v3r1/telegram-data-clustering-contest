Минобранауки РФ повысило минимальные пороговые баллы ЕГЭ для 252 вузов

2 ноября, 10:05

Минобранауки подняло для 252 подведомственных вузов необходимые минимальные баллы для поступления, установив их значение выше определённых Рособрнадзором. Об этом сообщает «Российская газета».

Университеты уже опубликовали на своих информационных порталах новые правила приёма и необходимые входные баллы на учебные программы в 2020–2021 годах. По словам заместителя министра науки и высшего образования Марины Боровской, новые значения минимальных баллов ЕГЭ были установлены приказом ещё в сентябре 2019 года.

«По математике — 39 баллов, по русскому и иностранным языкам, географии, литературе, физике, химии — 40, по биологии, информатике, истории — 42. Самый высокий минимум — 44 балла по обществознанию. И это опять же — минимум. Для каждого вуза отдельно разработана своя „сетка“ баллов по каждой конкретной специальности», — рассказала о новых пороговых значениях Боровская.

Также замминистра добавила, что изменения коснутся 10-12 процентов абитуриентов. По её мнению, благодаря новой системе школьник сможет «правильнее определиться с возможностью дальнейшего обучения, не испытывая ложных надежд на поступление в вуз».

Microsoft представила логотип нового браузера Edge на базе Chromium

02 Nov 2019, 20:23 by Евгений Делюкин

Теперь никакого сходства с Internet Explorer.

Новый логотип обнаружили пользователи последней версии Microsoft Edge ветки Canary в спрятанной мини-игре, посвящённой сёрфингу.

Чтобы обнаружить команду для запуска игры, энтузиастам пришлось разгадывать головоломки от сотрудников Microsoft. Для этого они объединялись в чатах Discord, на форумах Microsoft, Reddit и в других сообществах «инсайдеров». Например, для одного из решений «инсайдеры» создали 3D-логотип Edge, расшифровывая его код модели с помощью изображения, присланного Microsoft.

Головоломки привели энтузиастов к серии слов, обнаруженных в семи подсказках — их нужно было ввести в Javascript-функцию на сайте Microsoft Edge Insider. Запустив код, «инсайдеры» получили финальные инструкции для разблокирования команды запуска игры: создать «коллекцию», назвать её Microsoft Edge и добавить в неё четыре элемента, которые образуют слово «S.U.R.F.»

Игру можно запустить командой edge://surf/

Она похожа на SkuFree — игру про лыжника, которая в 1991 году входила в состав пакета Microsoft Entertainment Pack 3 для Windows. Игрокам нужно использовать клавиатуру для навигации, сбора бонусов и уворачивания от щитов.

После завершения игры появляется объявление с новым логотипом.

Бета-версия Microsoft Edge была выпущена в августе 2019 года. Microsoft пока не объявила дату релиза нового браузера.
